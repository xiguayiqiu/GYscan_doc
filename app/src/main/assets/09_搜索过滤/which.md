# which - 查找命令路径

## 1. 命令简介
which 命令用于查找命令路径，是 Linux 系统中常用的搜索过滤命令之一。

## 2. 命令语法
```bash
which [选项] [命令]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -a | 显示所有匹配的路径 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 查找命令路径
```bash
which ls
```

### 4.2 显示所有匹配的路径
```bash
which -a ls
```

## 5. 注意事项
- which 命令用于查找命令路径，搜索 PATH 环境变量中的目录
- which 命令是一个常用的 shell 工具，用于查找命令的可执行文件路径
