# egrep - 扩展正则表达式搜索

## 1. 命令简介
egrep 命令用于扩展正则表达式搜索，是 Linux 系统中常用的搜索过滤命令之一。

## 2. 命令语法
```bash
egrep [选项] [模式] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -i | 忽略大小写 |
| -v | 反向匹配 |
| -c | 显示匹配计数 |
| -n | 显示行号 |
| -r | 递归搜索 |
| -l | 只显示文件名 |
| -A | 显示匹配行后的指定行数 |
| -B | 显示匹配行前的指定行数 |
| -C | 显示匹配行前后的指定行数 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 扩展正则表达式搜索
```bash
egrep "pattern1\|pattern2" file.txt
```

### 4.2 忽略大小写
```bash
egrep -i "pattern" file.txt
```

### 4.3 递归搜索
```bash
egrep -r "pattern" directory/
```

### 4.4 显示行号
```bash
egrep -n "pattern" file.txt
```

## 5. 注意事项
- egrep 命令用于扩展正则表达式搜索，支持更多的正则表达式特性
- egrep 命令是 grep 命令的一个变体，使用扩展正则表达式
- egrep 命令是一个常用的文本搜索工具，用于查找文件中的特定模式
