# uniq - 去除重复行

## 1. 命令简介
uniq 命令用于去除重复行，是 Linux 系统中常用的搜索过滤命令之一。

## 2. 命令语法
```bash
uniq [选项] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -c | 显示重复次数 |
| -d | 只显示重复行 |
| -u | 只显示不重复行 |
| -i | 忽略大小写 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 去除重复行
```bash
sort file.txt | uniq
```

### 4.2 显示重复次数
```bash
sort file.txt | uniq -c
```

### 4.3 只显示重复行
```bash
sort file.txt | uniq -d
```

### 4.4 只显示不重复行
```bash
sort file.txt | uniq -u
```

## 5. 注意事项
- uniq 命令用于去除重复行，要求输入已经排序
- uniq 命令通常与 sort 命令配合使用
- uniq 命令是一个常用的文本处理工具，用于去除重复行
