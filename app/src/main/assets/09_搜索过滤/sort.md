# sort - 排序文本

## 1. 命令简介
sort 命令用于排序文本，是 Linux 系统中常用的搜索过滤命令之一。

## 2. 命令语法
```bash
sort [选项] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -n | 按数值排序 |
| -r | 反向排序 |
| -k | 指定排序键 |
| -t | 指定分隔符 |
| -u | 去除重复行 |
| -f | 忽略大小写 |
| -o | 输出到文件 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 排序文本
```bash
sort file.txt
```

### 4.2 按数值排序
```bash
sort -n file.txt
```

### 4.3 反向排序
```bash
sort -r file.txt
```

### 4.4 去除重复行
```bash
sort -u file.txt
```

## 5. 注意事项
- sort 命令用于排序文本，默认按字典顺序排序
- sort 命令可以按数值、日期等方式排序
- sort 命令是一个常用的文本处理工具，用于排序文本数据
