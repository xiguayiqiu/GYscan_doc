# cut - 剪切文本

## 1. 命令简介
cut 命令用于剪切文本，是 Linux 系统中常用的搜索过滤命令之一。

## 2. 命令语法
```bash
cut [选项] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -b | 按字节剪切 |
| -c | 按字符剪切 |
| -f | 按字段剪切 |
| -d | 指定分隔符 |
| -s | 只显示有分隔符的行 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 按字段剪切
```bash
cut -f 1 file.txt
```

### 4.2 指定分隔符
```bash
cut -d "," -f 1 file.csv
```

### 4.3 按字节剪切
```bash
cut -b 1-5 file.txt
```

### 4.4 按字符剪切
```bash
cut -c 1-5 file.txt
```

## 5. 注意事项
- cut 命令用于剪切文本，按字节、字符或字段提取文本
- cut 命令是一个常用的文本处理工具，用于提取文本数据
