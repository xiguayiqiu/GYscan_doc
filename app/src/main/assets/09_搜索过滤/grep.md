# grep - 搜索文本模式

## 1. 命令简介
grep 命令用于搜索文本模式，是 Linux 系统中常用的搜索过滤命令之一。

## 2. 命令语法
```bash
grep [选项] [模式] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -i | 忽略大小写 |
| -v | 反向匹配 |
| -c | 显示匹配计数 |
| -n | 显示行号 |
| -r | 递归搜索 |
| -l | 只显示文件名 |
| -A | 显示匹配行后的指定行数 |
| -B | 显示匹配行前的指定行数 |
| -C | 显示匹配行前后的指定行数 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 搜索文本模式
```bash
grep "pattern" file.txt
```

### 4.2 忽略大小写
```bash
grep -i "pattern" file.txt
```

### 4.3 递归搜索
```bash
grep -r "pattern" directory/
```

### 4.4 显示行号
```bash
grep -n "pattern" file.txt
```

## 5. 注意事项
- grep 命令用于搜索文本模式，支持正则表达式
- grep 命令是一个常用的文本搜索工具，用于查找文件中的特定内容
