# xz - 使用xz压缩

## 1. 命令简介
xz 命令用于使用xz算法压缩文件，是 Linux 系统中常用的压缩备份命令之一。

## 2. 命令语法
```bash
xz [选项] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -c | 将压缩数据输出到标准输出 |
| -d | 解压缩文件 |
| -f | 强制压缩文件，覆盖已存在的文件 |
| -k | 保留原始文件 |
| -l | 显示压缩文件的信息 |
| -t | 测试压缩文件的完整性 |
| -v | 显示详细信息 |
| -1 | 最快压缩速度 |
| -9 | 最高压缩率 |

## 4. 实战示例

### 4.1 压缩文件
```bash
xz file.txt
```

### 4.2 解压缩文件
```bash
xz -d file.txt.xz
```

### 4.3 保留原始文件
```bash
xz -k file.txt
```

### 4.4 显示压缩文件的信息
```bash
xz -l file.txt.xz
```

### 4.5 测试压缩文件的完整性
```bash
xz -t file.txt.xz
```

### 4.6 最快压缩速度
```bash
xz -1 file.txt
```

### 4.7 最高压缩率
```bash
xz -9 file.txt
```

## 5. 注意事项
- xz 命令会将文件压缩为 .xz 格式
- xz 命令默认会删除原始文件，使用 -k 选项可以保留原始文件
- xz 命令的压缩率通常比 bzip2 命令高，但压缩速度更慢
- 使用 unxz 命令可以解压缩 .xz 文件
