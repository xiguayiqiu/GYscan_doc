# gzip - 压缩文件

## 1. 命令简介
gzip 命令用于压缩文件，是 Linux 系统中常用的压缩备份命令之一。

## 2. 命令语法
```bash
gzip [选项] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -c | 将压缩数据输出到标准输出 |
| -d | 解压缩文件 |
| -f | 强制压缩文件，覆盖已存在的文件 |
| -k | 保留原始文件 |
| -l | 显示压缩文件的信息 |
| -r | 递归压缩目录中的文件 |
| -t | 测试压缩文件的完整性 |
| -v | 显示详细信息 |
| -1 | 最快压缩速度 |
| -9 | 最高压缩率 |

## 4. 实战示例

### 4.1 压缩文件
```bash
gzip file.txt
```

### 4.2 解压缩文件
```bash
gzip -d file.txt.gz
```

### 4.3 保留原始文件
```bash
gzip -k file.txt
```

### 4.4 显示压缩文件的信息
```bash
gzip -l file.txt.gz
```

### 4.5 递归压缩目录中的文件
```bash
gzip -r directory
```

### 4.6 测试压缩文件的完整性
```bash
gzip -t file.txt.gz
```

### 4.7 最快压缩速度
```bash
gzip -1 file.txt
```

### 4.8 最高压缩率
```bash
gzip -9 file.txt
```

## 5. 注意事项
- gzip 命令会将文件压缩为 .gz 格式
- gzip 命令默认会删除原始文件，使用 -k 选项可以保留原始文件
- gzip 命令不能压缩目录，只能压缩文件
- 使用 gunzip 命令可以解压缩 .gz 文件
