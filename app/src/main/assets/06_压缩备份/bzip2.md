# bzip2 - 使用bzip2压缩

## 1. 命令简介
bzip2 命令用于使用bzip2算法压缩文件，是 Linux 系统中常用的压缩备份命令之一。

## 2. 命令语法
```bash
bzip2 [选项] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -c | 将压缩数据输出到标准输出 |
| -d | 解压缩文件 |
| -f | 强制压缩文件，覆盖已存在的文件 |
| -k | 保留原始文件 |
| -l | 显示压缩文件的信息 |
| -t | 测试压缩文件的完整性 |
| -v | 显示详细信息 |
| -1 | 最快压缩速度 |
| -9 | 最高压缩率 |

## 4. 实战示例

### 4.1 压缩文件
```bash
bzip2 file.txt
```

### 4.2 解压缩文件
```bash
bzip2 -d file.txt.bz2
```

### 4.3 保留原始文件
```bash
bzip2 -k file.txt
```

### 4.4 显示压缩文件的信息
```bash
bzip2 -l file.txt.bz2
```

### 4.5 测试压缩文件的完整性
```bash
bzip2 -t file.txt.bz2
```

### 4.6 最快压缩速度
```bash
bzip2 -1 file.txt
```

### 4.7 最高压缩率
```bash
bzip2 -9 file.txt
```

## 5. 注意事项
- bzip2 命令会将文件压缩为 .bz2 格式
- bzip2 命令默认会删除原始文件，使用 -k 选项可以保留原始文件
- bzip2 命令的压缩率通常比 gzip 命令高，但压缩速度较慢
- 使用 bunzip2 命令可以解压缩 .bz2 文件
