# passwd - 修改用户密码

## 1. 命令简介
passwd 命令用于修改用户密码，是 Linux 系统中常用的用户和组管理命令之一。

## 2. 命令语法
```bash
passwd [选项] [用户名]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -d | 删除密码 |
| -l | 锁定用户 |
| -u | 解锁用户 |
| -e | 强制用户下次登录时修改密码 |
| -n | 设置密码最小使用期限 |
| -x | 设置密码最大使用期限 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 修改当前用户密码
```bash
passwd
```

### 4.2 修改指定用户密码
```bash
passwd username
```

### 4.3 锁定用户
```bash
passwd -l username
```

### 4.4 解锁用户
```bash
passwd -u username
```

## 5. 注意事项
- passwd 命令用于修改用户密码
- 修改其他用户密码需要 root 权限
- passwd 命令是一个常用的系统管理工具，用于密码管理
