# grpck - 检查组文件

## 1. 命令简介
grpck 命令用于检查组文件的完整性，是 Linux 系统中常用的用户和组管理命令之一。

## 2. 命令语法
```bash
grpck [选项] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -r | 只读模式 |
| -s | 排序组文件 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 检查组文件
```bash
grpck
```

### 4.2 只读模式检查
```bash
grpck -r
```

### 4.3 排序组文件
```bash
grpck -s
```

## 5. 注意事项
- grpck 命令用于检查组文件的完整性，确保组信息的一致性
- grpck 命令需要 root 权限
- grpck 命令是一个常用的系统管理工具，用于组文件管理
