# id - 显示用户和组信息

## 1. 命令简介
id 命令用于显示用户和组信息，是 Linux 系统中常用的用户和组管理命令之一。

## 2. 命令语法
```bash
id [选项] [用户名]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -u | 显示用户ID |
| -g | 显示组ID |
| -G | 显示所有组ID |
| -n | 显示名称而不是ID |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 显示当前用户信息
```bash
id
```

### 4.2 显示指定用户信息
```bash
id username
```

### 4.3 显示用户ID
```bash
id -u username
```

### 4.4 显示所有组ID
```bash
id -G username
```

## 5. 注意事项
- id 命令用于显示用户和组信息
- id 命令不需要 root 权限
- id 命令是一个常用的系统管理工具，用于查看用户和组信息
