# chage - 修改用户密码过期信息

## 1. 命令简介
chage 命令用于修改用户密码过期信息，是 Linux 系统中常用的用户和组管理命令之一。

## 2. 命令语法
```bash
chage [选项] [用户名]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -l | 显示密码过期信息 |
| -d | 设置密码最后修改日期 |
| -m | 设置密码最小使用期限 |
| -M | 设置密码最大使用期限 |
| -W | 设置密码过期前警告天数 |
| -I | 设置密码过期后锁定天数 |
| -E | 设置账户过期日期 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 显示密码过期信息
```bash
chage -l username
```

### 4.2 设置密码最大使用期限
```bash
chage -M 90 username
```

### 4.3 设置密码过期前警告天数
```bash
chage -W 7 username
```

### 4.4 设置账户过期日期
```bash
chage -E 2024-12-31 username
```

## 5. 注意事项
- chage 命令用于修改用户密码过期信息
- chage 命令需要 root 权限
- chage 命令是一个常用的系统管理工具，用于密码策略管理
