# su - 切换用户

## 1. 命令简介
su 命令用于切换用户，是 Linux 系统中常用的用户和组管理命令之一。

## 2. 命令语法
```bash
su [选项] [用户名]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| - | 登录shell |
| -c | 执行命令 |
| -l | 登录shell |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 切换到root用户
```bash
su
```

### 4.2 切换到指定用户
```bash
su username
```

### 4.3 登录shell
```bash
su -
```

### 4.4 执行命令
```bash
su -c "command" username
```

## 5. 注意事项
- su 命令用于切换用户，临时改变用户身份
- 切换到root用户需要输入root密码
- su 命令是一个常用的系统管理工具，用于用户切换
