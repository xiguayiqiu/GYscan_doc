# sudo - 以超级用户权限执行命令

## 1. 命令简介
sudo 命令用于以超级用户权限执行命令，是 Linux 系统中常用的安全相关命令之一。

## 2. 命令语法
```bash
sudo [选项] [命令]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -i | 以root身份登录 |
| -l | 显示权限 |
| -u | 指定用户 |
| -k | 清除时间戳 |
| -v | 验证时间戳 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 以超级用户权限执行命令
```bash
sudo command
```

### 4.2 以root身份登录
```bash
sudo -i
```

### 4.3 显示权限
```bash
sudo -l
```

### 4.4 以指定用户身份执行命令
```bash
sudo -u user command
```

## 5. 注意事项
- sudo 命令用于以超级用户权限执行命令，提高系统安全性
- sudo 命令需要输入用户密码进行验证
- sudo 命令是一个常用的安全工具，用于权限管理
