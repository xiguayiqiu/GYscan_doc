# su - 切换用户

## 1. 命令简介
su 命令用于切换用户，是 Linux 系统中常用的安全相关命令之一。

## 2. 命令语法
```bash
su [选项] [用户]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -c | 执行命令 |
| -l | 登录shell |
| -s | 指定shell |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 切换到root用户
```bash
su
```

### 4.2 切换到指定用户
```bash
su user
```

### 4.3 执行命令
```bash
su -c "command"
```

### 4.4 登录shell
```bash
su -l user
```

## 5. 注意事项
- su 命令用于切换用户，默认切换到root用户
- su 命令需要输入目标用户的密码
- su 命令是一个常用的安全工具，用于用户切换
