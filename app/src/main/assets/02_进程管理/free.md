# free - 显示内存使用情况

## 1. 命令简介
free 命令用于显示系统内存使用情况，是 Linux 系统中常用的进程管理命令之一。

## 2. 命令语法
```bash
free [选项]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -b | 以字节为单位显示 |
| -k | 以千字节为单位显示 |
| -m | 以兆字节为单位显示 |
| -g | 以吉字节为单位显示 |
| -h | 以人类可读的格式显示 |
| -t | 显示总计 |
| -s | 持续显示 |

## 4. 实战示例

### 4.1 显示内存使用情况
```bash
free
```

### 4.2 以人类可读的格式显示
```bash
free -h
```

### 4.3 显示总计
```bash
free -t
```

### 4.4 持续显示
```bash
free -s 1
```

## 5. 注意事项
- free 命令用于显示系统内存使用情况，包括物理内存和交换空间
- 使用不同的选项可以以不同的单位显示内存使用情况
- free 命令是一个常用的系统监控工具，用于查看内存使用状态
