# pgrep - 按名称查找进程

## 1. 命令简介
pgrep 命令用于按名称查找进程，是 Linux 系统中常用的进程管理命令之一。

## 2. 命令语法
```bash
pgrep [选项] 进程名
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -f | 匹配整个命令行 |
| -i | 忽略大小写 |
| -l | 显示进程名 |
| -n | 显示进程ID和进程名 |
| -o | 只显示最旧的进程 |
| -n | 只显示最新的进程 |
| -P | 只显示指定父进程的子进程 |
| -u | 只显示指定用户的进程 |
| -x | 精确匹配进程名 |

## 4. 实战示例

### 4.1 查找指定名称的进程
```bash
pgrep nginx
```

### 4.2 显示进程名
```bash
pgrep -l nginx
```

### 4.3 忽略大小写
```bash
pgrep -i nginx
```

### 4.4 匹配整个命令行
```bash
pgrep -f "nginx -g 'daemon off;'"
```

### 4.5 只显示指定用户的进程
```bash
pgrep -u user nginx
```

### 4.6 精确匹配进程名
```bash
pgrep -x nginx
```

## 5. 注意事项
- pgrep 命令根据进程名查找进程，而不是进程ID
- 使用 -f 选项可以匹配整个命令行，而不仅仅是进程名
- pgrep 命令默认只显示进程ID，可以使用 -l 选项显示进程名
