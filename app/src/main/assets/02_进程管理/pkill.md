# pkill - 按名称终止进程

## 1. 命令简介
pkill 命令用于按名称终止进程，是 Linux 系统中常用的进程管理命令之一。

## 2. 命令语法
```bash
pkill [选项] 进程名
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -f | 匹配整个命令行 |
| -i | 忽略大小写 |
| -n | 只匹配最新的进程 |
| -o | 只匹配最旧的进程 |
| -P | 只匹配指定父进程的子进程 |
| -u | 只匹配指定用户的进程 |
| -x | 精确匹配进程名 |

## 4. 实战示例

### 4.1 终止指定名称的进程
```bash
pkill nginx
```

### 4.2 忽略大小写
```bash
pkill -i nginx
```

### 4.3 匹配整个命令行
```bash
pkill -f "nginx -g 'daemon off;'"
```

### 4.4 只匹配指定用户的进程
```bash
pkill -u user nginx
```

### 4.5 精确匹配进程名
```bash
pkill -x nginx
```

## 5. 注意事项
- pkill 命令根据进程名终止进程，而不是进程ID
- 使用 -f 选项可以匹配整个命令行，而不仅仅是进程名
- pkill 命令默认发送 SIGTERM 信号，可以使用 -信号选项发送其他信号
