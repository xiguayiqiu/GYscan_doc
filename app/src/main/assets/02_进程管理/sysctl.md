# sysctl - 控制系统参数

## 1. 命令简介
sysctl 命令用于控制系统参数，是 Linux 系统中常用的进程管理命令之一。

## 2. 命令语法
```bash
sysctl [选项] [参数]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -a | 显示所有系统参数 |
| -w | 写入系统参数 |
| -p | 从配置文件加载参数 |
| -n | 只显示值 |
| -e | 忽略错误 |

## 4. 实战示例

### 4.1 显示所有系统参数
```bash
sysctl -a
```

### 4.2 显示指定系统参数
```bash
sysctl kernel.version
```

### 4.3 写入系统参数
```bash
sysctl -w kernel.sysrq=1
```

### 4.4 从配置文件加载参数
```bash
sysctl -p
```

## 5. 注意事项
- sysctl 命令用于控制系统参数，如内核参数、网络参数等
- 系统参数通常存储在 /etc/sysctl.conf 文件中
- 修改系统参数需要 root 权限
- sysctl 命令是一个常用的系统管理工具，用于调整系统性能
