# uptime - 显示系统运行时间

## 1. 命令简介
uptime 命令用于显示系统运行时间和系统负载，是 Linux 系统中常用的系统监控命令之一。

## 2. 命令语法
```bash
uptime [选项]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -p | 以漂亮的格式显示运行时间 |
| -s | 显示系统启动时间 |
| -h | 显示帮助信息 |

## 4. 实战示例

### 4.1 显示系统运行时间和负载
```bash
uptime
```

### 4.2 以漂亮的格式显示运行时间
```bash
uptime -p
```

### 4.3 显示系统启动时间
```bash
uptime -s
```

## 5. 注意事项
- uptime 命令显示的信息包括系统运行时间、当前时间、登录用户数和系统负载
- 系统负载是指系统在过去1分钟、5分钟和15分钟的平均负载
- 负载值越低，系统越空闲；负载值越高，系统越繁忙
