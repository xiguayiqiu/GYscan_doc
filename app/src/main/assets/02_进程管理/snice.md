# snice - 调整进程优先级

## 1. 命令简介
snice 命令用于调整进程的优先级，是 Linux 系统中常用的进程管理命令之一。

## 2. 命令语法
```bash
snice [选项] 进程ID
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -n | 设置优先级值 |

## 4. 优先级值
| 优先级 | 说明 |
|--------|------|
| -20 | 最高优先级 |
| 0 | 默认优先级 |
| 19 | 最低优先级 |

## 5. 实战示例

### 5.1 调整进程优先级
```bash
snice -n 10 1234
```

### 5.2 提高进程优先级
```bash
snice -n -5 1234
```

### 5.3 降低进程优先级
```bash
snice -n 15 1234
```

## 6. 注意事项
- snice 命令需要 root 权限才能设置负的优先级值
- 优先级值越低，进程获得的CPU时间越多
- 优先级值越高，进程获得的CPU时间越少
