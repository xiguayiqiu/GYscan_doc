# top - 实时显示系统进程

## 1. 命令简介
top 命令用于实时显示系统中的进程状态和系统资源使用情况，是 Linux 系统中最常用的系统监控命令之一。

## 2. 命令语法
```bash
top [选项]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -d | 指定刷新间隔（秒） |
| -p | 显示指定进程ID的进程 |
| -u | 显示指定用户的进程 |
| -b | 批处理模式 |
| -n | 指定显示的次数 |
| -H | 显示线程 |

## 4. 交互式命令
| 命令 | 说明 |
|------|------|
| h | 显示帮助 |
| q | 退出 |
| k | 终止进程 |
| r | 重新设置进程优先级 |
| f | 选择显示的字段 |
| M | 按内存使用排序 |
| P | 按CPU使用排序 |
| T | 按时间排序 |
| 1 | 显示所有CPU核心 |

## 5. 实战示例

### 5.1 启动top命令
```bash
top
```

### 5.2 指定刷新间隔
```bash
top -d 1
```

### 5.3 显示指定用户的进程
```bash
top -u user
```

### 5.4 批处理模式
```bash
top -b -n 5
```

### 5.5 显示线程
```bash
top -H
```

## 6. 注意事项
- top 命令默认按CPU使用率排序
- 按 q 键可以退出 top 命令
- 按 M 键可以按内存使用率排序
- 按 P 键可以按CPU使用率排序
- 按 1 键可以显示所有CPU核心的使用情况
