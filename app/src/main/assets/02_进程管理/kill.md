# kill - 终止进程

## 1. 命令简介
kill 命令用于向进程发送信号，最常用的是终止进程，是 Linux 系统中最常用的进程管理命令之一。

## 2. 命令语法
```bash
kill [选项] 进程ID
```

## 3. 常用信号
| 信号 | 数值 | 说明 |
|------|------|------|
| SIGHUP | 1 | 挂起信号 |
| SIGINT | 2 | 中断信号（Ctrl+C） |
| SIGKILL | 9 | 强制终止信号 |
| SIGTERM | 15 | 终止信号（默认） |
| SIGSTOP | 19 | 暂停信号 |
| SIGCONT | 18 | 继续信号 |

## 4. 常用选项
| 选项 | 说明 |
|------|------|
| -s | 指定信号 |
| -l | 显示所有信号 |
| -9 | 发送 SIGKILL 信号 |

## 5. 实战示例

### 5.1 终止进程
```bash
kill 1234
```

### 5.2 强制终止进程
```bash
kill -9 1234
kill -s SIGKILL 1234
```

### 5.3 发送中断信号
```bash
kill -2 1234
kill -s SIGINT 1234
```

### 5.4 显示所有信号
```bash
kill -l
```

### 5.5 终止多个进程
```bash
kill 1234 5678 9012
```

## 6. 注意事项
- kill 命令默认发送 SIGTERM 信号，进程可以捕获并处理这个信号
- 使用 -9 选项发送 SIGKILL 信号，进程无法捕获，会被强制终止
- 要终止进程，需要有足够的权限，通常需要是进程的所有者或 root 用户
