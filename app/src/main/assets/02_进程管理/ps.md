# ps - 显示进程状态

## 1. 命令简介
ps 命令用于显示系统中的进程状态，是 Linux 系统中最常用的进程管理命令之一。

## 2. 命令语法
```bash
ps [选项]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -e | 显示所有进程 |
| -f | 显示完整格式 |
| -l | 显示长格式 |
| -a | 显示所有终端进程 |
| -u | 按用户显示进程 |
| -x | 显示无终端进程 |
| -p | 显示指定进程ID的进程 |
| -t | 显示指定终端的进程 |
| --forest | 显示进程树 |

## 4. 实战示例

### 4.1 显示所有进程
```bash
ps -e
```

### 4.2 显示完整格式
```bash
ps -ef
```

### 4.3 显示长格式
```bash
ps -el
```

### 4.4 按用户显示进程
```bash
ps -u user
```

### 4.5 显示进程树
```bash
ps --forest
```

### 4.6 显示指定进程ID的进程
```bash
ps -p 1234
```

## 5. 注意事项
- ps 命令默认只显示当前终端的进程
- 使用 -e 选项可以显示所有进程
- 使用 -f 选项可以显示更详细的进程信息，包括进程ID、父进程ID、用户、启动时间等
