# skill - 发送信号给进程

## 1. 命令简介
skill 命令用于向进程发送信号，是 Linux 系统中常用的进程管理命令之一。

## 2. 命令语法
```bash
skill [选项] 进程名
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -f | 强制发送信号 |
| -i | 交互式操作 |
| -v | 显示详细信息 |
| -w | 等待进程终止 |

## 4. 常用信号
| 信号 | 数值 | 说明 |
|------|------|------|
| SIGHUP | 1 | 挂起信号 |
| SIGINT | 2 | 中断信号（Ctrl+C） |
| SIGKILL | 9 | 强制终止信号 |
| SIGTERM | 15 | 终止信号（默认） |

## 5. 实战示例

### 5.1 向进程发送终止信号
```bash
skill nginx
```

### 5.2 向进程发送强制终止信号
```bash
skill -9 nginx
```

### 5.3 交互式操作
```bash
skill -i nginx
```

### 5.4 显示详细信息
```bash
skill -v nginx
```

## 6. 注意事项
- skill 命令根据进程名向进程发送信号
- skill 命令默认发送 SIGTERM 信号
- skill 命令在某些系统中可能已经被 pkill 命令替代
