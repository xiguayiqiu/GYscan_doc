# whereis - 查找命令路径

## 1. 命令简介
whereis 命令用于查找命令的可执行文件、源代码和手册页路径，是 Linux 系统中常用的其他命令之一。

## 2. 命令语法
```bash
whereis [选项] [命令]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -b | 只查找可执行文件 |
| -m | 只查找手册页 |
| -s | 只查找源代码 |
| -u | 显示不寻常的条目 |
| -l | 显示搜索路径 |

## 4. 实战示例

### 4.1 查找命令的所有相关文件
```bash
whereis ls
```

### 4.2 只查找可执行文件
```bash
whereis -b ls
```

### 4.3 只查找手册页
```bash
whereis -m ls
```

### 4.4 只查找源代码
```bash
whereis -s ls
```

## 5. 注意事项
- whereis 命令用于查找命令的可执行文件、源代码和手册页
- whereis 命令通过搜索特定的系统目录来查找文件
- whereis 命令的搜索速度比 find 命令快，但搜索范围有限
- whereis 命令是一个实用工具，用于快速定位命令的相关文件
