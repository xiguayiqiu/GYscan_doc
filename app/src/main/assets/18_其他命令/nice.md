# nice - 调整进程优先级

## 1. 命令简介
nice 命令用于调整进程的优先级，是 Linux 系统中常用的其他命令之一。

## 2. 命令语法
```bash
nice [选项] [命令]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -n | 指定优先级值（-20到19） |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 优先级值
- 优先级值范围：-20到19
- 数值越小，优先级越高
- 默认为0

## 5. 实战示例

### 5.1 以默认优先级执行命令
```bash
nice command
```

### 5.2 以更高优先级执行命令
```bash
nice -n -10 command
```

### 5.3 以更低优先级执行命令
```bash
nice -n 10 command
```

## 6. 注意事项
- nice 命令用于调整进程的优先级
- 只有 root 用户可以设置负的优先级值
- 优先级值范围是-20到19，数值越小优先级越高
- nice 命令是一个常用的系统工具，用于管理进程优先级
