# fg - 将作业放到前台

## 1. 命令简介
fg 命令用于将后台作业放到前台运行，是 Linux 系统中常用的其他命令之一。

## 2. 命令语法
```bash
fg [作业ID]
```

## 3. 实战示例

### 3.1 将最近的后台作业放到前台
```bash
fg
```

### 3.2 将指定作业放到前台
```bash
fg %1
```

## 4. 退出状态
- 0：命令执行成功
- 非0：命令执行失败

## 5. 注意事项
- fg 命令用于将后台作业放到前台运行
- 使用 jobs 命令可以查看作业列表
- fg 命令是一个Shell内置命令，用于管理作业
