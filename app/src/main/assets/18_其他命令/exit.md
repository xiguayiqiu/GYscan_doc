# exit - 退出Shell

## 1. 命令简介
exit 命令用于退出当前 Shell 会话，是 Linux 系统中常用的其他命令之一。

## 2. 命令语法
```bash
exit [状态码]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| 状态码 | 退出状态码，默认为0 |

## 4. 实战示例

### 4.1 退出Shell
```bash
exit
```

### 4.2 带状态码退出Shell
```bash
exit 1
```

## 5. 注意事项
- exit 命令用于退出当前 Shell 会话
- 可以指定退出状态码，默认为0表示成功
- 在脚本中，exit 命令用于结束脚本执行并返回状态码
- 退出状态码可以被其他命令或脚本捕获和使用
