# time - 测量命令执行时间

## 1. 命令简介
time 命令用于测量命令执行的时间，是 Linux 系统中常用的其他命令之一。

## 2. 命令语法
```bash
time [选项] [命令]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -p | 以POSIX格式显示 |
| -v | 显示详细信息 |

## 4. 实战示例

### 4.1 测量命令执行时间
```bash
time ls -la
```

### 4.2 以POSIX格式显示
```bash
time -p ls -la
```

### 4.3 显示详细信息
```bash
time -v ls -la
```

## 5. 注意事项
- time 命令用于测量命令执行的时间
- time 命令会显示命令的实际时间、用户时间和系统时间
- time 命令是一个常用的工具，用于性能测试和优化
