# yes - 重复输出字符串

## 1. 命令简介
yes 命令用于重复输出指定的字符串，是 Linux 系统中常用的其他命令之一。

## 2. 命令语法
```bash
yes [字符串]
```

## 3. 实战示例

### 3.1 重复输出"y"
```bash
yes
```

### 3.2 重复输出指定字符串
```bash
yes hello
```

### 3.3 与其他命令配合使用
```bash
yes | rm -i file*
```

## 4. 退出状态
- 0：命令执行成功
- 非0：命令执行失败

## 5. 注意事项
- yes 命令用于重复输出指定的字符串，默认输出"y"
- yes 命令会一直输出，直到被中断
- yes 命令常用于需要批量确认的场景
- 使用 Ctrl+C 可以中断 yes 命令的执行
