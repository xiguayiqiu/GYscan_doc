# jobs - 显示后台作业

## 1. 命令简介
jobs 命令用于显示当前Shell中的后台作业，是 Linux 系统中常用的其他命令之一。

## 2. 命令语法
```bash
jobs [选项]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -l | 显示作业的进程ID |
| -p | 只显示作业的进程ID |
| -n | 只显示上次通知后状态发生变化的作业 |

## 4. 实战示例

### 4.1 显示后台作业
```bash
jobs
```

### 4.2 显示作业的进程ID
```bash
jobs -l
```

### 4.3 只显示作业的进程ID
```bash
jobs -p
```

### 4.4 只显示状态发生变化的作业
```bash
jobs -n
```

## 5. 注意事项
- jobs 命令用于显示当前Shell中的后台作业
- 作业是指在后台运行的命令
- 使用 & 可以将命令放到后台运行
- jobs 命令是一个Shell内置命令，用于管理后台作业
