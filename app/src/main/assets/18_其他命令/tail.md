# tail - 显示文件结尾

## 1. 命令简介
tail 命令用于显示文件的结尾部分，是 Linux 系统中常用的其他命令之一。

## 2. 命令语法
```bash
tail [选项] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -n | 显示指定行数 |
| -c | 显示指定字节数 |
| -f | 实时跟踪文件变化 |
| -q | 静默模式，不显示文件名 |
| -v | 显示文件名 |

## 4. 实战示例

### 4.1 显示文件的最后10行
```bash
tail file.txt
```

### 4.2 显示文件的最后5行
```bash
tail -n 5 file.txt
```

### 4.3 实时跟踪文件变化
```bash
tail -f file.txt
```

### 4.4 显示文件的最后100字节
```bash
tail -c 100 file.txt
```

## 5. 注意事项
- tail 命令用于显示文件的结尾部分，默认显示最后10行
- 使用 -n 选项可以指定显示的行数
- 使用 -c 选项可以指定显示的字节数
- 使用 -f 选项可以实时跟踪文件变化，适合查看日志文件
- tail 命令是一个常用的文本处理工具，用于查看文件的结尾内容
