# paste - 合并文本行

## 1. 命令简介
paste 命令用于合并文本行，是 Linux 系统中常用的其他命令之一。

## 2. 命令语法
```bash
paste [选项] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -d | 指定分隔符 |
| -s | 串行合并，将一个文件的所有行合并为一行 |

## 4. 实战示例

### 4.1 合并两个文件的行
```bash
paste file1.txt file2.txt
```

### 4.2 指定分隔符
```bash
paste -d "," file1.txt file2.txt
```

### 4.3 串行合并
```bash
paste -s file.txt
```

### 4.4 串行合并并指定分隔符
```bash
paste -s -d "," file.txt
```

## 5. 注意事项
- paste 命令用于合并文本行，默认使用制表符作为分隔符
- 使用 -d 选项可以指定自定义分隔符
- 使用 -s 选项可以将一个文件的所有行合并为一行
- paste 命令是一个常用的文本处理工具，用于合并多个文件的内容
