# which - 查找命令路径

## 1. 命令简介
which 命令用于查找命令的可执行文件路径，是 Linux 系统中常用的其他命令之一。

## 2. 命令语法
```bash
which [选项] [命令]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -a | 显示所有匹配的可执行文件路径 |
| -p | 显示完整路径 |
| -s | 静默模式，只返回退出状态 |

## 4. 实战示例

### 4.1 查找命令路径
```bash
which ls
```

### 4.2 显示所有匹配的可执行文件路径
```bash
which -a ls
```

### 4.3 查找多个命令的路径
```bash
which ls cat grep
```

## 5. 注意事项
- which 命令通过搜索 PATH 环境变量指定的目录来查找命令
- which 命令只查找可执行文件，不查找内置命令和别名
- 使用 -a 选项可以显示所有匹配的可执行文件路径
- which 命令是一个简单但实用的工具，用于确定命令的具体位置
