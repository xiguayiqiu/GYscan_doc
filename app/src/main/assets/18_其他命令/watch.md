# watch - 定期执行命令

## 1. 命令简介
watch 命令用于定期执行命令并显示结果，是 Linux 系统中常用的其他命令之一。

## 2. 命令语法
```bash
watch [选项] [命令]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -n | 指定执行间隔（秒） |
| -d | 高亮显示变化的部分 |
| -t | 不显示标题 |
| -g | 当命令输出变化时退出 |

## 4. 实战示例

### 4.1 每隔2秒执行一次命令
```bash
watch -n 2 ls -la
```

### 4.2 高亮显示变化的部分
```bash
watch -d ls -la
```

### 4.3 不显示标题
```bash
watch -t ls -la
```

### 4.4 当命令输出变化时退出
```bash
watch -g ls -la
```

## 5. 注意事项
- watch 命令用于定期执行命令并显示结果
- 默认每隔2秒执行一次命令
- 使用 -n 选项可以指定执行间隔
- watch 命令是一个常用的系统工具，用于监控命令输出的变化
