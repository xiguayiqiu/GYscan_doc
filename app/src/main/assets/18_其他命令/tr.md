# tr - 转换或删除字符

## 1. 命令简介
tr 命令用于转换或删除字符，是 Linux 系统中常用的其他命令之一。

## 2. 命令语法
```bash
tr [选项] [字符集1] [字符集2]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -c | 补集，使用字符集1的补集 |
| -d | 删除字符 |
| -s | 压缩重复字符 |
| -t | 截断字符集1到字符集2的长度 |

## 4. 实战示例

### 4.1 转换字符
```bash
echo "hello" | tr "a-z" "A-Z"
```

### 4.2 删除字符
```bash
echo "hello 123" | tr -d "0-9"
```

### 4.3 压缩重复字符
```bash
echo "heeeelllo" | tr -s "e"
```

### 4.4 补集转换
```bash
echo "hello 123" | tr -c "a-z" "_"
```

## 5. 注意事项
- tr 命令用于转换或删除字符
- tr 命令只能从标准输入读取数据
- 使用不同的选项可以实现不同的字符处理功能
- tr 命令是一个常用的文本处理工具，用于字符转换和删除
