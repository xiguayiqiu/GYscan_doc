# wc - 统计文件行数、字数、字节数

## 1. 命令简介
wc 命令用于统计文件的行数、字数和字节数，是 Linux 系统中常用的其他命令之一。

## 2. 命令语法
```bash
wc [选项] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -l | 只统计行数 |
| -w | 只统计字数 |
| -c | 只统计字节数 |
| -m | 只统计字符数 |
| -L | 显示最长行的长度 |

## 4. 实战示例

### 4.1 统计文件的行数、字数和字节数
```bash
wc file.txt
```

### 4.2 只统计行数
```bash
wc -l file.txt
```

### 4.3 只统计字数
```bash
wc -w file.txt
```

### 4.4 只统计字节数
```bash
wc -c file.txt
```

### 4.5 显示最长行的长度
```bash
wc -L file.txt
```

## 5. 注意事项
- wc 命令用于统计文件的行数、字数和字节数
- 使用不同的选项可以只统计特定的内容
- wc 命令是一个常用的文本处理工具，用于统计文件的基本信息
