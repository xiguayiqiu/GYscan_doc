# wait - 等待进程完成

## 1. 命令简介
wait 命令用于等待指定进程完成，是 Linux 系统中常用的其他命令之一。

## 2. 命令语法
```bash
wait [进程ID]
```

## 3. 实战示例

### 3.1 等待所有后台进程完成
```bash
wait
```

### 3.2 等待指定进程完成
```bash
wait 1234
```

## 4. 退出状态
- 0：命令执行成功
- 非0：命令执行失败

## 5. 注意事项
- wait 命令用于等待指定进程完成
- 如果不指定进程ID，wait 命令会等待所有后台进程完成
- wait 命令是一个Shell内置命令，用于管理进程
