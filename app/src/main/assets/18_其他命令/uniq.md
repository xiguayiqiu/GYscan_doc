# uniq - 去除重复行

## 1. 命令简介
uniq 命令用于去除文件中的重复行，是 Linux 系统中常用的其他命令之一。

## 2. 命令语法
```bash
uniq [选项] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -c | 显示每行出现的次数 |
| -d | 只显示重复的行 |
| -u | 只显示不重复的行 |
| -i | 忽略大小写 |
| -f | 忽略前指定列 |
| -s | 忽略前指定字符数 |

## 4. 实战示例

### 4.1 去除重复行
```bash
sort file.txt | uniq
```

### 4.2 显示每行出现的次数
```bash
sort file.txt | uniq -c
```

### 4.3 只显示重复的行
```bash
sort file.txt | uniq -d
```

### 4.4 只显示不重复的行
```bash
sort file.txt | uniq -u
```

## 5. 注意事项
- uniq 命令用于去除文件中的重复行，但只去除相邻的重复行
- 通常需要与 sort 命令配合使用，先排序再去重
- 使用 -c 选项可以显示每行出现的次数
- uniq 命令是一个常用的文本处理工具，用于去重和统计
