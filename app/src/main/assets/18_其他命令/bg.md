# bg - 将作业放到后台

## 1. 命令简介
bg 命令用于将作业放到后台运行，是 Linux 系统中常用的其他命令之一。

## 2. 命令语法
```bash
bg [作业ID]
```

## 3. 实战示例

### 3.1 将最近的暂停作业放到后台
```bash
bg
```

### 3.2 将指定作业放到后台
```bash
bg %1
```

## 4. 退出状态
- 0：命令执行成功
- 非0：命令执行失败

## 5. 注意事项
- bg 命令用于将作业放到后台运行
- 作业必须是已经暂停的作业
- 使用 jobs 命令可以查看作业列表
- bg 命令是一个Shell内置命令，用于管理作业
