# vim - Vim文本编辑器

## 1. 命令简介
vim 命令是 Vim 文本编辑器，是 Vi 编辑器的增强版本，是 Linux 系统中常用的文本编辑器之一。

## 2. 命令语法
```bash
vim [选项] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -c | 执行指定的命令 |
| -r | 恢复崩溃的编辑会话 |
| -s | 安静模式 |
| -t | 跳转到指定的标签 |
| -v | 详细模式 |
| -o | 水平分割窗口 |
| -O | 垂直分割窗口 |
| -p | 以标签页形式打开文件 |

## 4. 常用操作

### 4.1 进入编辑模式
- i：在光标前插入
- I：在行首插入
- a：在光标后插入
- A：在行尾插入
- o：在当前行后插入新行
- O：在当前行前插入新行

### 4.2 命令模式操作
- h：左移光标
- j：下移光标
- k：上移光标
- l：右移光标
- w：移到下一个单词
- b：移到上一个单词
- 0：移到行首
- $：移到行尾
- gg：移到文件开头
- G：移到文件结尾
- dd：删除当前行
- yy：复制当前行
- p：粘贴
- u：撤销
- Ctrl+r：重做
- /：搜索
- n：查找下一个
- N：查找上一个
- :w：保存
- :q：退出
- :wq：保存并退出
- :q!：强制退出

### 4.3 Vim特有功能
- :set number：显示行号
- :set tabstop=4：设置制表符宽度为4
- :set expandtab：将制表符转换为空格
- :set autoindent：自动缩进
- :set syntax on：开启语法高亮
- Ctrl+n：自动补全
- Ctrl+w+w：切换窗口
- :tabnew：新建标签页
- :tabnext：切换到下一个标签页
- :tabprev：切换到上一个标签页

## 5. 实战示例

### 5.1 编辑文件
```bash
vim file.txt
```

### 5.2 水平分割窗口
```bash
vim -o file1.txt file2.txt
```

### 5.3 垂直分割窗口
```bash
vim -O file1.txt file2.txt
```

### 5.4 以标签页形式打开文件
```bash
vim -p file1.txt file2.txt
```

## 6. 注意事项
- vim 命令是 Vi 编辑器的增强版本，提供了更多功能
- vim 支持语法高亮、自动补全、多窗口编辑等高级功能
- vim 是一个功能强大的文本编辑器，被广泛用于编程和系统管理
- 学习 vim 的基本操作需要一定的时间，但掌握后会大大提高编辑效率
