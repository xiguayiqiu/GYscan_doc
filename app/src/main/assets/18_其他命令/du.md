# du - 显示目录大小

## 1. 命令简介
du 命令用于显示目录或文件的大小，是 Linux 系统中常用的其他命令之一。

## 2. 命令语法
```bash
du [选项] [文件或目录]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -a | 显示所有文件和目录的大小 |
| -h | 以人类可读的格式显示 |
| -H | 以人类可读的格式显示（使用1000作为基数） |
| -s | 只显示总大小 |
| -c | 显示总计 |
| -d | 指定深度 |
| -k | 以KB为单位显示 |
| -m | 以MB为单位显示 |
| -g | 以GB为单位显示 |

## 4. 实战示例

### 4.1 显示目录大小
```bash
du
```

### 4.2 以人类可读的格式显示
```bash
du -h
```

### 4.3 只显示总大小
```bash
du -s
```

### 4.4 显示所有文件和目录的大小
```bash
du -a
```

### 4.5 显示总计
```bash
du -c
```

### 4.6 指定深度
```bash
du -d 2
```

## 5. 注意事项
- du 命令用于显示目录或文件的大小
- 使用 -h 选项可以以人类可读的格式显示（如GB、MB、KB）
- 使用 -s 选项可以只显示总大小
- 使用 -d 选项可以指定显示的深度
- du 命令是一个常用的系统工具，用于监控目录大小
