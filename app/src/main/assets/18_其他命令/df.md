# df - 显示磁盘使用情况

## 1. 命令简介
df 命令用于显示磁盘使用情况，是 Linux 系统中常用的其他命令之一。

## 2. 命令语法
```bash
df [选项] [文件系统]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -a | 显示所有文件系统 |
| -h | 以人类可读的格式显示 |
| -H | 以人类可读的格式显示（使用1000作为基数） |
| -T | 显示文件系统类型 |
| -t | 只显示指定类型的文件系统 |
| -x | 排除指定类型的文件系统 |
| -i | 显示inode使用情况 |

## 4. 实战示例

### 4.1 显示磁盘使用情况
```bash
df
```

### 4.2 以人类可读的格式显示
```bash
df -h
```

### 4.3 显示文件系统类型
```bash
df -T
```

### 4.4 显示inode使用情况
```bash
df -i
```

### 4.5 只显示指定类型的文件系统
```bash
df -t ext4
```

## 5. 注意事项
- df 命令用于显示文件系统的磁盘使用情况
- 使用 -h 选项可以以人类可读的格式显示（如GB、MB）
- 使用 -T 选项可以显示文件系统类型
- 使用 -i 选项可以显示inode使用情况
- df 命令是一个常用的系统工具，用于监控磁盘使用情况
