# disown - 从作业表中移除作业

## 1. 命令简介
disown 命令用于从作业表中移除作业，是 Linux 系统中常用的其他命令之一。

## 2. 命令语法
```bash
disown [选项] [作业ID]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -h | 标记作业为不被 SIGHUP 信号终止 |
| -a | 移除所有作业 |
| -r | 只移除正在运行的作业 |

## 4. 实战示例

### 4.1 从作业表中移除指定作业
```bash
disown %1
```

### 4.2 标记作业为不被 SIGHUP 信号终止
```bash
disown -h %1
```

### 4.3 移除所有作业
```bash
disown -a
```

### 4.4 只移除正在运行的作业
```bash
disown -r
```

## 5. 注意事项
- disown 命令用于从作业表中移除作业
- 移除的作业仍然在后台运行，但不再受当前Shell的控制
- 使用 -h 选项可以标记作业为不被 SIGHUP 信号终止
- disown 命令是一个常用的Shell内置命令，用于管理后台作业
