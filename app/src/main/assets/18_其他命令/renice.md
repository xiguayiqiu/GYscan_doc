# renice - 调整运行进程的优先级

## 1. 命令简介
renice 命令用于调整运行中进程的优先级，是 Linux 系统中常用的其他命令之一。

## 2. 命令语法
```bash
renice [选项] [优先级值] [进程ID]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -n | 指定优先级值（-20到19） |
| -p | 调整指定进程ID的优先级 |
| -g | 调整指定进程组的优先级 |
| -u | 调整指定用户的进程优先级 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 优先级值
- 优先级值范围：-20到19
- 数值越小，优先级越高

## 5. 实战示例

### 5.1 调整指定进程的优先级
```bash
renice -n 10 1234
```

### 5.2 调整指定用户的进程优先级
```bash
renice -n 5 -u user
```

### 5.3 调整指定进程组的优先级
```bash
renice -n 8 -g 5678
```

## 6. 注意事项
- renice 命令用于调整运行中进程的优先级
- 只有 root 用户可以设置负的优先级值
- 优先级值范围是-20到19，数值越小优先级越高
- renice 命令是一个常用的系统工具，用于管理运行中进程的优先级
