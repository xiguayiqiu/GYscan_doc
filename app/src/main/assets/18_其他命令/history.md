# history - 显示命令历史

## 1. 命令简介
history 命令用于显示命令历史记录，是 Linux 系统中常用的其他命令之一。

## 2. 命令语法
```bash
history [选项]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -c | 清除命令历史 |
| -d | 删除指定的历史记录 |
| -a | 将当前会话的历史记录追加到历史文件 |
| -n | 从历史文件中读取新的历史记录 |
| -r | 读取历史文件并替换当前历史记录 |
| -w | 将当前历史记录写入历史文件 |
| -p | 展开历史记录中的参数 |
| -s | 将指定的命令添加到历史记录 |

## 4. 实战示例

### 4.1 显示命令历史
```bash
history
```

### 4.2 显示最近的10条命令
```bash
history 10
```

### 4.3 清除命令历史
```bash
history -c
```

### 4.4 删除指定的历史记录
```bash
history -d 100
```

### 4.5 将当前会话的历史记录追加到历史文件
```bash
history -a
```

## 5. 注意事项
- history 命令用于显示和管理命令历史记录
- 命令历史记录默认保存在 ~/.bash_history 文件中
- 使用上下箭头可以浏览命令历史
- 使用 ! 可以执行历史记录中的命令，如 !100 执行第100条命令
- 命令历史记录可以提高命令输入效率，避免重复输入相同的命令
