# cut - 剪切文本

## 1. 命令简介
cut 命令用于剪切文本，是 Linux 系统中常用的其他命令之一。

## 2. 命令语法
```bash
cut [选项] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -b | 按字节剪切 |
| -c | 按字符剪切 |
| -f | 按字段剪切 |
| -d | 指定分隔符 |
| -s | 不显示没有分隔符的行 |

## 4. 实战示例

### 4.1 按字段剪切
```bash
cut -f 1 file.txt
```

### 4.2 指定分隔符
```bash
cut -d "," -f 1,3 file.txt
```

### 4.3 按字符剪切
```bash
cut -c 1-5 file.txt
```

### 4.4 按字节剪切
```bash
cut -b 1-5 file.txt
```

## 5. 注意事项
- cut 命令用于剪切文本，支持按字节、字符和字段剪切
- 使用 -f 选项可以按字段剪切，默认分隔符是制表符
- 使用 -d 选项可以指定自定义分隔符
- cut 命令是一个常用的文本处理工具，用于提取文本中的特定部分
