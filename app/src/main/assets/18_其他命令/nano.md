# nano - Nano文本编辑器

## 1. 命令简介
nano 命令是 Nano 文本编辑器，是 Linux 系统中常用的文本编辑器之一，以简单易用著称。

## 2. 命令语法
```bash
nano [选项] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -c | 显示行号 |
| -i | 自动缩进 |
| -t | 启用制表符 |
| -s | 启用软换行 |
| -w | 禁用软换行 |
| -m | 启用鼠标支持 |
| -o | 覆盖模式 |
| -p | 启用持久化配置 |

## 4. 常用操作
| 操作 | 说明 |
|------|------|
| Ctrl+O | 保存文件 |
| Ctrl+X | 退出编辑器 |
| Ctrl+G | 显示帮助 |
| Ctrl+K | 剪切当前行 |
| Ctrl+U | 粘贴 |
| Ctrl+W | 搜索 |
| Ctrl+R | 替换 |
| Ctrl+C | 显示光标位置 |
| Ctrl+A | 移到行首 |
| Ctrl+E | 移到行尾 |
| Ctrl+P | 上一行 |
| Ctrl+N | 下一行 |
| Ctrl+F | 右移 |
| Ctrl+B | 左移 |

## 5. 实战示例

### 5.1 编辑文件
```bash
nano file.txt
```

### 5.2 显示行号
```bash
nano -c file.txt
```

### 5.3 自动缩进
```bash
nano -i file.txt
```

### 5.4 保存并退出
```bash
# 按 Ctrl+O 保存，然后按 Enter 确认，最后按 Ctrl+X 退出
```

## 6. 注意事项
- nano 命令是一个简单易用的文本编辑器，适合初学者使用
- nano 编辑器在底部显示常用操作的快捷键，方便用户使用
- nano 支持基本的编辑功能，如剪切、粘贴、搜索、替换等
- nano 是许多 Linux 发行版的默认文本编辑器之一
