# alias - 创建命令别名

## 1. 命令简介
alias 命令用于创建命令别名，是 Linux 系统中常用的其他命令之一。

## 2. 命令语法
```bash
alias [选项] [别名=命令]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -p | 显示所有别名 |

## 4. 实战示例

### 4.1 创建命令别名
```bash
alias ll='ls -la'
```

### 4.2 显示所有别名
```bash
alias -p
```

### 4.3 移除别名
```bash
unalias ll
```

### 4.4 创建带参数的别名
```bash
alias cd..='cd ..'
```

## 5. 注意事项
- alias 命令创建的别名只在当前会话中有效
- 要使别名永久生效，需要将其添加到 shell 配置文件中（如 ~/.bashrc）
- 使用 unalias 命令可以移除别名
- 别名可以简化复杂命令的输入，提高工作效率
