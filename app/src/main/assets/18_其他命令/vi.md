# vi - Vi文本编辑器

## 1. 命令简介
vi 命令是 Vi 文本编辑器，是 Linux 系统中常用的文本编辑器之一。

## 2. 命令语法
```bash
vi [选项] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -c | 执行指定的命令 |
| -r | 恢复崩溃的编辑会话 |
| -s | 安静模式 |
| -t | 跳转到指定的标签 |
| -v | 详细模式 |

## 4. 常用操作

### 4.1 进入编辑模式
- i：在光标前插入
- I：在行首插入
- a：在光标后插入
- A：在行尾插入
- o：在当前行后插入新行
- O：在当前行前插入新行

### 4.2 命令模式操作
- h：左移光标
- j：下移光标
- k：上移光标
- l：右移光标
- w：移到下一个单词
- b：移到上一个单词
- 0：移到行首
- $：移到行尾
- gg：移到文件开头
- G：移到文件结尾
- dd：删除当前行
- yy：复制当前行
- p：粘贴
- u：撤销
- Ctrl+r：重做
- /：搜索
- n：查找下一个
- N：查找上一个
- :w：保存
- :q：退出
- :wq：保存并退出
- :q!：强制退出

## 5. 实战示例

### 5.1 编辑文件
```bash
vi file.txt
```

### 5.2 进入编辑模式
```bash
# 按 i 键进入插入模式
```

### 5.3 保存并退出
```bash
# 按 Esc 键退出编辑模式，然后输入

```

## 6. 注意事项
- vi 命令是 Linux 系统中最常用的文本编辑器之一
- vi 有两种模式：命令模式和编辑模式
- 在命令模式下，可以执行各种编辑命令
- 在编辑模式下，可以输入文本
- 按 Esc 键可以从编辑模式切换到命令模式
- vi 是一个功能强大的文本编辑器，支持各种编辑操作
