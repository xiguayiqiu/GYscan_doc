# sleep - 暂停执行

## 1. 命令简介
sleep 命令用于暂停执行指定的时间，是 Linux 系统中常用的其他命令之一。

## 2. 命令语法
```bash
sleep [时间]
```

## 3. 时间单位
| 单位 | 说明 |
|------|------|
| s | 秒（默认） |
| m | 分钟 |
| h | 小时 |
| d | 天 |

## 4. 实战示例

### 4.1 暂停1秒
```bash
sleep 1
```

### 4.2 暂停1分钟
```bash
sleep 1m
```

### 4.3 暂停1小时
```bash
sleep 1h
```

### 4.4 暂停1天
```bash
sleep 1d
```

## 5. 注意事项
- sleep 命令用于暂停执行指定的时间
- 可以指定不同的时间单位
- sleep 命令是一个常用的命令，用于在脚本中添加延迟
- sleep 命令的退出状态总是0，除非被信号中断
