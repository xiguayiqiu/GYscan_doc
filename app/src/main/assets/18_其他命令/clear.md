# clear - 清屏

## 1. 命令简介
clear 命令用于清除终端屏幕，是 Linux 系统中常用的其他命令之一。

## 2. 命令语法
```bash
clear
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 清屏
```bash
clear
```

## 5. 注意事项
- clear 命令用于清除终端屏幕，使屏幕显示空白
- clear 命令没有参数，直接执行即可
- clear 命令只是将屏幕滚动到新的一页，并没有真正清除历史记录
- 在大多数终端中，使用 Ctrl+L 快捷键也可以达到清屏的效果
