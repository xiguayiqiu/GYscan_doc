# uname - 显示系统信息

## 1. 命令简介
uname 命令用于显示系统信息，是 Linux 系统中常用的其他命令之一。

## 2. 命令语法
```bash
uname [选项]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -a | 显示所有系统信息 |
| -s | 显示内核名称 |
| -n | 显示主机名 |
| -r | 显示内核版本 |
| -v | 显示内核版本详情 |
| -m | 显示机器硬件名称 |
| -p | 显示处理器类型 |
| -i | 显示硬件平台 |
| -o | 显示操作系统名称 |

## 4. 实战示例

### 4.1 显示所有系统信息
```bash
uname -a
```

### 4.2 显示内核名称
```bash
uname -s
```

### 4.3 显示主机名
```bash
uname -n
```

### 4.4 显示内核版本
```bash
uname -r
```

### 4.5 显示机器硬件名称
```bash
uname -m
```

## 5. 注意事项
- uname 命令用于显示系统信息，包括内核名称、主机名、内核版本等
- 使用 -a 选项可以显示所有系统信息
- uname 命令是一个简单但实用的工具，用于了解系统的基本信息
