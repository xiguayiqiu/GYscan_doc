# timeout - 设置命令超时

## 1. 命令简介
timeout 命令用于设置命令的执行超时时间，是 Linux 系统中常用的其他命令之一。

## 2. 命令语法
```bash
timeout [选项] [时间] [命令]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -s | 指定超时后发送的信号 |
| -k | 指定超时后等待的时间 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 时间单位
| 单位 | 说明 |
|------|------|
| s | 秒（默认） |
| m | 分钟 |
| h | 小时 |
| d | 天 |

## 5. 实战示例

### 5.1 设置命令超时时间为5秒
```bash
timeout 5s sleep 10
```

### 5.2 指定超时后发送的信号
```bash
timeout -s SIGINT 5s sleep 10
```

### 5.3 指定超时后等待的时间
```bash
timeout -k 2s 5s sleep 10
```

## 6. 注意事项
- timeout 命令用于设置命令的执行超时时间
- 当命令执行时间超过指定的超时时间时，timeout 命令会终止该命令
- 使用不同的时间单位可以指定不同的超时时间
- timeout 命令是一个常用的工具，用于防止命令执行时间过长
