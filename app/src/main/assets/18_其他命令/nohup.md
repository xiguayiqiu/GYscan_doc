# nohup - 忽略挂起信号运行命令

## 1. 命令简介
nohup 命令用于忽略挂起信号运行命令，是 Linux 系统中常用的其他命令之一。

## 2. 命令语法
```bash
nohup [选项] [命令]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 后台运行命令并忽略挂起信号
```bash
nohup command &
```

### 4.2 将输出重定向到文件
```bash
nohup command > output.log 2>&1 &
```

## 5. 注意事项
- nohup 命令用于忽略挂起信号运行命令
- nohup 命令通常与 & 一起使用，使命令在后台运行
- nohup 命令执行的命令不会因为终端关闭而停止
- nohup 命令的输出默认保存到 nohup.out 文件中
