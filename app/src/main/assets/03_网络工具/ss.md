# ss - 显示套接字统计

## 1. 命令简介
ss 命令用于显示套接字统计，是 Linux 系统中常用的网络工具命令之一。

## 2. 命令语法
```bash
ss [选项]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -t | 显示TCP套接字 |
| -u | 显示UDP套接字 |
| -x | 显示UNIX套接字 |
| -l | 显示监听状态的套接字 |
| -p | 显示进程信息 |
| -n | 显示数字地址 |
| -s | 显示摘要信息 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 显示所有套接字
```bash
ss
```

### 4.2 显示TCP套接字
```bash
ss -t
```

### 4.3 显示监听状态的套接字
```bash
ss -l
```

### 4.4 显示进程信息
```bash
ss -p
```

## 5. 注意事项
- ss 命令用于显示套接字统计，是 netstat 命令的替代品
- ss 命令提供了更快速、更详细的套接字信息
- ss 命令是一个常用的网络工具，用于查看网络连接状态
