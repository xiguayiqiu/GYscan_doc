# nmtui 命令

## 命令说明

`nmtui` 是 NetworkManager 的文本用户界面工具，用于在终端中以交互式方式管理网络连接。

## 语法

```bash
nmtui [选项]
```

## 常用选项

| 选项 | 描述 |
|------|------|
| `edit` | 编辑网络连接 |
| `connect` | 连接到网络 |
| `hostname` | 设置主机名 |

## 操作方法

1. 运行 `nmtui` 命令后，会显示一个文本界面
2. 使用方向键在菜单中导航
3. 使用 Enter 键选择菜单项
4. 使用 Tab 键在不同字段之间切换
5. 使用 Space 键选择或取消选择选项
6. 使用 Ctrl+C 退出程序

## 示例

### 运行 nmtui 主界面

```bash
nmtui
```

### 直接编辑网络连接

```bash
nmtui edit
```

### 直接连接到网络

```bash
nmtui connect
```

### 直接设置主机名

```bash
nmtui hostname
```

## 功能说明

- **编辑连接**：创建、修改或删除网络连接
- **激活连接**：激活或停用网络连接
- **设置系统主机名**：修改系统的主机名

## 注意事项

- `nmtui` 是 NetworkManager 的文本用户界面工具，提供了比 `nmcli` 更友好的交互式操作
- 适合在没有图形界面的环境中使用
- 操作完成后，NetworkManager 会自动应用更改
- 某些操作可能需要输入密码进行授权
