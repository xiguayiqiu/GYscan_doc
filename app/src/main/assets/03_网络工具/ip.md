# ip - 显示和配置网络

## 1. 命令简介
ip 命令用于显示和配置网络，是 Linux 系统中常用的网络工具命令之一。

## 2. 命令语法
```bash
ip [选项] [命令]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -h | 显示帮助信息 |
| -V | 显示版本信息 |

## 4. 常用命令
| 命令 | 说明 |
|------|------|
| addr | 显示和配置网络地址 |
| route | 显示和配置路由表 |
| link | 显示和配置网络接口 |
| neigh | 显示和配置邻居表 |
| rule | 显示和配置路由规则 |
| tunnel | 显示和配置隧道 |
| maddr | 显示和配置多播地址 |

## 5. 实战示例

### 5.1 显示网络地址
```bash
ip addr
```

### 5.2 显示路由表
```bash
ip route
```

### 5.3 显示网络接口
```bash
ip link
```

### 5.4 显示邻居表
```bash
ip neigh
```

## 6. 注意事项
- ip 命令用于显示和配置网络，是 ifconfig 命令的替代品
- ip 命令提供了更强大的网络配置功能
- ip 命令是一个常用的网络工具，用于管理网络配置
