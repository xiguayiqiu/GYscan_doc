# route - 显示和修改路由表

## 1. 命令简介
route 命令用于显示和修改路由表，是 Linux 系统中常用的网络工具命令之一。

## 2. 命令语法
```bash
route [选项] [命令]
```

## 3. 常用命令
| 命令 | 说明 |
|------|------|
| add | 添加路由 |
| del | 删除路由 |
| -n | 以数字形式显示IP地址 |
| -v | 显示详细信息 |
| -A | 指定地址族 |

## 4. 实战示例

### 4.1 显示路由表
```bash
route
```

### 4.2 以数字形式显示路由表
```bash
route -n
```

### 4.3 添加默认路由
```bash
sudo route add default gw 192.168.1.1
```

### 4.4 添加网络路由
```bash
sudo route add -net 192.168.2.0 netmask 255.255.255.0 gw 192.168.1.1
```

### 4.5 添加主机路由
```bash
sudo route add -host 192.168.2.100 gw 192.168.1.1
```

### 4.6 删除路由
```bash
sudo route del -net 192.168.2.0 netmask 255.255.255.0
```

## 5. 注意事项
- route 命令需要 root 权限才能修改路由表
- route 命令在现代 Linux 系统中已经被 ip route 命令替代，但仍然被广泛使用
- 路由表是用于决定网络数据包如何转发的关键配置
- 使用 route -n 命令可以以数字形式显示路由表，避免DNS解析延迟
