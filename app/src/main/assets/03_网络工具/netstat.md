# netstat - 显示网络状态

## 1. 命令简介
netstat 命令用于显示网络状态，是 Linux 系统中常用的网络工具命令之一。

## 2. 命令语法
```bash
netstat [选项]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -a | 显示所有连接和监听端口 |
| -t | 显示TCP连接 |
| -u | 显示UDP连接 |
| -l | 显示监听状态的端口 |
| -p | 显示进程信息 |
| -n | 显示数字地址 |
| -s | 显示统计信息 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 显示所有连接和监听端口
```bash
netstat -a
```

### 4.2 显示TCP连接
```bash
netstat -t
```

### 4.3 显示监听状态的端口
```bash
netstat -l
```

### 4.4 显示进程信息
```bash
netstat -p
```

## 5. 注意事项
- netstat 命令用于显示网络状态，包括连接、监听端口和路由表等信息
- netstat 命令在现代 Linux 系统中已被 ss 命令替代，但仍然被广泛使用
- netstat 命令是一个常用的网络工具，用于网络故障排查
