# arp - 显示和修改ARP缓存

## 1. 命令简介
arp 命令用于显示和修改 ARP 缓存，是 Linux 系统中常用的网络工具命令之一。

## 2. 命令语法
```bash
arp [选项] [主机]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -a | 显示所有ARP条目 |
| -d | 删除ARP条目 |
| -s | 添加ARP条目 |
| -n | 使用数字地址 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 显示所有ARP条目
```bash
arp -a
```

### 4.2 使用数字地址
```bash
arp -n
```

### 4.3 添加ARP条目
```bash
arp -s 192.168.1.1 00:11:22:33:44:55
```

### 4.4 删除ARP条目
```bash
arp -d 192.168.1.1
```

## 5. 注意事项
- arp 命令用于显示和修改 ARP 缓存，管理IP地址和MAC地址的映射
- arp 命令需要 root 权限来修改 ARP 缓存
- arp 命令是一个常用的网络工具，用于管理ARP缓存
