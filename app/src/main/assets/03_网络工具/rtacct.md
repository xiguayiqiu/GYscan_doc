# rtacct - 显示路由统计

## 1. 命令简介
rtacct 命令用于显示路由统计，是 Linux 系统中常用的网络工具命令之一。

## 2. 命令语法
```bash
rtacct [选项]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -h | 显示帮助信息 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 显示路由统计
```bash
rtacct
```

## 5. 注意事项
- rtacct 命令用于显示路由统计，包括路由表的使用情况和统计信息
- rtacct 命令是一个常用的网络工具，用于路由监控
