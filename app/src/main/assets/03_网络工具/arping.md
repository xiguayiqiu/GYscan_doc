# arping - 发送ARP请求

## 1. 命令简介
arping 命令用于发送ARP请求，是 Linux 系统中常用的网络工具命令之一。

## 2. 命令语法
```bash
arping [选项] [目标]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -c | 指定发送的ARP请求数量 |
| -i | 指定网络接口 |
| -I | 指定源IP地址 |
| -s | 指定源MAC地址 |
| -U | 发送无ARP请求 |
| -A | 发送ARP应答 |
| -F | 发送伪造的ARP应答 |
| -D | 重复地址检测 |
| -v | 显示详细信息 |

## 4. 实战示例

### 4.1 发送ARP请求
```bash
arping 192.168.1.1
```

### 4.2 指定发送的ARP请求数量
```bash
arping -c 4 192.168.1.1
```

### 4.3 指定网络接口
```bash
arping -i eth0 192.168.1.1
```

### 4.4 重复地址检测
```bash
arping -D 192.168.1.100
```

### 4.5 发送ARP应答
```bash
arping -A 192.168.1.1
```

## 5. 注意事项
- arping 命令通过发送ARP请求来测试网络连接
- arping 命令可以用于检测网络中是否有IP地址冲突
- arping 命令需要 root 权限才能执行某些操作
- arping 命令与 ping 命令类似，但它使用ARP协议而不是ICMP协议
