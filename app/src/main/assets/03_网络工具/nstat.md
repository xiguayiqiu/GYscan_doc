# nstat - 显示网络统计

## 1. 命令简介
nstat 命令用于显示网络统计，是 Linux 系统中常用的网络工具命令之一。

## 2. 命令语法
```bash
nstat [选项]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -a | 显示所有统计 |
| -t | 显示时间戳 |
| -s | 显示摘要 |
| -z | 只显示非零值 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 显示网络统计
```bash
nstat
```

### 4.2 显示所有统计
```bash
nstat -a
```

### 4.3 显示时间戳
```bash
nstat -t
```

### 4.4 只显示非零值
```bash
nstat -z
```

## 5. 注意事项
- nstat 命令用于显示网络统计，包括各种网络协议的统计信息
- nstat 命令是一个常用的网络工具，用于网络性能分析
