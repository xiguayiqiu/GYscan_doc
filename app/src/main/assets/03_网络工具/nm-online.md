# nm-online 命令

 ## 命令说明

`nm-online` 是 NetworkManager 的命令行工具，用于检查网络连接状态，确定网络是否在线。

## 语法

```bash
nm-online [选项]
```

## 常用选项

| 选项 | 描述 |
|------|------|
| `-q` | 静默模式，不输出任何信息 |
| `-s` | 短输出模式，只显示 "online" 或 "offline" |
| `-t` | 指定超时时间（秒），默认 30 秒 |
| `-x` | 如果网络离线，返回非零退出状态 |
| `-h` | 显示帮助信息 |
| `-V` | 显示版本信息 |

## 示例

### 检查网络连接状态

```bash
nm-online
```

### 静默模式检查网络连接

```bash
nm-online -q
```

### 短输出模式检查网络连接

```bash
nm-online -s
```

### 指定超时时间检查网络连接

```bash
nm-online -t 10
```

### 如果网络离线返回非零退出状态

```bash
nm-online -x
if [ $? -eq 0 ]; then
    echo "网络在线"
else
    echo "网络离线"
fi
```

## 退出状态

| 状态码 | 含义 |
|--------|------|
| 0 | 网络在线 |
| 1 | 网络离线 |
| 2 | 命令行参数错误 |

## 注意事项

- `nm-online` 命令用于检查 NetworkManager 管理的网络连接是否在线
- 默认情况下，它会等待最多 30 秒，直到网络连接建立
- 可以使用 `-t` 选项指定超时时间
- 适合在脚本中用于检查网络连接状态
