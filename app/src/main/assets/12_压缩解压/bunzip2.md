# bunzip2 - 解压缩文件

## 1. 命令简介
bunzip2 命令用于解压缩文件，是 Linux 系统中常用的压缩解压命令之一。

## 2. 命令语法
```bash
bunzip2 [选项] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -f | 强制解压缩 |
| -k | 保留原文件 |
| -l | 显示压缩信息 |
| -r | 递归解压缩 |
| -t | 测试压缩文件 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 解压缩文件
```bash
bunzip2 file.txt.bz2
```

### 4.2 强制解压缩
```bash
bunzip2 -f file.txt.bz2
```

### 4.3 保留原文件
```bash
bunzip2 -k file.txt.bz2
```

### 4.4 显示压缩信息
```bash
bunzip2 -l file.txt.bz2
```

## 5. 注意事项
- bunzip2 命令用于解压缩文件，处理 .bz2 后缀的压缩文件
- bunzip2 命令默认会删除原压缩文件，使用 -k 选项可以保留原文件
- bunzip2 命令是一个常用的压缩工具，用于文件解压缩
