# gzip - 压缩文件

## 1. 命令简介
gzip 命令用于压缩文件，是 Linux 系统中常用的压缩解压命令之一。

## 2. 命令语法
```bash
gzip [选项] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -d | 解压缩 |
| -f | 强制压缩 |
| -k | 保留原文件 |
| -l | 显示压缩信息 |
| -r | 递归压缩 |
| -t | 测试压缩文件 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 压缩文件
```bash
gzip file.txt
```

### 4.2 解压缩文件
```bash
gzip -d file.txt.gz
```

### 4.3 保留原文件
```bash
gzip -k file.txt
```

### 4.4 显示压缩信息
```bash
gzip -l file.txt.gz
```

## 5. 注意事项
- gzip 命令用于压缩文件，生成 .gz 后缀的压缩文件
- gzip 命令默认会删除原文件，使用 -k 选项可以保留原文件
- gzip 命令是一个常用的压缩工具，用于文件压缩
