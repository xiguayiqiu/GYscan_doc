# passwd - 修改用户密码

## 1. 命令简介
passwd 命令用于修改用户密码，是 Linux 系统中常用的用户管理命令之一。

## 2. 命令语法
```bash
passwd [选项] [用户名]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -d | 删除用户密码 |
| -l | 锁定用户账户 |
| -u | 解锁用户账户 |
| -e | 强制用户下次登录时修改密码 |
| -n | 设置密码最短使用期限 |
| -x | 设置密码最长使用期限 |
| -w | 设置密码过期前警告天数 |
| -i | 设置密码过期后禁用账户的天数 |

## 4. 实战示例

### 4.1 修改当前用户密码
```bash
passwd
```

### 4.2 修改指定用户密码
```bash
sudo passwd user1
```

### 4.3 删除用户密码
```bash
sudo passwd -d user1
```

### 4.4 锁定用户账户
```bash
sudo passwd -l user1
```

### 4.5 解锁用户账户
```bash
sudo passwd -u user1
```

### 4.6 强制用户下次登录时修改密码
```bash
sudo passwd -e user1
```

## 5. 注意事项
- passwd 命令修改当前用户密码时不需要 root 权限，但修改其他用户密码时需要
- 使用 -d 选项可以删除用户密码，这样用户可以无密码登录
- 使用 -l 选项可以锁定用户账户，使用 -u 选项可以解锁用户账户
- 使用 -e 选项可以强制用户下次登录时修改密码
