# last 命令

## 命令说明

`last` 是 Linux 系统中用于显示用户的登录历史记录的命令。

## 语法

```bash
last [选项] [用户名] [终端]
```

## 常用选项

| 选项 | 描述 |
|------|------|
| `-a` | 将主机名显示在最后一列 |
| `-d` | 将IP地址转换为主机名 |
| `-f` | 指定使用的日志文件 |
| `-i` | 将IP地址显示为数字形式 |
| `-n` | 显示指定数量的记录 |
| `-R` | 不显示主机名 |
| `-t` | 显示指定时间之前的记录 |
| `-w` | 显示完整的用户名和主机名 |

## 输出说明

| 列名 | 描述 |
|------|------|
| USER | 用户名 |
| TTY | 终端设备 |
| FROM | 登录来源 |
| LOGIN@ | 登录时间 |
| LOGOUT | 注销时间 |
| DURATION | 登录持续时间 |

## 示例

### 显示所有用户的登录历史

```bash
last
```

### 显示指定用户的登录历史

```bash
last username
```

### 显示指定数量的记录

```bash
last -n 10
```

### 显示指定时间之前的记录

```bash
last -t 20230101000000
```

## 注意事项

- `last` 命令不需要特殊权限
- 登录历史记录保存在 `/var/log/wtmp` 文件中
- 可以使用 `lastb` 命令查看失败的登录尝试
- 输出信息包括用户名、终端设备、登录来源、登录时间、注销时间和登录持续时间
