# lastlog 命令

## 命令说明

`lastlog` 是 Linux 系统中用于显示用户最后登录信息的命令。

## 语法

```bash
lastlog [选项]
```

## 常用选项

| 选项 | 描述 |
|------|------|
| `-b` | 显示指定天数之前的记录 |
| `-t` | 显示指定天数内的记录 |
| `-u` | 显示指定用户的记录 |
| `-h` | 显示帮助信息 |

## 输出说明

| 列名 | 描述 |
|------|------|
| Username | 用户名 |
| Port | 登录端口 |
| From | 登录来源 |
| Latest | 最后登录时间 |

## 示例

### 显示所有用户的最后登录信息

```bash
lastlog
```

### 显示指定用户的最后登录信息

```bash
lastlog -u username
```

### 显示指定天数内的记录

```bash
lastlog -t 7
```

### 显示指定天数之前的记录

```bash
lastlog -b 30
```

## 注意事项

- `lastlog` 命令不需要特殊权限
- 最后登录信息保存在 `/var/log/lastlog` 文件中
- 输出信息包括用户名、登录端口、登录来源和最后登录时间
- 对于从未登录过的用户，会显示 "Never logged in"
