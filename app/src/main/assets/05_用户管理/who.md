# who 命令

## 命令说明

`who` 是 Linux 系统中用于显示当前登录用户的命令。

## 语法

```bash
who [选项]
```

## 常用选项

| 选项 | 描述 |
|------|------|
| `-a` | 显示所有信息 |
| `-b` | 显示系统启动时间 |
| `-d` | 显示已死的进程 |
| `-H` | 显示标题行 |
| `-l` | 显示登录进程 |
| `-m` | 只显示当前用户 |
| `-q` | 显示登录用户数量和用户名 |
| `-r` | 显示当前运行级别 |
| `-s` | 以短格式显示（默认） |
| `-T` | 显示终端状态 |
| `-u` | 显示用户的空闲时间 |

## 输出说明

| 列名 | 描述 |
|------|------|
| USER | 用户名 |
| TTY | 终端设备 |
| FROM | 登录来源 |
| LOGIN@ | 登录时间 |
| IDLE | 空闲时间 |
| PID | 进程ID |
| COMMENT | 注释信息 |

## 示例

### 显示当前登录用户

```bash
who
```

### 只显示当前用户

```bash
who -m
```

### 显示系统启动时间

```bash
who -b
```

### 显示登录用户数量和用户名

```bash
who -q
```

### 显示所有信息

```bash
who -a
```

## 注意事项

- `who` 命令不需要特殊权限
- 输出信息包括用户名、终端设备、登录来源和登录时间
- 可以使用 `w` 命令查看更详细的用户活动信息
