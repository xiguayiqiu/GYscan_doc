# users 命令

## 命令说明

`users` 是 Linux 系统中用于显示当前登录用户名的命令。

## 语法

```bash
users [选项]
```

## 常用选项

| 选项 | 描述 |
|------|------|
| `-h` | 显示帮助信息 |
| `-V` | 显示版本信息 |

## 示例

### 显示当前登录的用户名

```bash
users
```

## 注意事项

- `users` 命令不需要特殊权限
- 输出信息只包含当前登录的用户名，每个用户名会根据登录次数重复显示
- 可以使用 `who` 命令查看更详细的用户登录信息
- 可以使用 `w` 命令查看用户的活动信息
