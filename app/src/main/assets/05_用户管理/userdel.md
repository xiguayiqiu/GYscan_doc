# userdel - 删除用户

## 1. 命令简介
userdel 命令用于删除用户，是 Linux 系统中常用的用户管理命令之一。

## 2. 命令语法
```bash
userdel [选项] [用户名]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -f | 强制删除用户，即使用户当前已登录 |
| -r | 删除用户主目录和邮件目录 |

## 4. 实战示例

### 4.1 删除用户
```bash
sudo userdel user1
```

### 4.2 删除用户并删除主目录
```bash
sudo userdel -r user1
```

### 4.3 强制删除用户
```bash
sudo userdel -f user1
```

## 5. 注意事项
- userdel 命令需要 root 权限，使用 sudo 执行
- 使用 -r 选项可以删除用户主目录和邮件目录
- 使用 -f 选项可以强制删除用户，即使用户当前已登录
- 删除用户前，建议先备份用户的重要数据
