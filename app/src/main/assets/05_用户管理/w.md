# w 命令

## 命令说明

`w` 是 Linux 系统中用于显示当前登录用户及其活动的命令。

## 语法

```bash
w [选项]
```

## 常用选项

| 选项 | 描述 |
|------|------|
| `-h` | 不显示标题行 |
| `-u` | 显示用户的空闲时间 |
| `-s` | 以短格式显示 |
| `-f` | 显示用户从哪里登录 |

## 输出说明

| 列名 | 描述 |
|------|------|
| USER | 用户名 |
| TTY | 终端设备 |
| FROM | 登录来源 |
| LOGIN@ | 登录时间 |
| IDLE | 空闲时间 |
| JCPU | 该终端所有进程使用的CPU时间 |
| PCPU | 当前进程使用的CPU时间 |
| WHAT | 当前正在运行的命令 |

## 示例

### 显示当前登录用户

```bash
w
```

### 以短格式显示

```bash
w -s
```

### 显示用户的空闲时间

```bash
w -u
```

## 注意事项

- `w` 命令不需要特殊权限
- 输出信息包括当前登录的用户、他们的终端、登录时间、空闲时间以及正在运行的命令
- 可以使用 `who` 命令查看更简单的用户登录信息
