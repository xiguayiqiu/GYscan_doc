# gawk - GNU awk文本处理工具

## 1. 命令简介
gawk 命令是 GNU 版本的 awk 文本处理工具，是 Linux 系统中常用的文本处理命令之一。

## 2. 命令语法
```bash
gawk [选项] [脚本] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -F | 指定字段分隔符 |
| -v | 定义变量 |
| -f | 从文件中读取脚本 |
| -m | 内存限制 |
| -O | 优化模式 |
| --version | 显示版本信息 |

## 4. 内置变量
| 变量 | 说明 |
|------|------|
| $0 | 整行内容 |
| $1-$n | 第1到n个字段 |
| NF | 字段数量 |
| NR | 行号 |
| FS | 字段分隔符 |
| OFS | 输出字段分隔符 |
| RS | 记录分隔符 |
| ORS | 输出记录分隔符 |

## 5. 实战示例

### 5.1 打印指定字段
```bash
gawk '{print $1, $3}' file.txt
```

### 5.2 使用自定义分隔符
```bash
gawk -F "," '{print $1}' file.csv
```

### 5.3 计算字段之和
```bash
gawk '{sum += $1} END {print sum}' file.txt
```

### 5.4 条件过滤
```bash
gawk '$3 > 100 {print $1}' file.txt
```

## 6. 注意事项
- gawk 命令是 GNU 版本的 awk，提供了更多功能
- gawk 命令支持更多的内置函数和特性
- gawk 命令是一个常用的文本处理工具，用于数据提取和分析
