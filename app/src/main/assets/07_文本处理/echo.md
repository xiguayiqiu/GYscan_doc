# echo - 显示文本或变量值

## 1. 命令简介
echo 命令用于显示文本或变量值，是 Linux 系统中常用的文本处理命令之一。

## 2. 命令语法
```bash
echo [选项] [字符串]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -n | 不输出换行符 |
| -e | 解释转义序列 |
| -E | 不解释转义序列（默认） |

## 4. 常用转义序列
| 转义序列 | 说明 |
|---------|------|
| \n | 换行 |
| \t | 制表符 |
| \r | 回车 |
| \\ | 反斜杠 |
| \" | 双引号 |

## 5. 实战示例

### 5.1 显示文本
```bash
echo "Hello, World!"
```

### 5.2 不输出换行符
```bash
echo -n "Hello, " && echo "World!"
```

### 5.3 解释转义序列
```bash
echo -e "Hello\nWorld"
```

### 5.4 显示变量值
```bash
name="John" && echo "Hello, $name!"
```

## 6. 注意事项
- echo 命令用于显示文本或变量值
- echo 命令默认会添加换行符，使用 -n 选项可以不添加
- echo 命令使用 -e 选项可以解释转义序列
- echo 命令是一个常用的命令，用于输出信息和调试
