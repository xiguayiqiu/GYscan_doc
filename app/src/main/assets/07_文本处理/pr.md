# pr - 格式化打印

## 1. 命令简介
pr 命令用于格式化打印文本，是 Linux 系统中常用的文本处理命令之一。

## 2. 命令语法
```bash
pr [选项] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -a | 多列输出 |
| -d | 双倍间距 |
| -h | 指定页眉 |
| -l | 指定页长 |
| -n | 显示行号 |
| -o | 指定左缩进 |
| -s | 指定分隔符 |
| -t | 不显示页眉和页脚 |
| -w | 指定页宽 |

## 4. 实战示例

### 4.1 格式化打印文本
```bash
pr file.txt
```

### 4.2 多列输出
```bash
pr -a file.txt
```

### 4.3 显示行号
```bash
pr -n file.txt
```

### 4.4 指定页眉
```bash
pr -h "Report" file.txt
```

## 5. 注意事项
- pr 命令用于格式化打印文本，添加页眉、页脚和行号
- pr 命令可以设置页长、页宽和缩进
- pr 命令是一个常用的文本处理工具，用于准备打印文档
