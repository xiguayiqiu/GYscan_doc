# vi - Vi文本编辑器

## 1. 命令简介
vi 是 Linux 系统中最经典的文本编辑器之一，是许多其他编辑器的基础。它以模式化编辑为特点，提供了强大的文本编辑功能。

## 2. 命令语法
```bash
vi [选项] [文件]
```

## 3. 常用选项

| 选项 | 说明 |
|------|------|
| -c <命令> | 在打开文件后执行指定的命令 |
| -r | 恢复崩溃时的文件 |
| -s | 静默模式，不显示任何提示 |
| -V | 显示详细的版本信息 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 工作模式

vi 编辑器有三种基本工作模式：

| 模式 | 描述 | 进入方式 | 退出方式 |
|------|------|----------|----------|
| 命令模式 | 执行命令，如移动光标、删除文本等 | 启动 vi 后默认进入 | 按 i, a, o 等键进入插入模式 |
| 插入模式 | 输入文本 | 在命令模式下按 i, a, o 等键 | 按 Esc 键返回命令模式 |
| 末行模式 | 执行保存、退出等操作 | 在命令模式下按 : 键 | 执行命令后自动返回命令模式 |

## 6. 实战示例

### 6.1 打开文件
```bash
vi file.txt
```

### 6.2 编辑文件
1. 打开文件后，按 `i` 进入插入模式
2. 输入文本
3. 按 `Esc` 返回命令模式
4. 按 `:wq` 保存并退出

### 6.3 查找文本
1. 在命令模式下按 `/`
2. 输入要查找的文本
3. 按 `Enter` 开始查找
4. 按 `n` 查找下一个匹配项
5. 按 `N` 查找上一个匹配项

### 6.4 替换文本
```bash
# 替换当前行中的第一个 old 为 new
:s/old/new

# 替换当前行中的所有 old 为 new
:s/old/new/g

# 替换文件中的所有 old 为 new
:%s/old/new/g
```

### 6.5 复制和粘贴
1. 在命令模式下，将光标移动到要复制的行
2. 按 `yy` 复制当前行
3. 将光标移动到要粘贴的位置
4. 按 `p` 在光标下方粘贴

### 6.6 删除文本
1. 在命令模式下，将光标移动到要删除的行
2. 按 `dd` 删除当前行
3. 或按 `x` 删除光标所在字符

## 7. 高级用法

### 7.1 分屏编辑
```bash
# 水平分屏
:split file.txt

# 垂直分屏
:vsplit file.txt

# 切换分屏
Ctrl+w w

# 调整分屏大小
Ctrl+w +
Ctrl+w -
```

### 7.2 录制宏
1. 在命令模式下按 `q` 后跟一个字母（如 `q`）开始录制
2. 执行要录制的操作
3. 按 `q` 结束录制
4. 按 `@` 后跟宏名称（如 `@q`）执行宏

### 7.3 批量编辑
```bash
# 选择多行
Ctrl+v 进入可视块模式
移动光标选择要编辑的行
按 I 进入插入模式
输入要添加的文本
按 Esc 应用到所有选中的行
```

### 7.4 保存到新文件
```bash
:w newfile.txt
```

### 7.5 读取其他文件
```bash
:r anotherfile.txt
```

## 8. 常见问题与解决方案

### 8.1 问题：无法退出 vi
**解决方案**：按 `Esc` 确保处于命令模式，然后输入 `:q!` 强制退出

### 8.2 问题：文件被锁定
**解决方案**：使用 `-r` 选项恢复文件
```bash
vi -r file.txt
```

### 8.3 问题：显示行号
**解决方案**：在末行模式下输入 `:set nu`

### 8.4 问题：替换所有匹配项
**解决方案**：使用全局替换命令
```bash
:%s/old/new/g
```

### 8.5 问题：撤销操作
**解决方案**：在命令模式下按 `u`

### 8.6 问题：复制多行
**解决方案**：使用 `nyy` 命令，如 `3yy` 复制3行

## 10. 注意事项
- vi 是 Linux 系统中最经典的文本编辑器之一，是许多其他编辑器的基础
- vi 以模式化编辑为特点，提供了强大的文本编辑功能
- 常用的操作包括：文本编辑、查找替换、复制粘贴、分屏编辑等
- vi 有三种基本工作模式：命令模式、插入模式和末行模式
- 在使用 vi 时，需要熟悉各种命令和快捷键
- vi 的命令通常是单个字母或组合键，需要一定的练习才能熟练掌握
- 对于初学者来说，可能需要一些时间来适应 vi 的编辑方式
- vi 是一个功能强大的编辑器，掌握它可以提高文本编辑的效率
- 许多 Linux 系统默认安装了 vi 或其增强版 vim
- vi 的命令语法在不同系统中基本一致，具有很好的可移植性
