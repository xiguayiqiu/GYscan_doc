# nano - 简单文本编辑器

## 1. 命令简介
nano 命令是简单文本编辑器，是 Linux 系统中常用的文本处理命令之一。

## 2. 命令语法
```bash
nano [选项] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -w | 禁用自动换行 |
| -c | 显示行号 |
| -l | 显示行号 |
| -m | 启用鼠标支持 |
| -s | 启用拼写检查 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 常用操作
| 操作 | 说明 |
|------|------|
| Ctrl+O | 保存文件 |
| Ctrl+X | 退出编辑器 |
| Ctrl+K | 剪切当前行 |
| Ctrl+U | 粘贴剪切的内容 |
| Ctrl+W | 搜索 |
| Ctrl+Y | 向上翻页 |
| Ctrl+V | 向下翻页 |

## 5. 实战示例

### 5.1 编辑文件
```bash
nano file.txt
```

### 5.2 显示行号
```bash
nano -c file.txt
```

### 5.3 禁用自动换行
```bash
nano -w file.txt
```

## 6. 注意事项
- nano 命令是简单文本编辑器，适合初学者使用
- nano 命令提供了直观的操作界面和快捷键提示
- nano 命令是一个常用的文本编辑工具，用于编辑配置文件和脚本
