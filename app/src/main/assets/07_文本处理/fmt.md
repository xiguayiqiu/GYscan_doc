# fmt - 格式化文本

## 1. 命令简介
fmt 命令用于格式化文本，是 Linux 系统中常用的文本处理命令之一。

## 2. 命令语法
```bash
fmt [选项] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -c | 采用冠状格式 |
| -p | 保留段落前缀 |
| -s | 简化空格 |
| -t | 统一缩进 |
| -u | 使用统一的空格 |
| -w | 指定行宽（默认75） |

## 4. 实战示例

### 4.1 格式化文本
```bash
fmt file.txt
```

### 4.2 指定行宽
```bash
fmt -w 50 file.txt
```

### 4.3 简化空格
```bash
fmt -s file.txt
```

### 4.4 统一缩进
```bash
fmt -t file.txt
```

## 5. 注意事项
- fmt 命令用于格式化文本，调整文本的行宽和缩进
- fmt 命令默认每行75个字符
- fmt 命令可以保留段落结构和缩进
- fmt 命令是一个常用的文本处理工具，用于美化文本格式
