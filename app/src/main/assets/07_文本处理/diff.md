# diff - 比较文件差异

## 1. 命令简介
diff 命令用于比较文件的差异，是 Linux 系统中常用的文本处理命令之一。

## 2. 命令语法
```bash
diff [选项] [文件1] [文件2]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -u | 统一格式输出 |
| -c | 上下文格式输出 |
| -i | 忽略大小写 |
| -b | 忽略空白字符 |
| -w | 忽略所有空白字符 |
| -r | 递归比较目录 |
| -q | 只显示是否有差异 |

## 4. 实战示例

### 4.1 比较两个文件
```bash
diff file1.txt file2.txt
```

### 4.2 统一格式输出
```bash
diff -u file1.txt file2.txt
```

### 4.3 递归比较目录
```bash
diff -r dir1 dir2
```

### 4.4 忽略大小写
```bash
diff -i file1.txt file2.txt
```

## 5. 注意事项
- diff 命令用于比较文件的差异，显示文件之间的不同之处
- diff 命令的输出格式有多种，如统一格式、上下文格式等
- diff 命令可以递归比较目录
- diff 命令是一个常用的文件比较工具，用于版本控制和代码审查
