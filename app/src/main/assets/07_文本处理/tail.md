# tail - 显示文件尾部

## 1. 命令简介
tail 命令用于显示文件尾部，是 Linux 系统中常用的文本处理命令之一。

## 2. 命令语法
```bash
tail [选项] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -n | 指定显示的行数 |
| -c | 指定显示的字节数 |
| -f | 实时跟踪文件变化 |
| -q | 不显示文件名 |
| -v | 显示文件名 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 显示文件尾部
```bash
tail file.txt
```

### 4.2 指定显示的行数
```bash
tail -n 10 file.txt
```

### 4.3 实时跟踪文件变化
```bash
tail -f file.txt
```

### 4.4 显示多个文件的尾部
```bash
tail file1.txt file2.txt
```

## 5. 注意事项
- tail 命令用于显示文件尾部，默认显示后10行
- tail 命令的 -f 选项常用于实时监控日志文件
- tail 命令是一个常用的文本处理工具，用于查看文件的结尾部分
