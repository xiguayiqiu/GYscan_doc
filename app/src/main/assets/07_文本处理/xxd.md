# xxd - 十六进制转储

## 1. 命令简介
xxd 命令用于十六进制转储，是 Linux 系统中常用的文本处理命令之一。

## 2. 命令语法
```bash
xxd [选项] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -a | 自动跳过空白内容 |
| -b | 以二进制模式显示 |
| -c | 指定每行显示的字节数 |
| -g | 指定分组大小 |
| -l | 只显示指定字节数 |
| -s | 跳过指定字节数 |
| -r | 反向操作，将十六进制转换为二进制 |

## 4. 实战示例

### 4.1 十六进制转储
```bash
xxd file.txt
```

### 4.2 以二进制模式显示
```bash
xxd -b file.txt
```

### 4.3 指定每行显示的字节数
```bash
xxd -c 16 file.txt
```

### 4.4 反向操作
```bash
xxd -r hex.txt binary.bin
```

## 5. 注意事项
- xxd 命令用于十六进制转储，显示文件的二进制内容
- xxd 命令可以将十六进制转换回二进制
- xxd 命令是一个常用的二进制文件查看和编辑工具
