# less - 分屏显示文件内容

## 1. 命令简介
less 命令用于分屏显示文件内容，是 Linux 系统中常用的文本处理命令之一。

## 2. 命令语法
```bash
less [选项] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -N | 显示行号 |
| -S | 不换行显示 |
| -i | 忽略大小写搜索 |
| -m | 显示百分比 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 常用操作
| 操作 | 说明 |
|------|------|
| 空格键 | 向下翻页 |
| b | 向上翻页 |
| 回车键 | 向下翻一行 |
| k | 向上翻一行 |
| /pattern | 搜索模式 |
| n | 下一个匹配 |
| N | 上一个匹配 |
| q | 退出 |

## 5. 实战示例

### 5.1 分屏显示文件内容
```bash
less file.txt
```

### 5.2 显示行号
```bash
less -N file.txt
```

### 5.3 不换行显示
```bash
less -S file.txt
```

## 6. 注意事项
- less 命令用于分屏显示文件内容，支持翻页和搜索
- less 命令比 more 命令更强大，支持向前和向后翻页
- less 命令是一个常用的文本处理工具，用于查看大文件
