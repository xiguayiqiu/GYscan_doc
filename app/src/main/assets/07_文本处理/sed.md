# sed - 流编辑器

## 1. 命令简介
sed 命令是一个流编辑器，用于对文本进行处理，是 Linux 系统中常用的文本处理命令之一。

## 2. 命令语法
```bash
sed [选项] [命令] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -n | 不输出默认内容 |
| -i | 直接修改文件 |
| -e | 执行多个命令 |
| -f | 从文件中读取命令 |
| -r | 使用扩展正则表达式 |

## 4. 常用命令
| 命令 | 说明 |
|------|------|
| s | 替换 |
| d | 删除 |
| p | 打印 |
| a | 在指定行后添加 |
| i | 在指定行前插入 |
| c | 替换指定行 |

## 5. 实战示例

### 5.1 替换文本
```bash
sed 's/old/new/g' file.txt
```

### 5.2 删除行
```bash
sed '3d' file.txt
```

### 5.3 打印指定行
```bash
sed -n '1,5p' file.txt
```

### 5.4 直接修改文件
```bash
sed -i 's/old/new/g' file.txt
```

## 6. 注意事项
- sed 命令是一个流编辑器，用于对文本进行处理
- sed 命令可以执行替换、删除、插入等操作
- 使用 -i 选项可以直接修改文件
- sed 命令是一个常用的文本处理工具，用于自动化文本处理任务
