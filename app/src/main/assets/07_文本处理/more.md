# more - 分屏显示文件内容

## 1. 命令简介
more 命令用于分屏显示文件内容，是 Linux 系统中常用的文本处理命令之一。

## 2. 命令语法
```bash
more [选项] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -n | 显示行号 |
| -d | 显示提示信息 |
| -c | 不滚动显示 |
| -s | 压缩连续的空行 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 常用操作
| 操作 | 说明 |
|------|------|
| 空格键 | 向下翻页 |
| 回车键 | 向下翻一行 |
| b | 向上翻页 |
| q | 退出 |

## 5. 实战示例

### 5.1 分屏显示文件内容
```bash
more file.txt
```

### 5.2 显示行号
```bash
more -n file.txt
```

### 5.3 压缩连续的空行
```bash
more -s file.txt
```

## 6. 注意事项
- more 命令用于分屏显示文件内容，支持基本的翻页操作
- more 命令只能向前翻页，不能向后翻页
- more 命令是一个常用的文本处理工具，用于查看文件
