# nl - 添加行号

## 1. 命令简介
nl 命令用于给文本文件添加行号，是 Linux 系统中常用的文本处理命令之一。

## 2. 命令语法
```bash
nl [选项] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -b | 指定行号计算方式 |
| -n | 指定行号格式 |
| -w | 指定行号宽度 |
| -s | 指定行号分隔符 |

## 4. 行号计算方式
| 选项 | 说明 |
|------|------|
| a | 为所有行添加行号 |
| t | 只为非空行添加行号（默认） |
| n | 不为任何行添加行号 |

## 5. 行号格式
| 选项 | 说明 |
|------|------|
| ln | 左对齐，无前导零 |
| rn | 右对齐，无前导零 |
| rz | 右对齐，有前导零 |

## 6. 实战示例

### 6.1 为文件添加行号
```bash
nl file.txt
```

### 6.2 为所有行添加行号
```bash
nl -ba file.txt
```

### 6.3 指定行号格式
```bash
nl -n rz file.txt
```

### 6.4 指定行号宽度和分隔符
```bash
nl -w 5 -s ": " file.txt
```

## 7. 退出状态
- 0：命令执行成功
- 非0：命令执行失败

## 8. 注意事项
- nl 命令用于给文本文件添加行号
- nl 命令可以指定行号的计算方式、格式、宽度和分隔符
- nl 命令是一个常用的文本处理工具，用于给文件添加行号
