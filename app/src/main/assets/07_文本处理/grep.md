# grep - 文本搜索工具

## 1. 命令简介
grep 命令用于在文本文件中搜索指定的模式，是 Linux 系统中最常用的文本搜索工具。

## 2. 命令语法
```bash
grep [选项] 模式 文件
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -i | 忽略大小写 |
| -v | 反向匹配，显示不包含模式的行 |
| -n | 显示行号 |
| -A | 显示匹配行及其后指定行数 |
| -B | 显示匹配行及其前指定行数 |
| -C | 显示匹配行及其前后指定行数 |
| -r | 递归搜索目录 |
| -l | 只显示包含匹配的文件名 |
| -c | 只显示匹配的行数 |
| -E | 使用扩展正则表达式 |

## 4. 实战示例

### 4.1 基本搜索
```bash
grep "pattern" file.txt
```

### 4.2 忽略大小写搜索
```bash
grep -i "pattern" file.txt
```

### 4.3 显示行号
```bash
grep -n "pattern" file.txt
```

### 4.4 反向匹配
```bash
grep -v "pattern" file.txt
```

### 4.5 递归搜索目录
```bash
grep -r "pattern" directory/
```

### 4.6 显示匹配行及其前后内容
```bash
grep -A 2 -B 2 "pattern" file.txt
```

### 4.7 使用正则表达式
```bash
grep -E "^[0-9]+" file.txt
```

## 5. 注意事项
- grep 支持基本正则表达式，使用 -E 选项启用扩展正则表达式
- 搜索包含特殊字符的模式时，需要使用转义字符\
- 递归搜索大型目录时可能会比较慢，可以使用 -l 选项只显示文件名
