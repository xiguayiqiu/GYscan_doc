# head - 显示文件头部

## 1. 命令简介
head 命令用于显示文件头部，是 Linux 系统中常用的文本处理命令之一。

## 2. 命令语法
```bash
head [选项] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -n | 指定显示的行数 |
| -c | 指定显示的字节数 |
| -q | 不显示文件名 |
| -v | 显示文件名 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 显示文件头部
```bash
head file.txt
```

### 4.2 指定显示的行数
```bash
head -n 10 file.txt
```

### 4.3 指定显示的字节数
```bash
head -c 100 file.txt
```

### 4.4 显示多个文件的头部
```bash
head file1.txt file2.txt
```

## 5. 注意事项
- head 命令用于显示文件头部，默认显示前10行
- head 命令是一个常用的文本处理工具，用于查看文件的开头部分
