# fold - 折叠文本

## 1. 命令简介
fold 命令用于折叠文本，是 Linux 系统中常用的文本处理命令之一。

## 2. 命令语法
```bash
fold [选项] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -b | 按字节折叠 |
| -s | 以单词为边界折叠 |
| -w | 指定行宽（默认80） |

## 4. 实战示例

### 4.1 折叠文本
```bash
fold file.txt
```

### 4.2 指定行宽
```bash
fold -w 50 file.txt
```

### 4.3 以单词为边界折叠
```bash
fold -s file.txt
```

### 4.4 按字节折叠
```bash
fold -b file.txt
```

## 5. 注意事项
- fold 命令用于折叠文本，将长行分成多行
- fold 命令默认每行80个字符
- 使用 -s 选项可以以单词为边界折叠，避免单词被分割
- fold 命令是一个常用的文本处理工具，用于处理长行文本
