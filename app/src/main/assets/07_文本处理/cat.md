# cat - 连接文件并打印到标准输出

## 1. 命令简介
cat 命令用于连接文件并打印到标准输出，是 Linux 系统中常用的文本处理命令之一。

## 2. 命令语法
```bash
cat [选项] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -n | 显示行号 |
| -b | 显示非空行的行号 |
| -s | 压缩连续的空行 |
| -A | 显示所有字符，包括控制字符 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 查看文件内容
```bash
cat file.txt
```

### 4.2 显示行号
```bash
cat -n file.txt
```

### 4.3 连接多个文件
```bash
cat file1.txt file2.txt > file3.txt
```

### 4.4 显示所有字符
```bash
cat -A file.txt
```

## 5. 注意事项
- cat 命令用于连接文件并打印到标准输出
- cat 命令是一个常用的文本处理工具，用于查看和合并文件
