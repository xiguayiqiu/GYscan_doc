# cmp - 逐字节比较文件

## 1. 命令简介
cmp 命令用于逐字节比较文件，是 Linux 系统中常用的文本处理命令之一。

## 2. 命令语法
```bash
cmp [选项] [文件1] [文件2]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -b | 显示不同字节的八进制值 |
| -i | 跳过指定字节数 |
| -l | 显示所有不同字节的位置和值 |
| -n | 只比较指定字节数 |
| -s | 静默模式，只返回退出状态 |

## 4. 实战示例

### 4.1 比较两个文件
```bash
cmp file1.txt file2.txt
```

### 4.2 显示所有不同字节的位置和值
```bash
cmp -l file1.txt file2.txt
```

### 4.3 跳过指定字节数
```bash
cmp -i 10 file1.txt file2.txt
```

### 4.4 只比较指定字节数
```bash
cmp -n 100 file1.txt file2.txt
```

## 5. 注意事项
- cmp 命令用于逐字节比较文件，显示文件之间的不同之处
- cmp 命令比 diff 命令更底层，逐字节比较文件
- cmp 命令是一个常用的文件比较工具，用于检查文件是否完全相同
