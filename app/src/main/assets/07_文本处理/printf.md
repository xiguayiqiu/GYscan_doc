# printf - 格式化输出

## 1. 命令简介
printf 命令用于格式化输出，是 Linux 系统中常用的文本处理命令之一。

## 2. 命令语法
```bash
printf [格式] [参数]
```

## 3. 常用格式说明
| 格式 | 说明 |
|------|------|
| %s | 字符串 |
| %d | 十进制整数 |
| %f | 浮点数 |
| %c | 字符 |
| %x | 十六进制整数 |
| %o | 八进制整数 |
| %b | 解释转义序列 |
| %n | 存储已打印字符数 |

## 4. 常用转义序列
| 转义序列 | 说明 |
|---------|------|
| \n | 换行 |
| \t | 制表符 |
| \r | 回车 |
| \\ | 反斜杠 |
| \" | 双引号 |

## 5. 实战示例

### 5.1 格式化输出字符串
```bash
printf "Hello, %s!\n" "world"
```

### 5.2 格式化输出数字
```bash
printf "The number is %d\n" 42
```

### 5.3 格式化输出浮点数
```bash
printf "The float is %.2f\n" 3.14159
```

### 5.4 格式化输出多个参数
```bash
printf "Name: %s, Age: %d\n" "John" 30
```

## 6. 注意事项
- printf 命令用于格式化输出，比 echo 命令更灵活
- printf 命令支持多种格式说明符，可以格式化不同类型的数据
- printf 命令不会自动添加换行符，需要显式指定 \n
- printf 命令是一个常用的文本处理工具，用于生成格式化的输出
