# md5sum - MD5校验和

## 1. 命令简介
md5sum 命令用于计算文件的 MD5 校验和，是 Linux 系统中常用的安全加密命令之一。

## 2. 命令语法
```bash
md5sum [选项] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -b | 以二进制模式读取 |
| -c | 从文件中读取校验和并验证 |
| -t | 以文本模式读取（默认） |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 计算文件的MD5校验和
```bash
md5sum file.txt
```

### 4.2 验证文件的MD5校验和
```bash
md5sum -c checksums.txt
```

### 4.3 以二进制模式读取
```bash
md5sum -b file.bin
```

## 5. 注意事项
- md5sum 命令用于计算文件的 MD5 校验和
- MD5 是一种密码散列函数，用于验证文件的完整性
- md5sum 命令是一个常用的安全工具，用于验证文件的完整性
