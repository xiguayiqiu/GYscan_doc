# sum - 校验和

## 1. 命令简介
sum 命令用于计算文件的校验和，是 Linux 系统中常用的安全加密命令之一。

## 2. 命令语法
```bash
sum [选项] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -r | 使用BSD算法（默认） |
| -s | 使用SysV算法 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 计算文件的校验和
```bash
sum file.txt
```

### 4.2 使用SysV算法
```bash
sum -s file.txt
```

## 5. 注意事项
- sum 命令用于计算文件的校验和
- sum 命令支持两种算法：BSD算法和SysV算法
- sum 命令是一个常用的安全工具，用于验证文件的完整性
