# readelf - 显示ELF文件信息

## 1. 命令简介
readelf 命令用于显示 ELF 文件的信息，是 Linux 系统中常用的终端控制台命令之一。

## 2. 命令语法
```bash
readelf [选项] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -a | 显示所有信息 |
| -h | 显示文件头信息 |
| -l | 显示程序头信息 |
| -S | 显示节头信息 |
| -s | 显示符号表信息 |
| -d | 显示动态节信息 |
| -r | 显示重定位信息 |
| -A | 显示架构信息 |

## 4. 实战示例

### 4.1 显示所有信息
```bash
readelf -a file
```

### 4.2 显示文件头信息
```bash
readelf -h file
```

### 4.3 显示程序头信息
```bash
readelf -l file
```

### 4.4 显示节头信息
```bash
readelf -S file
```

## 5. 注意事项
- readelf 命令用于显示 ELF 文件的信息，ELF 是 Linux 系统中可执行文件和共享库的标准格式
- readelf 命令可以显示文件头、程序头、节头、符号表等信息
- readelf 命令是一个常用的工具，用于分析和调试 ELF 文件
