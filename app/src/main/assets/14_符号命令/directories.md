# Shell 目录符号

## 命令简介

Shell 目录符号用于表示不同的目录位置，是 Shell 命令中常用的功能。

## 目录符号

| 符号 | 说明 | 示例 |
|------|------|------|
| `~` | 用户主目录：当前用户的主目录路径 | `cd ~` |
| `-` | 上一个工作目录：切换到上一个工作目录 | `cd -` |
| `..` | 上级目录：当前目录的父目录 | `cd ..` |
| `.` | 当前目录：当前工作目录 | `./script.sh` |
| `/` | 根目录：文件系统的根目录 | `cd /` |

## 命令示例

### 切换到用户主目录

```bash
cd ~
```

### 切换到上一个工作目录

```bash
cd -
```

### 切换到上级目录

```bash
cd ..
```

### 切换到当前目录（无实际效果，但可用于脚本中）

```bash
cd .
```

### 切换到根目录

```bash
cd /
```

### 使用目录符号执行脚本

```bash
./script.sh
```

### 使用目录符号指定路径

```bash
ls ~/Documents
ls ../Downloads
ls /usr/bin
```

## 常用组合

### 组合目录符号

```bash
cd ~/Documents/../Downloads
```

### 使用目录符号作为参数

```bash
cp file.txt ~/Documents/
```

### 在脚本中使用目录符号

```bash
#!/bin/bash

# 切换到脚本所在目录
cd "$(dirname "$0")"

# 执行其他操作
```

## 相关命令

- `cd`: 切换目录
- `ls`: 列出目录内容
- `cp`: 复制文件或目录
- `mkdir`: 创建目录
