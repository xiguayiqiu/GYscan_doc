# blkid 命令

## 命令说明

`blkid` 是 Linux 系统中用于显示块设备属性的命令，包括文件系统类型、UUID等信息。

## 语法

```bash
blkid [选项] [设备]
```

## 常用选项

| 选项 | 描述 |
|------|------|
| `-L` | 按标签查找设备 |
| `-U` | 按UUID查找设备 |
| `-c` | 指定缓存文件 |
| `-s` | 只显示指定的属性 |
| `-o` | 指定输出格式 |

## 示例

### 显示所有块设备的属性

```bash
blkid
```

### 显示指定设备的属性

```bash
blkid /dev/sda1
```

### 按标签查找设备

```bash
blkid -L mylabel
```

### 按UUID查找设备

```bash
blkid -U 12345678-1234-1234-1234-1234567890ab
```

### 只显示文件系统类型

```bash
blkid -s TYPE /dev/sda1
```

## 注意事项

- `blkid` 命令需要 root 权限
- 可以使用 `blkid` 命令获取设备的 UUID，用于 fstab 中的挂载配置
- 输出信息包括设备路径、UUID、文件系统类型和标签等
