# mkfs - 创建文件系统

## 1. 命令简介
mkfs 命令用于创建文件系统，是 Linux 系统中常用的磁盘管理命令之一。

## 2. 命令语法
```bash
mkfs [选项] [文件系统类型] [设备]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -t | 指定文件系统类型 |
| -c | 检查坏块 |
| -v | 详细模式 |

## 4. 常用文件系统类型
| 类型 | 说明 |
|------|------|
| ext2 | 第二代扩展文件系统 |
| ext3 | 第三代扩展文件系统 |
| ext4 | 第四代扩展文件系统 |
| xfs | XFS文件系统 |
| btrfs | Btrfs文件系统 |
| vfat | FAT文件系统 |

## 5. 实战示例

### 5.1 创建ext4文件系统
```bash
mkfs -t ext4 /dev/sda1
```

### 5.2 创建XFS文件系统
```bash
mkfs -t xfs /dev/sda1
```

### 5.3 检查坏块并创建文件系统
```bash
mkfs -c -t ext4 /dev/sda1
```

## 6. 注意事项
- mkfs 命令用于创建文件系统，需要 root 权限
- 创建文件系统会格式化分区，导致数据丢失
- 不同的文件系统类型有不同的特点和适用场景
- 常用的文件系统类型包括 ext4、XFS、Btrfs 等
