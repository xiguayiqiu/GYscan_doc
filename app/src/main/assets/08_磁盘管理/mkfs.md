# mkfs 命令

## 命令说明

`mkfs` 是 Linux 系统中用于创建文件系统的命令，它可以在设备上创建指定类型的文件系统。

## 语法

```bash
mkfs [选项] [-t 文件系统类型] 设备 [大小]
```

## 常用选项

| 选项 | 描述 |
|------|------|
| `-t` | 指定文件系统类型 |
| `-c` | 检查坏块 |
| `-v` | 详细输出 |

## 常用文件系统类型

| 类型 | 描述 |
|------|------|
| `ext2` | 第二代扩展文件系统 |
| `ext3` | 第三代扩展文件系统 |
| `ext4` | 第四代扩展文件系统 |
| `xfs` | XFS 文件系统 |
| `btrfs` | Btrfs 文件系统 |
| `fat` | FAT 文件系统 |
| `ntfs` | NTFS 文件系统 |

## 示例

### 创建 ext4 文件系统

```bash
mkfs -t ext4 /dev/sdb1
```

### 检查坏块并创建文件系统

```bash
mkfs -c -t ext4 /dev/sdb1
```

### 使用简写形式创建文件系统

```bash
mkfs.ext4 /dev/sdb1
```

## 注意事项

- `mkfs` 命令需要 root 权限
- 创建文件系统会清除设备上的所有数据，使用前请确保已备份重要数据
- 不同文件系统类型有专门的创建工具，如 `mke2fs` 用于 ext2/ext3/ext4 文件系统
