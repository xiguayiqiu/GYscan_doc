# umount - 卸载文件系统

## 1. 命令简介
umount 命令用于卸载已挂载的文件系统，是 Linux 系统中常用的磁盘管理命令之一。

## 2. 命令语法
```bash
umount [选项] [挂载点或设备]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -l | 懒卸载，即不立即卸载，而是在文件系统不再被使用时卸载 |
| -f | 强制卸载 |
| -a | 卸载/etc/mtab中所有已挂载的文件系统 |
| -v | 显示详细信息 |

## 4. 实战示例

### 4.1 卸载文件系统（通过挂载点）
```bash
sudo umount /mnt
```

### 4.2 卸载文件系统（通过设备）
```bash
sudo umount /dev/sda1
```

### 4.3 懒卸载
```bash
sudo umount -l /mnt
```

### 4.4 强制卸载
```bash
sudo umount -f /mnt
```

### 4.5 卸载所有已挂载的文件系统
```bash
sudo umount -a
```

## 5. 注意事项
- umount 命令需要 root 权限，使用 sudo 执行
- 卸载前需要确保文件系统未被其他进程使用
- 如果文件系统正在被使用，可以使用 -l 选项进行懒卸载
- 可以使用 mount 命令查看当前已挂载的文件系统
