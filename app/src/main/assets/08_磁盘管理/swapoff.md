# swapoff - 禁用交换分区

## 1. 命令简介
swapoff 命令用于禁用交换分区，是 Linux 系统中常用的磁盘管理命令之一。

## 2. 命令语法
```bash
swapoff [选项] [交换设备或文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -a | 禁用所有交换设备 |
| -v | 显示详细信息 |
| --help | 显示帮助信息 |

## 4. 实战示例

### 4.1 禁用交换分区
```bash
sudo swapoff /dev/sda2
```

### 4.2 禁用所有交换设备
```bash
sudo swapoff -a
```

## 5. 注意事项
- swapoff 命令需要 root 权限，使用 sudo 执行
- 可以使用 swapon 命令启用交换设备
- 禁用交换设备前需要确保系统有足够的内存，否则可能导致系统崩溃
