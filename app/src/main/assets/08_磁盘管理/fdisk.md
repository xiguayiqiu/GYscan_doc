# fdisk - 磁盘分区工具

## 1. 命令简介
fdisk 命令用于管理磁盘分区，是 Linux 系统中常用的磁盘管理命令之一。

## 2. 命令语法
```bash
fdisk [选项] [设备]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -l | 显示磁盘分区信息 |
| -u | 以扇区为单位显示 |
| -s | 显示分区大小 |

## 4. 常用操作
| 操作 | 说明 |
|------|------|
| n | 创建新分区 |
| d | 删除分区 |
| p | 显示分区表 |
| t | 更改分区类型 |
| w | 保存更改并退出 |
| q | 退出不保存 |
| m | 显示帮助 |

## 5. 实战示例

### 5.1 显示磁盘分区信息
```bash
fdisk -l
```

### 5.2 管理磁盘分区
```bash
fdisk /dev/sda
```

## 6. 注意事项
- fdisk 命令用于管理磁盘分区，需要 root 权限
- 使用 fdisk 命令可以创建、删除、修改分区
- 修改分区表后需要使用 w 命令保存更改
- 分区操作有风险，请谨慎操作
