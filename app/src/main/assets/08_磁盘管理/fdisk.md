# fdisk 命令

## 命令说明

`fdisk` 是 Linux 系统中用于磁盘分区的命令行工具。

## 语法

```bash
fdisk [选项] 设备
```

## 常用选项

| 选项 | 描述 |
|------|------|
| `-l` | 列出所有磁盘分区 |
| `-u` | 以扇区为单位显示分区大小 |
| `-b` | 指定扇区大小 |

## 交互式命令

| 命令 | 描述 |
|------|------|
| `n` | 创建新分区 |
| `d` | 删除分区 |
| `p` | 显示分区表 |
| `t` | 更改分区类型 |
| `w` | 写入分区表并退出 |
| `q` | 退出而不保存更改 |

## 示例

### 列出所有磁盘分区

```bash
fdisk -l
```

### 编辑磁盘分区

```bash
fdisk /dev/sda
```

## 注意事项

- `fdisk` 命令需要 root 权限
- 修改分区表后需要重启系统或使用 `partprobe` 命令让内核重新读取分区表
- 操作分区表时要小心，错误的操作可能导致数据丢失
