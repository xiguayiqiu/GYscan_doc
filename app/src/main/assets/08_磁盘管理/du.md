# du - 显示目录大小

## 1. 命令简介
du 命令用于显示目录或文件的大小，是 Linux 系统中最常用的磁盘管理命令之一。

## 2. 命令语法
```bash
du [选项] [文件或目录]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -h | 以人类可读的格式显示 |
| -s | 显示总大小，不显示子目录 |
| -c | 显示总计 |
| -a | 显示所有文件和目录 |
| -k | 以KB为单位显示 |
| -m | 以MB为单位显示 |
| -g | 以GB为单位显示 |
| -d | 指定深度 |

## 4. 实战示例

### 4.1 显示目录大小
```bash
du -h directory/
```

### 4.2 显示目录总大小
```bash
du -sh directory/
```

### 4.3 显示所有文件和目录大小
```bash
du -ah directory/
```

### 4.4 显示指定深度的目录大小
```bash
du -h --max-depth=2 directory/
```

### 4.5 显示总计
```bash
du -ch directory/
```

### 4.6 以MB为单位显示
```bash
du -m directory/
```

## 5. 注意事项
- du 命令默认显示目录及其子目录的大小
- 使用 -h 选项可以使输出更加易读
- 使用 -s 选项可以只显示总大小，不显示子目录
