# cfdisk 命令

## 命令说明

`cfdisk` 是 Linux 系统中一个交互式的磁盘分区工具，提供了图形化的界面来管理磁盘分区。

## 语法

```bash
cfdisk [选项] 设备
```

## 常用选项

| 选项 | 描述 |
|------|------|
| `-z` | 不读取分区表 |
| `-c` | 显示柱面边界 |
| `-P` | 显示分区表 |
| `-V` | 显示版本信息 |

## 交互式操作

- 使用方向键选择分区
- 使用 `Enter` 键选择操作
- 可用操作包括：新建、删除、修改分区类型、设置启动标志等
- 操作完成后选择 "Write" 写入分区表，然后选择 "Quit" 退出

## 示例

### 交互式编辑磁盘分区

```bash
cfdisk /dev/sda
```

## 注意事项

- `cfdisk` 命令需要 root 权限
- 修改分区表后需要重启系统或使用 `partprobe` 命令让内核重新读取分区表
- 操作分区表时要小心，错误的操作可能导致数据丢失
- `cfdisk` 提供了比 `fdisk` 更友好的交互式界面
