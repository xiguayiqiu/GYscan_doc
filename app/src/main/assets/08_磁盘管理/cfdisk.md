# cfdisk - 交互式磁盘分区工具

## 1. 命令简介
cfdisk 命令是一个交互式磁盘分区工具，是 Linux 系统中常用的磁盘管理命令之一。

## 2. 命令语法
```bash
cfdisk [选项] [设备]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -h | 显示帮助信息 |
| -z | 不读取分区表 |
| -c | 指定柱面数 |
| -h | 指定磁头数 |
| -s | 指定扇区数 |

## 4. 常用操作
| 操作 | 说明 |
|------|------|
| 方向键 | 选择分区 |
| Enter | 选择操作 |
| n | 创建新分区 |
| d | 删除分区 |
| t | 更改分区类型 |
| w | 保存更改并退出 |
| q | 退出不保存 |

## 5. 实战示例

### 5.1 交互式管理磁盘分区
```bash
cfdisk /dev/sda
```

## 6. 注意事项
- cfdisk 命令是一个交互式磁盘分区工具
- cfdisk 命令提供了图形化界面，比 fdisk 更易用
- 分区操作有风险，请谨慎操作
