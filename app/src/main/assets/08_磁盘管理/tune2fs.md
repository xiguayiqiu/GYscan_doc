# tune2fs 命令

## 命令说明

`tune2fs` 是 Linux 系统中用于调整 ext2/ext3/ext4 文件系统参数的命令。

## 语法

```bash
tune2fs [选项] 设备
```

## 常用选项

| 选项 | 描述 |
|------|------|
| `-l` | 显示文件系统信息 |
| `-c` | 设置最大挂载次数 |
| `-i` | 设置检查间隔时间 |
| `-m` | 设置保留块百分比 |
| `-L` | 设置文件系统标签 |
| `-O` | 启用或禁用文件系统特性 |
| `-o` | 调整挂载选项 |

## 示例

### 显示文件系统信息

```bash
tune2fs -l /dev/sda1
```

### 设置最大挂载次数

```bash
tune2fs -c 100 /dev/sda1
```

### 设置检查间隔时间

```bash
tune2fs -i 7d /dev/sda1
```

### 设置保留块百分比

```bash
tune2fs -m 5 /dev/sda1
```

### 设置文件系统标签

```bash
tune2fs -L mylabel /dev/sda1
```

## 注意事项

- `tune2fs` 命令需要 root 权限
- 可以在文件系统挂载状态下使用 `tune2fs` 命令
- 调整文件系统参数时要小心，错误的参数可能导致文件系统损坏
