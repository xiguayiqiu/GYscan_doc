# mkswap 命令

## 命令说明

`mkswap` 是 Linux 系统中用于创建交换分区或交换文件的命令。

## 语法

```bash
mkswap [选项] 设备|文件 [大小]
```

## 常用选项

| 选项 | 描述 |
|------|------|
| `-f` | 强制创建交换空间，即使设备看起来不是分区 |
| `-L` | 设置交换空间标签 |
| `-p` | 设置页大小 |

## 示例

### 创建交换分区

```bash
mkswap /dev/sda2
```

### 创建交换文件

```bash
# 先创建文件
fallocate -l 2G /swapfile
# 然后设置权限
chmod 600 /swapfile
# 最后创建交换空间
mkswap /swapfile
```

### 设置交换空间标签

```bash
mkswap -L swap /dev/sda2
```

## 注意事项

- `mkswap` 命令需要 root 权限
- 创建交换空间会清除设备或文件上的所有数据，使用前请确保已备份重要数据
- 创建交换文件后，需要使用 `swapon` 命令启用它
- 交换空间的大小通常建议为物理内存的 1-2 倍
