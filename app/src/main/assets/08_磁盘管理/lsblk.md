# lsblk 命令

## 命令说明

`lsblk` 是 Linux 系统中用于列出块设备的命令，以树形结构显示设备信息。

## 语法

```bash
lsblk [选项]
```

## 常用选项

| 选项 | 描述 |
|------|------|
| `-a` | 显示所有设备，包括空设备 |
| `-b` | 以字节为单位显示大小 |
| `-d` | 只显示设备，不显示分区 |
| `-f` | 显示文件系统信息 |
| `-i` | 显示设备的主从设备号 |
| `-m` | 显示权限信息 |
| `-o` | 指定要显示的列 |
| `-p` | 显示完整的设备路径 |
| `-t` | 显示拓扑信息 |

## 示例

### 显示所有块设备

```bash
lsblk
```

### 显示所有设备，包括空设备

```bash
lsblk -a
```

### 显示文件系统信息

```bash
lsblk -f
```

### 只显示设备，不显示分区

```bash
lsblk -d
```

### 显示完整的设备路径

```bash
lsblk -p
```

## 注意事项

- `lsblk` 命令不需要 root 权限
- 输出以树形结构显示，便于查看设备之间的关系
- 可以使用 `-o` 选项自定义显示的列，例如：`lsblk -o NAME,SIZE,TYPE,MOUNTPOINT`
