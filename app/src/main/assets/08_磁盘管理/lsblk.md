# lsblk - 列出块设备

## 1. 命令简介
lsblk 命令用于列出块设备，是 Linux 系统中常用的磁盘管理命令之一。

## 2. 命令语法
```bash
lsblk [选项]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -a | 显示所有设备 |
| -b | 以字节为单位显示大小 |
| -d | 不显示子设备 |
| -f | 显示文件系统信息 |
| -i | 显示设备号 |
| -m | 显示权限信息 |
| -n | 不显示标题 |
| -o | 指定输出列 |
| -p | 显示完整路径 |
| -t | 以树状结构显示 |

## 4. 实战示例

### 4.1 列出所有块设备
```bash
lsblk
```

### 4.2 显示文件系统信息
```bash
lsblk -f
```

### 4.3 以树状结构显示
```bash
lsblk -t
```

### 4.4 显示完整路径
```bash
lsblk -p
```

## 5. 注意事项
- lsblk 命令用于列出块设备，包括磁盘、分区和挂载点
- lsblk 命令提供了详细的设备信息，如大小、类型、挂载点等
- lsblk 命令是一个常用的磁盘管理工具，用于查看设备信息
