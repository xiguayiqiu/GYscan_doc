# blkdiscard 命令

## 命令说明

`blkdiscard` 是 Linux 系统中用于丢弃块设备数据的命令，它会向块设备发送丢弃请求，使设备可以回收这些空间。

## 语法

```bash
blkdiscard [选项] 设备
```

## 常用选项

| 选项 | 描述 |
|------|------|
| `-f` | 强制操作，忽略错误 |
| `-o` | 指定起始偏移量 |
| `-l` | 指定长度 |
| `-s` | 安全擦除，确保数据被完全清除 |

## 示例

### 丢弃整个设备的数据

```bash
blkdiscard /dev/sdb
```

### 丢弃设备的部分数据

```bash
blkdiscard -o 1048576 -l 2097152 /dev/sdb
```

### 安全擦除设备数据

```bash
blkdiscard -s /dev/sdb
```

## 注意事项

- `blkdiscard` 命令需要 root 权限
- 丢弃操作会清除设备上的数据，使用前请确保已备份重要数据
- 不是所有设备都支持丢弃操作
- 安全擦除选项 `-s` 会确保数据被完全清除，比普通丢弃更安全
