# swapon 命令

## 命令说明

`swapon` 是 Linux 系统中用于启用交换分区或交换文件的命令。

## 语法

```bash
swapon [选项] [设备|文件]
```

## 常用选项

| 选项 | 描述 |
|------|------|
| `-a` | 启用 /etc/fstab 中所有的交换设备 |
| `-e` | 跳过不存在的设备 |
| `-p` | 设置交换优先级 |
| `-s` | 显示交换使用情况 |

## 示例

### 启用交换分区

```bash
swapon /dev/sda2
```

### 启用交换文件

```bash
swapon /swapfile
```

### 启用所有 /etc/fstab 中的交换设备

```bash
swapon -a
```

### 显示交换使用情况

```bash
swapon -s
```

### 设置交换优先级

```bash
swapon -p 10 /dev/sda2
```

## 注意事项

- `swapon` 命令需要 root 权限
- 可以使用 `swapoff` 命令禁用交换设备
- 交换分区或交换文件用于当物理内存不足时，临时将数据存储到磁盘上
- 交换空间的大小通常建议为物理内存的 1-2 倍
