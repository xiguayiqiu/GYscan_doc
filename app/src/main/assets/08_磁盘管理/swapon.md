# swapon - 启用交换分区

## 1. 命令简介
swapon 命令用于启用交换分区，是 Linux 系统中常用的磁盘管理命令之一。

## 2. 命令语法
```bash
swapon [选项] [交换设备或文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -a | 启用/etc/fstab中所有交换设备 |
| -e | 如果交换设备不存在，跳过它 |
| -f | 强制启用交换设备 |
| -p | 设置交换设备的优先级 |
| -s | 显示交换设备的使用情况 |
| --help | 显示帮助信息 |

## 4. 实战示例

### 4.1 启用交换分区
```bash
sudo swapon /dev/sda2
```

### 4.2 启用所有交换设备
```bash
sudo swapon -a
```

### 4.3 显示交换设备的使用情况
```bash
sudo swapon -s
```

### 4.4 设置交换设备的优先级
```bash
sudo swapon -p 10 /dev/sda2
```

## 5. 注意事项
- swapon 命令需要 root 权限，使用 sudo 执行
- 可以使用 swapoff 命令禁用交换设备
- 交换设备的优先级越高，系统越优先使用它
- 可以在/etc/fstab中设置交换设备的优先级
