# top - 实时显示系统进程

## 1. 命令简介
top 命令用于实时显示系统进程，是 Linux 系统中常用的系统监控命令之一。它可以实时显示系统的资源使用情况和进程状态。

## 2. 命令语法
```bash
top [选项]
```

## 3. 常用选项

| 选项 | 说明 |
|------|------|
| -h, --help | 显示帮助信息 |
| -b, --batch | 批处理模式 |
| -n, --iterations=<次数> | 设置迭代次数 |
| -d, --delay=<秒> | 设置刷新间隔 |
| -p, --pid=<PID> | 显示指定PID的进程 |
| -u, --user=<用户> | 显示指定用户的进程 |
| -H, --threads | 显示线程 |
| -c, --command | 显示完整命令 |
| -i, --idle | 不显示空闲进程 |
| -s, --secure | 安全模式 |
| -o, --sort-field=<字段> | 按指定字段排序 |

## 5. 实战示例

### 5.1 基本用法
```bash
# 启动top

top

# 设置刷新间隔为2秒
top -d 2

# 显示指定用户的进程
top -u user
```

### 5.2 批处理模式
```bash
# 批处理模式，显示5次后退出
top -b -n 5

# 批处理模式，显示指定PID的进程
top -b -p 1234
```

### 5.3 显示线程
```bash
# 显示线程
top -H

# 显示指定进程的线程
top -H -p 1234
```

## 6. 高级用法

### 6.1 组合选项
```bash
# 批处理模式，显示5次，刷新间隔2秒
top -b -n 5 -d 2

# 显示指定用户的进程，显示线程
top -u user -H
```

### 6.2 与其他命令结合使用
```bash
# 实时监控进程并保存到文件
top -b -n 10 > top.log

# 监控特定进程
top -p $(pgrep bash)
```

### 6.3 自动化脚本
```bash
#!/bin/bash

# 自动化监控系统
top_monitor() {
    local options=${1:-""}
    local iterations=${2:-"5"}
    local delay=${3:-"2"}
    
    echo "监控系统状态，迭代次数: $iterations, 刷新间隔: $delay 秒"
    top -b $options -n $iterations -d $delay
}

# 使用函数
top_monitor
top_monitor "-u root" 10 1
top_monitor "-H" 5 3
```

## 7. 常见问题与解决方案

### 7.1 问题：top: command not found
**解决方案**：安装 procps
- Ubuntu/Debian: `apt install procps`
- CentOS/RHEL: `yum install procps`

### 7.2 问题：无法显示所有CPU
**解决方案**：按下数字键 1

### 7.3 问题：无法终止进程
**解决方案**：按下 k 键，输入进程PID

### 7.4 问题：top 命令执行缓慢
**解决方案**：系统负载较高，这是正常现象

### 7.5 问题：无法显示完整命令
**解决方案**：按下 c 键
