# reboot - 重启系统

## 1. 命令简介
reboot 命令用于重启系统，是 Linux 系统中常用的系统管理命令之一。

## 2. 命令语法
```bash
reboot [选项]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -f | 强制重启 |
| -i | 关闭所有网络接口后重启 |
| -n | 不执行 sync 操作 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 重启系统
```bash
reboot
```

### 4.2 强制重启
```bash
reboot -f
```

### 4.3 关闭所有网络接口后重启
```bash
reboot -i
```

## 5. 注意事项
- reboot 命令用于重启系统
- reboot 命令需要 root 权限
- reboot 命令是一个常用的系统管理工具，用于系统重启
