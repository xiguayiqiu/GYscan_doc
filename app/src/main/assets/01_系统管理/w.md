# w - 显示登录用户信息

## 1. 命令简介
w 命令用于显示登录用户信息，是 Linux 系统中常用的系统监控命令之一。它可以显示当前登录的用户、他们的进程以及系统负载等信息。

## 2. 命令语法
```bash
w [选项] [用户]
```

## 3. 常用选项

| 选项 | 说明 |
|------|------|
| -h, --help | 显示帮助信息 |
| --version | 显示版本信息 |
| -f, --from | 显示登录来源 |
| -h, --no-header | 不显示头部信息 |
| -i, --ip-addr | 显示IP地址而不是主机名 |
| -s, --short | 显示简短格式 |
| -u, --no-current | 不显示当前命令 |

## 4. 输出格式

| 字段 | 说明 |
|------|------|
| USER | 用户名 |
| TTY | 终端设备 |
| FROM | 登录来源 |
| LOGIN@ | 登录时间 |
| IDLE | 空闲时间 |
| JCPU | 该终端所有进程使用的CPU时间 |
| PCPU | 当前进程使用的CPU时间 |
| WHAT | 当前执行的命令 |

## 5. 实战示例

### 5.1 基本用法
```bash
# 显示所有登录用户信息
w

# 显示指定用户的信息
w user

# 显示简短格式
w -s
```

### 5.2 显示登录来源
```bash
# 显示登录来源
w -f

# 显示IP地址而不是主机名
w -i
```

### 5.3 不显示头部信息
```bash
# 不显示头部信息
w -h
```

## 6. 高级用法

### 6.1 组合选项
```bash
# 显示简短格式，显示IP地址
w -si

# 不显示头部信息，不显示当前命令
w -hu
```

### 6.2 与其他命令结合使用
```bash
# 显示登录用户信息并管道到其他命令
w | grep user

# 监控登录用户信息
watch w
```

### 6.3 自动化脚本
```bash
#!/bin/bash

# 自动化显示登录用户信息
display_users() {
    local options=${1:-""}
    local user=${2:-""}
    
    echo "登录用户信息:"
    w $options $user
}

# 使用函数
display_users
display_users "-s"
display_users "-f" "user"
```

## 7. 常见问题与解决方案

### 7.1 问题：w: command not found
**解决方案**：安装 procps
- Ubuntu/Debian: `apt install procps`
- CentOS/RHEL: `yum install procps`

### 7.2 问题：无法显示用户信息
**解决方案**：确保系统正常运行

### 7.3 问题：w 命令执行缓慢
**解决方案**：系统负载较高，这是正常现象
