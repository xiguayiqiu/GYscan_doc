# init - 初始化系统

## 1. 命令简介
init 命令用于初始化系统，是 Linux 系统中常用的系统管理命令之一。

## 2. 命令语法
```bash
init [运行级别]
```

## 3. 常用运行级别
| 运行级别 | 说明 |
|---------|------|
| 0 | 关机 |
| 1 | 单用户模式 |
| 2 | 多用户模式（无网络） |
| 3 | 多用户模式（有网络） |
| 4 | 保留 |
| 5 | 图形界面模式 |
| 6 | 重启 |

## 4. 实战示例

### 4.1 切换到单用户模式
```bash
init 1
```

### 4.2 切换到多用户模式
```bash
init 3
```

### 4.3 切换到图形界面模式
```bash
init 5
```

### 4.4 关机
```bash
init 0
```

## 5. 注意事项
- init 命令用于初始化系统和切换运行级别
- init 命令需要 root 权限
- 在 systemd 系统中，init 命令已被 systemctl 命令替代
- init 命令是一个常用的系统管理工具，用于系统初始化
