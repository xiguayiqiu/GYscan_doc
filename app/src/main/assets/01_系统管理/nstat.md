# nstat命令

## 命令简介

`nstat`命令用于显示网络统计信息，它从`/proc/net/netstat`和`/proc/net/snmp`文件中读取数据，显示各种网络协议的统计信息。

## 语法

```bash
nstat [选项]
```

## 选项

| 选项 | 描述 |
|------|------|
| `-a` | 显示所有统计信息 |
| `-z` | 显示零值统计信息 |
| `-d` | 显示差值（与上次显示的差值） |
| `-h` | 显示帮助信息 |
| `-V` | 显示版本信息 |

## 示例

### 显示所有网络统计信息

```bash
nstat -a
```

### 显示零值统计信息

```bash
nstat -z
```

### 显示差值统计信息

```bash
nstat -d
```

### 显示特定协议的统计信息

```bash
# 显示TCP协议的统计信息
nstat | grep TCP

# 显示UDP协议的统计信息
nstat | grep UDP
```

## 高级用法

### 监控网络统计信息变化

```bash
# 每1秒显示一次网络统计信息的变化
watch -n 1 nstat -d
```

### 保存网络统计信息到文件

```bash
# 保存当前网络统计信息到文件
nstat -a > network_stats.txt

# 一段时间后再次保存，比较变化
sleep 60 && nstat -a > network_stats_60s.txt
```

## 常见问题

### 问题：nstat命令不存在

**原因**：系统中没有安装iproute2包。

**解决方法**：安装iproute2包，例如在Ubuntu上执行`sudo apt install iproute2`。

### 问题：nstat命令显示的信息过多，难以阅读

**原因**：nstat命令显示的统计信息非常详细，包含了各种网络协议的统计数据。

**解决方法**：使用grep命令过滤感兴趣的统计信息。

## 命令对比

| 命令 | 功能 | 特点 |
|------|------|------|
| `nstat` | 显示网络统计信息 | 显示详细的网络协议统计信息 |
| `netstat` | 显示网络状态 | 显示网络连接、路由表等信息 |
| `ss` | 显示套接字统计 | 显示套接字的详细信息 |
| `ifconfig` | 显示网络接口信息 | 显示网络接口的配置和统计信息 |

## 注意事项

1. **权限要求**：nstat命令不需要特殊权限，普通用户也可以使用。

2. **输出解释**：nstat命令的输出包含大量统计信息，需要了解每个字段的含义才能正确解读。

3. **统计数据来源**：nstat命令从`/proc/net/netstat`和`/proc/net/snmp`文件中读取数据。

4. **适用场景**：nstat命令适用于网络性能分析、网络故障排查和网络安全审计等场景。

5. **性能影响**：nstat命令的执行对系统性能影响很小。
