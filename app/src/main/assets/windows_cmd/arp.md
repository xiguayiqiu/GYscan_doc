# arp 命令

## 命令说明
地址解析协议

## 语法
```
arp [选项]
```

## 功能
显示和修改地址解析协议（ARP）缓存。

## 选项
- `-a`：显示所有接口的ARP缓存
- `-g`：与 `-a` 相同
- `-d`：删除指定的ARP条目
- `-s`：添加静态ARP条目

## 示例
```
arp -a
arp -d 192.168.1.1
arp -s 192.168.1.1 00-11-22-33-44-55
```

## 注意事项
- 用于显示和修改ARP缓存
- ARP缓存存储IP地址和MAC地址的映射关系
- 静态ARP条目可以提高网络安全性
- 适用于网络故障排除
