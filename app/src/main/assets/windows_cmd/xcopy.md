# xcopy 命令

## 命令说明
复制文件和目录

## 语法
```
xcopy [源文件或目录] [目标文件或目录] [选项]
```

## 功能
复制文件和目录，支持复制子目录。

## 选项
- `/s`：复制子目录（不包括空目录）
- `/e`：复制子目录（包括空目录）
- `/i`：如果目标不存在且复制多个文件，则假定目标是目录
- `/y`：覆盖现有文件而不提示
- `/r`：复制只读文件
- `/k`：复制文件时保留属性

## 示例
```
xcopy C:\source D:\backup /s /e
xcopy file.txt D:\docs /y
xcopy C:\data D:\archive /i /k
```

## 注意事项
- 比 `copy` 命令功能更强大
- 支持复制整个目录结构
- 可以使用通配符（如 `*.txt`）来复制多个文件
- 可以使用相对路径或绝对路径
