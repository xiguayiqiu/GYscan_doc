# for 命令

## 命令说明
循环命令

## 语法
```
for %变量 in (集合) do 命令
for /l %变量 in (开始,步长,结束) do 命令
for /f %变量 in (文件) do 命令
for /d %变量 in (目录) do 命令
```

## 功能
在批处理文件或命令提示符中执行循环操作。

## 选项
- `/l`：执行数字范围循环
- `/f`：处理文件内容或命令输出
- `/d`：处理目录

## 示例
```
for %i in (*.txt) do echo %i
for /l %i in (1,1,10) do echo %i
for /f %i in (file.txt) do echo %i
```

## 注意事项
- 在批处理文件中，变量名需要使用双百分号（如 `%%i`）
- 可以遍历文件、目录或执行数字循环
- 是批处理文件中非常强大的命令
