# erase命令

## 命令简介

`erase`命令是Windows中的一个命令，用于删除文件，与`del`命令功能相同。

## 语法

```cmd
erase [选项] 文件名
```

## 选项

| 选项 | 描述 |
|------|------|
| `/p` | 删除前提示确认 |
| `/f` | 强制删除只读文件 |
| `/s` | 递归删除子目录中的文件 |
| `/q` | 安静模式，不提示确认 |
| `/a` | 根据文件属性删除文件 |

## 示例

### 删除单个文件

```cmd
# 删除单个文件
erase file.txt

# 删除前提示确认
erase /p file.txt

# 强制删除只读文件
erase /f read-only.txt
```

### 删除多个文件

```cmd
# 删除指定类型的文件
erase *.txt

# 删除多个类型的文件
erase *.doc *.docx

# 安静模式删除文件
erase /q *.tmp
```

### 递归删除文件

```cmd
# 递归删除子目录中的文件
erase /s *.tmp

# 递归删除并提示确认
erase /s /p *.log
```

### 根据文件属性删除文件

```cmd
# 删除隐藏文件
erase /a:h *.txt

# 删除系统文件
erase /a:s *.sys
```

## 高级用法

### 批量删除文件

```cmd
# 创建一个批处理文件
@echo off

# 批量删除临时文件
for %%f in (*.tmp *.temp *.log) do (
    erase /q %%f
)

echo 临时文件已删除
```

### 安全删除文件

```cmd
# 使用erase命令删除文件（可恢复）
erase file.txt

# 使用cipher命令安全删除文件（不可恢复）
cipher /w:file.txt
```

## 常见问题

### 问题：无法删除文件

**原因**：可能是权限不足，或者文件正在被使用。

**解决方法**：以管理员身份运行命令提示符，确保文件没有被其他程序使用。

### 问题：删除文件后无法恢复

**原因**：`erase`命令删除的文件会被放入回收站，但可以通过命令行选项跳过回收站。

**解决方法**：使用`erase`命令时要谨慎，重要文件请先备份。

## 命令对比

| 命令 | 功能 | 特点 |
|------|------|------|
| `erase` | 删除文件 | 基本的文件删除命令 |
| `del` | 删除文件 | 与erase命令功能相同 |
| `rmdir` | 删除目录 | 用于删除目录 |
| `rd` | 删除目录 | 与rmdir命令功能相同 |

## 注意事项

1. **权限要求**：删除文件需要对文件有删除权限。

2. **文件状态**：删除文件时，确保文件没有被其他程序使用。

3. **回收站**：默认情况下，`erase`命令删除的文件会被放入回收站，可以通过`/f`选项跳过回收站。

4. **确认提示**：默认情况下，删除多个文件时会提示确认，可以使用`/q`选项跳过提示。

5. **适用场景**：`erase`命令适用于删除文件的场景，与`del`命令功能相同。
