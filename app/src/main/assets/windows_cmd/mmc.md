# mmc 命令

## 命令说明
微软管理控制台

## 语法
```
mmc [选项]
mmc [控制台文件]
```

## 功能
打开微软管理控制台（MMC），用于管理系统设置和服务。

## 选项
- `/a`：以作者模式打开控制台
- `/s`：以用户模式打开控制台（无窗口）
- `/64`：使用 64 位版本的 MMC
- `/32`：使用 32 位版本的 MMC

## 示例
```
mmc
mmc compmgmt.msc
mmc diskmgmt.msc
```

## 注意事项
- MMC 是 Windows 系统的管理工具集合
- 可以通过加载不同的管理单元来管理系统的不同方面
- 常见的管理单元包括计算机管理、磁盘管理、服务等
