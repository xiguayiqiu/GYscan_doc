# prnport 命令

## 命令说明
打印机端口

## 语法
```
prnport [选项]
```

## 功能
管理打印机端口。

## 选项
- `-l`：列出所有打印机端口
- `-a`：添加打印机端口
- `-d`：删除打印机端口
- `-t`：设置打印机端口

## 示例
```
prnport -l
prnport -a -r "IP_192.168.1.100" -h "192.168.1.100" -o raw
prnport -d -r "IP_192.168.1.100"
```

## 注意事项
- 用于管理打印机端口
- 可以添加、删除和列出打印机端口
- 需要管理员权限才能使用
- 适用于系统管理员管理网络打印机端口
