# pbadmin 命令

## 命令说明
Phone Book管理

## 语法
```
pbadmin [选项]
```

## 功能
管理Phone Book配置。

## 选项
- `/list`：列出Phone Book条目
- `/add`：添加新的Phone Book条目
- `/delete`：删除Phone Book条目
- `/update`：更新Phone Book条目

## 示例
```
pbadmin /list
pbadmin /add "VPN Connection" "rasphone.pbk"
pbadmin /delete "VPN Connection"
```

## 注意事项
- 用于管理Phone Book配置
- 可以添加、删除和更新Phone Book条目
- 与远程访问服务相关
