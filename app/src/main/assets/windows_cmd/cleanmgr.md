# cleanmgr命令

## 命令简介

`cleanmgr`命令是Windows中的一个命令，用于磁盘清理。

## 语法

```cmd
cleanmgr [选项]
```

## 选项

| 选项 | 描述 |
|------|------|
| `/d <驱动器>` | 指定驱动器 |
| `/sageset:<编号>` | 设置清理选项 |
| `/sagerun:<编号>` | 运行指定的清理选项 |
| `/TUNE` | 优化系统性能 |
| `/LOWDISK` | 当磁盘空间低时运行 |

## 示例

### 磁盘清理

```cmd
# 启动磁盘清理
cleanmgr

# 清理C盘
cleanmgr /d C:

# 设置清理选项
cleanmgr /sageset:1

# 运行指定的清理选项
cleanmgr /sagerun:1
```

## 高级用法

### 批量磁盘清理

```cmd
# 创建一个批处理文件
@echo off

# 批量磁盘清理
echo 清理C盘
cleanmgr /d C: /sagerun:1

echo 清理D盘
cleanmgr /d D: /sagerun:1

echo 操作完成
```

## 常见问题

### 问题：cleanmgr命令执行失败

**原因**：可能是权限不足，或者驱动器不存在。

**解决方法**：以管理员身份运行命令提示符，确保驱动器存在。

### 问题：磁盘清理失败

**原因**：可能是磁盘被占用，或者权限不足。

**解决方法**：确保磁盘未被占用，以管理员身份运行命令提示符。

1. **权限要求**：使用`cleanmgr`命令需要管理员权限。

2. **驱动器**：需要指定有效的驱动器。

3. **适用场景**：`cleanmgr`命令适用于磁盘清理的场景。
