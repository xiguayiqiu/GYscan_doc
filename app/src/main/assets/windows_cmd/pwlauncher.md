# pwlauncher 命令

## 命令说明
密码启动器

## 语法
```
pwlauncher [选项]
```

## 功能
管理密码启动器设置。

## 选项
- `/enable`：启用密码启动器
- `/disable`：禁用密码启动器
- `/query`：查询当前设置

## 示例
```
pwlauncher /enable
pwlauncher /disable
pwlauncher /query
```

## 注意事项
- 用于管理密码启动器设置
- 密码启动器允许用户使用Windows Hello或其他生物识别方式登录
- 需要管理员权限才能使用
- 适用于Windows 10及以后版本
