# add命令

## 命令简介

`add`命令是Windows磁盘管理工具中的一个命令，用于添加项目，如添加镜像到虚拟磁盘、添加卷到磁盘等。

## 语法

```cmd
add [选项] [参数]
```

## 选项

| 选项 | 描述 |
|------|------|
| `mirror` | 添加镜像到虚拟磁盘 |
| `volume` | 添加卷到磁盘 |

## 示例

### 添加镜像到虚拟磁盘

```cmd
# 打开磁盘管理工具
diskpart

# 选择虚拟磁盘
select vdisk file="D:\VirtualDisk.vhdx"

# 附加虚拟磁盘
attach vdisk

# 添加镜像
add mirror

# 退出磁盘管理工具
exit
```

### 添加卷到磁盘

```cmd
# 打开磁盘管理工具
diskpart

# 选择磁盘
select disk 0

# 创建简单卷
create volume simple size=10000

# 选择卷
select volume 1

# 添加卷到磁盘
add volume

# 退出磁盘管理工具
exit
```

## 高级用法

### 创建带镜像的卷

```cmd
# 打开磁盘管理工具
diskpart

# 选择第一个磁盘
select disk 0

# 创建带镜像的卷
create volume simple size=10000 mirror disk=1

# 退出磁盘管理工具
exit
```

## 常见问题

### 问题：无法添加镜像

**原因**：可能是磁盘空间不足，或者虚拟磁盘格式不支持镜像。

**解决方法**：确保有足够的磁盘空间，使用支持镜像的虚拟磁盘格式。

### 问题：无法添加卷

**原因**：可能是磁盘空间不足，或者磁盘分区表已满。

**解决方法**：确保有足够的磁盘空间，检查磁盘分区表是否已满。

## 命令对比

| 命令 | 功能 | 特点 |
|------|------|------|
| `add` | 添加项目 | 用于添加镜像或卷 |
| `create` | 创建项目 | 用于创建新的磁盘、分区或卷 |
| `attach` | 附加虚拟磁盘 | 用于附加虚拟磁盘文件 |

## 注意事项

1. **权限要求**：使用diskpart工具需要管理员权限。

2. **磁盘空间**：添加镜像或卷时需要确保有足够的磁盘空间。

3. **操作顺序**：在使用add命令前，通常需要先选择要操作的对象。

4. **适用场景**：`add`命令适用于在磁盘管理中添加镜像或卷的场景。
