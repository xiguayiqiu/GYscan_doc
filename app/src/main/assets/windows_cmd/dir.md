# dir 命令

## 命令说明
显示目录内容

## 语法
```
dir [目录路径] [选项]
```

## 功能
显示指定目录中的文件和子目录列表。

## 选项
- `/a`：显示所有文件，包括隐藏文件和系统文件
- `/s`：显示指定目录及其子目录中的文件
- `/b`：以简洁格式显示（只显示文件名）
- `/l`：以小写形式显示
- `/o`：按指定顺序排序

## 示例
```
dir
dir C:\Windows
dir /a /s *.txt
dir /b
```

## 注意事项
- 不带参数时，显示当前目录的内容
- 可以使用通配符（如 `*.txt`）来过滤文件
- 可以使用相对路径或绝对路径
