# prnjobs 命令

## 命令说明
打印作业

## 语法
```
prnjobs [选项]
```

## 功能
管理打印作业。

## 选项
- `-l`：列出打印作业
- `-x`：取消打印作业
- `-m`：暂停打印作业
- `-r`：恢复打印作业

## 示例
```
prnjobs -l -p "HP LaserJet"
prnjobs -x -p "HP LaserJet" -j 123
prnjobs -m -p "HP LaserJet" -j 123
```

## 注意事项
- 用于管理打印作业
- 可以列出、取消、暂停和恢复打印作业
- 需要适当的权限才能管理打印作业
- 适用于系统管理员管理打印队列
