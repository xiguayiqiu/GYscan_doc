# forfiles 命令

## 命令说明
文件循环

## 语法
```
forfiles [选项]
```

## 功能
在目录中搜索文件并对找到的文件执行命令。

## 选项
- `/p`：指定搜索目录
- `/m`：指定文件掩码
- `/s`：递归搜索子目录
- `/d`：指定日期条件
- `/c`：指定要执行的命令

## 示例
```
forfiles /p C:\Data /m *.txt /c "cmd /c echo @file"
forfiles /s /m *.log /d -7 /c "cmd /c del @file"
```

## 注意事项
- 用于批量处理文件
- 可以根据文件的修改日期、名称等条件进行过滤
- 是批处理文件中处理文件的强大工具
