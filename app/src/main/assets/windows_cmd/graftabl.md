# graftabl命令

## 命令简介

`graftabl`命令是Windows中的一个命令，用于启用或禁用扩展字符集显示。

## 语法

```cmd
graftabl [选项]
```

## 选项

| 选项 | 描述 |
|------|------|
| `936` | 启用简体中文字符集 |
| `437` | 启用美国英语字符集 |
| `/status` | 显示当前字符集状态 |

## 示例

### 启用或禁用扩展字符集显示

```cmd
# 启用简体中文字符集
graftabl 936

# 启用美国英语字符集
graftabl 437

# 显示当前字符集状态
graftabl /status

# 在批处理文件中使用
@echo off
echo 启用简体中文字符集
graftabl 936
echo 操作完成
```

## 高级用法

### 批量启用或禁用扩展字符集显示

```cmd
# 创建一个批处理文件
@echo off

# 批量启用或禁用扩展字符集显示
echo 显示当前字符集状态
graftabl /status
echo 启用简体中文字符集
graftabl 936
echo 新的字符集状态
graftabl /status
echo 操作完成
```

**原因**：可能是权限不足，或者字符集不存在。
