# whoami命令

## 命令简介

`whoami`命令是Windows中的一个命令，用于显示当前用户。

## 语法

```cmd
whoami [选项]
```

## 选项

| 选项 | 描述 |
|------|------|
| `/all` | 显示当前用户的所有信息 |
| `/upn` | 显示用户主体名称 |
| `/fqdn` | 显示完全限定域名 |
| `/user` | 显示用户信息 |
| `/groups` | 显示用户所属的组 |
| `/priv` | 显示用户权限 |

## 示例

### 显示当前用户

```cmd
# 显示当前用户
whoami

# 显示当前用户的所有信息
whoami /all

# 显示用户主体名称
whoami /upn

# 显示完全限定域名
whoami /fqdn

# 显示用户信息
whoami /user

# 显示用户所属的组
whoami /groups

# 显示用户权限
whoami /priv

# 在批处理文件中使用
@echo off
echo 当前用户:
whoami
echo 操作完成
```

## 高级用法

### 批量显示用户信息

```cmd
# 创建一个批处理文件
@echo off

# 批量显示用户信息
echo 当前用户:
whoami
echo 用户所属的组:
whoami /groups
echo 操作完成
```

## 常见问题

### 问题：whoami命令执行失败

**原因**：可能是权限不足，或者命令语法错误。

**解决方法**：以管理员身份运行命令提示符，确保命令语法正确。

### 问题：显示用户信息失败

**原因**：可能是权限不足。

**解决方法**：以管理员身份运行命令提示符。

## 命令对比

| 命令 | 功能 | 特点 |
|------|------|------|
| `whoami` | 显示当前用户 | 用于显示当前用户 |
| `echo %USERNAME%` | 显示当前用户名 | 用于在批处理中显示当前用户名 |
| `net user` | 用户管理 | 用于管理用户 |

## 注意事项

1. **权限要求**：使用`whoami`命令需要管理员权限。

2. **适用场景**：`whoami`命令适用于显示当前用户的场景。
