# clean命令

## 命令简介

`clean`命令是Windows中的一个命令，用于清理磁盘。

## 语法

```cmd
clean [选项]
```

## 选项

| 选项 | 描述 |
|------|------|
| `all` | 清理整个磁盘 |
| `free` | 只清理可用空间 |
| `zeros` | 用零填充已清理的空间 |
| `secure` | 安全清理（多次覆盖） |

## 示例

### 清理磁盘

```cmd
# 清理磁盘
clean

# 清理整个磁盘
clean all

# 只清理可用空间
clean free

# 用零填充已清理的空间
clean zeros

# 安全清理
clean secure
```

## 高级用法

### 批量清理磁盘

```cmd
# 创建一个批处理文件
@echo off

# 批量清理磁盘
echo 清理C盘
diskpart
select disk 0
clean

echo 操作完成
```

## 常见问题

### 问题：clean命令执行失败

**原因**：可能是权限不足，或者磁盘被占用。

**解决方法**：以管理员身份运行命令提示符，确保磁盘未被占用。

### 问题：磁盘清理失败

**原因**：可能是磁盘损坏，或者权限不足。

**解决方法**：确保磁盘未损坏，以管理员身份运行命令提示符。

## 注意事项

1. **权限要求**：使用`clean`命令需要管理员权限。

2. **数据丢失**：清理磁盘会删除所有数据，请谨慎使用。

3. **适用场景**：`clean`命令适用于清理磁盘的场景。
