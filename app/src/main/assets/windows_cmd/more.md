# more 命令

## 命令说明
分页显示文件内容

## 语法
```
more [文件]
[命令] | more
```

## 功能
分页显示文件内容或命令输出。

## 选项
- `/c`：显示页面前先清屏
- `/p`：扩展换页符
- `/s`：将多个空行压缩为一个空行
- `/t:n`：将制表符转换为 n 个空格

## 示例
```
more file.txt
dir | more
more /s /p log.txt
```

## 注意事项
- 按空格键显示下一页
- 按 Enter 键显示下一行
- 按 Q 键退出
- 可以与管道 `|` 一起使用，分页显示其他命令的输出
