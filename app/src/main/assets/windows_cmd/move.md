# move 命令

## 命令说明
移动文件

## 语法
```
move [源文件] [目标文件]
move [源文件1] [源文件2] ... [目标目录]
```

## 功能
移动一个或多个文件到指定位置，也可以用于重命名文件。

## 选项
- `/y`：覆盖现有文件而不提示
- `/-y`：覆盖现有文件前提示

## 示例
```
move file1.txt file2.txt
move *.txt backup\
move /y old.txt new.txt
```

## 注意事项
- 可以使用通配符（如 `*.txt`）来移动多个文件
- 可以使用相对路径或绝对路径
- 如果目标是目录，源文件将被移动到该目录中
- 如果源和目标在同一目录，此命令将重命名文件
