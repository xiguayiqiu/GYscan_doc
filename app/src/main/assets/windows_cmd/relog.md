# relog命令

## 命令简介

`relog`命令是Windows中的一个命令，用于重新记录性能计数器数据。

## 语法

```cmd
relog [选项] <输入文件> [输出文件]
```

## 选项

| 选项 | 描述 |
|------|------|
| `/f <格式>` | 指定输出格式 |
| `/t <间隔>` | 指定采样间隔 |
| `/o <文件>` | 指定输出文件 |
| `/b <开始时间>` | 指定开始时间 |
| `/e <结束时间>` | 指定结束时间 |
| `/c <计数器>` | 指定计数器 |
| `/q` | 显示计数器列表 |
| `/y` | 覆盖现有文件 |

## 示例

### 重新记录性能计数器数据

```cmd
# 重新记录性能计数器数据
relog data.blg -o data.csv

# 指定输出格式
relog data.blg -f CSV -o data.csv

# 指定采样间隔
relog data.blg -t 5 -o data.csv

# 指定开始和结束时间
relog data.blg -b 09:00:00 -e 10:00:00 -o data.csv

# 指定计数器
relog data.blg -c "\Processor(_Total)\% Processor Time" -o data.csv

# 显示计数器列表
relog data.blg -q

# 覆盖现有文件
relog data.blg -y -o data.csv
```

## 高级用法

### 批量重新记录性能计数器数据

```cmd
# 创建一个批处理文件
```

**解决方法**：以管理员身份运行命令提示符，确保文件存在。
