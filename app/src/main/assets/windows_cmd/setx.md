# setx 命令

## 命令说明
设置环境变量（持久）

## 语法
```
setx [变量名] [值] [选项]
```

## 功能
设置系统环境变量，持久化保存。

## 选项
- `/m`：设置系统环境变量（需要管理员权限）
- `/s`：指定远程计算机
- `/u`：指定用户名
- `/p`：指定密码

## 示例
```
setx PATH "%PATH%;C:\MyTools"
setx JAVA_HOME C:\Java\jdk1.8.0 /m
setx TEMP D:\Temp
```

## 注意事项
- 用于设置持久化的环境变量
- 设置的环境变量在新的命令窗口中生效
- 需要管理员权限才能设置系统环境变量
- 可以设置用户环境变量和系统环境变量
