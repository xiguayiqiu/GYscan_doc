# mstsc 命令

## 命令说明
远程桌面连接

## 语法
```
mstsc [选项]
```

## 功能
启动远程桌面连接，用于连接到远程计算机。

## 选项
- `/v`：指定远程计算机
- `/admin`：以管理员身份连接
- `/f`：全屏模式
- `/w`：指定宽度
- `/h`：指定高度
- `/public`：公共模式

## 示例
```
mstsc
mstsc /v:192.168.1.100
mstsc /v:servername /admin /f
```

## 注意事项
- 用于远程连接到其他Windows计算机
- 需要远程计算机启用远程桌面功能
- 可以保存连接配置文件
- 支持全屏和窗口模式
