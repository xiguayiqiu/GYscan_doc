# change命令

## 命令简介

`change`命令是Windows中的一个命令，用于更改终端服务器会话设置。

## 语法

```cmd
change [选项]
```

## 选项

| 选项 | 描述 |
|------|------|
| `logon` | 更改终端服务器登录设置 |
| `port` | 更改终端服务器端口设置 |
| `user` | 更改终端服务器用户设置 |

## 示例

### 更改终端服务器登录设置

```cmd
# 启用终端服务器登录
change logon /enable

# 禁用终端服务器登录
change logon /disable

# 显示终端服务器登录状态
change logon /query
```

### 更改终端服务器端口设置

```cmd
# 显示终端服务器端口设置
change port

# 更改终端服务器端口设置
change port /disable com1
change port /enable com1
```

### 更改终端服务器用户设置

```cmd
# 显示终端服务器用户设置
change user /query

# 启用终端服务器用户模式
change user /execute

# 禁用终端服务器用户模式
change user /install
```

## 高级用法

### 批量更改设置

```cmd
# 创建一个批处理文件
@echo off

# 批量更改终端服务器设置
echo 启用终端服务器登录
change logon /enable

echo 启用终端服务器用户模式
change user /execute

echo 设置完成
```

## 常见问题

### 问题：change命令执行失败

**原因**：可能是权限不足，或者终端服务器服务未运行。

**解决方法**：以管理员身份运行命令提示符，确保终端服务器服务已启动。

### 问题：命令不可用

**原因**：可能是系统中没有安装终端服务器组件。

**解决方法**：确保系统中已安装终端服务器组件。

## 命令对比

| 命令 | 功能 | 特点 |
|------|------|------|
| `change` | 更改终端服务器设置 | 用于更改终端服务器会话设置 |
| `tscon` | 连接到终端服务器会话 | 用于连接到终端服务器会话 |
| `tsdiscon` | 断开终端服务器会话 | 用于断开终端服务器会话 |

## 注意事项

1. **权限要求**：使用`change`命令需要管理员权限。

2. **终端服务器**：`change`命令适用于终端服务器环境。

3. **适用场景**：`change`命令适用于更改终端服务器会话设置的场景。
