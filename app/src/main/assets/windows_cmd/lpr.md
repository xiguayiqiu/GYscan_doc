# lpr命令

## 命令简介

`lpr`命令是Windows中的一个命令，用于行式打印机。

## 语法

```cmd
lpr [选项] [文件]
```

## 选项

| 选项 | 描述 |
|------|------|
| `-S <服务器>` | 指定打印服务器 |
| `-P <打印机>` | 指定打印机 |
| `-C <类>` | 指定类 |
| `-J <作业>` | 指定作业名称 |
| `-# <份数>` | 指定份数 |
| `-o <选项>` | 指定选项 |

## 示例

### 发送打印作业

```cmd
# 发送文件到默认打印机
lpr file.txt

# 发送文件到指定打印机
lpr -P printer file.txt

# 发送文件到远程服务器上的打印机
lpr -S server -P printer file.txt

# 发送多份文件
lpr -# 3 file.txt
```

## 高级用法

### 批量发送打印作业

```cmd
# 创建一个批处理文件
@echo off

# 批量发送打印作业
echo 发送文件到默认打印机
lpr file1.txt

echo 发送文件到指定打印机
lpr -P printer file2.txt

echo 操作完成
```

## 常见问题

### 问题：lpr命令执行失败

**原因**：可能是权限不足，或者打印机不可用。

**解决方法**：确保打印机可用，以管理员身份运行命令提示符。

### 问题：打印作业发送失败

**原因**：可能是打印机未连接，或者权限不足。

**解决方法**：确保打印机已连接，以管理员身份运行命令提示符。

## 命令对比

| 命令 | 功能 | 特点 |
|------|------|------|
| `lpr` | 行式打印机 | 用于发送打印作业 |
| `lpq` | 打印队列 | 用于显示打印队列 |
| `print` | 打印文件 | 用于打印文件 |

## 注意事项

1. **权限要求**：使用`lpr`命令需要管理员权限。

2. **打印机**：需要确保打印机可用。

3. **适用场景**：`lpr`命令适用于发送打印作业的场景。
