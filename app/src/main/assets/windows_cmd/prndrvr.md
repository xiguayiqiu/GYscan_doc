# prndrvr 命令

## 命令说明
打印机驱动程序

## 语法
```
prndrvr [选项]
```

## 功能
管理打印机驱动程序。

## 选项
- `-l`：列出所有打印机驱动程序
- `-a`：添加打印机驱动程序
- `-d`：删除打印机驱动程序
- `-v`：显示详细信息

## 示例
```
prndrvr -l
prndrvr -a -m "HP LaserJet" -i "C:\drivers\hp.inf"
prndrvr -d -m "HP LaserJet"
```

## 注意事项
- 用于管理打印机驱动程序
- 可以添加、删除和列出驱动程序
- 需要管理员权限才能使用
- 适用于系统管理员管理打印机驱动
