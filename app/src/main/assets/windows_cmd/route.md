# route命令

## 命令简介

`route`命令是Windows中的一个命令，用于路由。

## 语法

```cmd
route [选项] [命令]
```

## 命令

| 命令 | 描述 |
|------|------|
| `print` | 显示路由表 |
| `add` | 添加路由 |
| `delete` | 删除路由 |
| `change` | 修改路由 |

## 选项

| 选项 | 描述 |
|------|------|
| `-f` | 清除路由表 |
| `-p` | 永久添加路由 |

## 示例

### 路由管理

```cmd
# 显示路由表
route print

# 添加路由
route add 192.168.2.0 mask 255.255.255.0 192.168.1.1

# 永久添加路由
route add 192.168.2.0 mask 255.255.255.0 192.168.1.1 -p

# 删除路由
route delete 192.168.2.0

# 修改路由
route change 192.168.2.0 mask 255.255.255.0 192.168.1.2

# 清除路由表
route print -f
```

## 高级用法

### 批量路由管理

```cmd
# 创建一个批处理文件
@echo off

# 批量路由管理
echo 显示路由表
route print

echo 添加路由
route add 192.168.2.0 mask 255.255.255.0 192.168.1.1

echo 操作完成
```

## 常见问题

### 问题：route命令执行失败

**原因**：可能是权限不足，或者命令语法错误。

**解决方法**：以管理员身份运行命令提示符，确保命令语法正确。

### 问题：路由管理失败

**原因**：可能是网络连接问题，或者权限不足。

**解决方法**：确保网络连接正常，以管理员身份运行命令提示符。

## 命令对比

| 命令 | 功能 | 特点 |
|------|------|------|
| `route` | 路由 | 用于路由管理 |
| `netstat -r` | 显示路由表 | 用于显示路由表 |
| `ipconfig` | 显示网络配置 | 用于网络配置管理 |

## 注意事项

1. **权限要求**：使用`route`命令需要管理员权限。

2. **网络连接**：需要确保网络连接正常。

3. **适用场景**：`route`命令适用于路由管理的场景。
