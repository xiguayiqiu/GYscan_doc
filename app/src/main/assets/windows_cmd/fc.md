# fc 命令

## 命令说明
文件比较

## 语法
```
fc [options] <file1> <file2>
```

## 功能
比较两个文件的内容。

## 选项
- `/a`：以ASCII模式比较
- `/b`：以二进制模式比较
- `/c`：忽略大小写
- `/l`：显示行号
- `/n`：显示差异的行号
- `/w`：忽略空白

## 示例
```
# 以ASCII模式比较两个文件

fc /a file1.txt file2.txt

# 以二进制模式比较两个文件

fc /b file1.bin file2.bin

# 忽略大小写比较

fc /c file1.txt file2.txt
```

## 注意事项
- 用于比较两个文件的内容
- 支持ASCII和二进制模式
- 可以忽略大小写和空白
- 适用于文件比较和差异分析
