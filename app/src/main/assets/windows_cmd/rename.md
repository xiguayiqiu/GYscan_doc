# rename命令

## 命令简介

`rename`命令是Windows中的一个命令，用于重命名文件或目录，与`ren`命令功能相同。

## 语法

```cmd
rename [旧文件名] [新文件名]
```

## 选项

`rename`命令没有额外的选项参数。

## 示例

### 重命名单个文件

```cmd
# 重命名单个文件
rename file.txt newfile.txt

# 重命名带空格的文件
rename "old file.txt" "new file.txt"
```

### 重命名多个文件

```cmd
# 重命名多个文件（添加前缀）
for %i in (*.txt) do rename "%i" "prefix_%i"

# 重命名多个文件（替换部分名称）
for %i in (*old*.txt) do rename "%i" "%i:old=new%"
```

### 重命名目录

```cmd
# 重命名目录
rename olddir newdir

# 重命名带空格的目录
rename "old dir" "new dir"
```

## 高级用法

### 批量重命名文件

```cmd
# 创建一个批处理文件
@echo off

# 批量重命名文件（添加日期前缀）
for %%i in (*.txt) do (
    set "filename=%%~ni"
    set "extension=%%~xi"
    rename "%%i" "%date:~0,4%-%date:~5,2%-%date:~8,2%_%%i"
)

echo 文件重命名完成
```

### 重命名文件扩展名

```cmd
# 重命名文件扩展名
rename *.txt *.md

# 批量重命名文件扩展名
for %i in (*.txt) do rename "%i" "%~ni.md"
```

## 常见问题

### 问题：无法重命名文件

**原因**：可能是权限不足，或者文件正在被使用。

**解决方法**：以管理员身份运行命令提示符，确保文件没有被其他程序使用。

### 问题：重命名后文件消失

**原因**：可能是重命名命令语法错误，或者目标文件已存在。

**解决方法**：检查重命名命令的语法是否正确，确保目标文件不存在。

## 命令对比

| 命令 | 功能 | 特点 |
|------|------|------|
| `rename` | 重命名文件或目录 | 与ren命令功能相同 |
| `ren` | 重命名文件或目录 | 基本的文件重命名命令 |
| `move` | 移动文件或目录 | 可以同时移动和重命名文件 |

## 注意事项

1. **权限要求**：重命名文件或目录需要对文件或目录有相应的权限。

2. **文件状态**：重命名文件时，确保文件没有被其他程序使用。

3. **目标文件**：如果目标文件已存在，`rename`命令会提示确认覆盖。

4. **路径格式**：Windows中使用反斜杠`\`作为路径分隔符。

5. **适用场景**：`rename`命令适用于重命名文件或目录的场景，与`ren`命令功能相同。
