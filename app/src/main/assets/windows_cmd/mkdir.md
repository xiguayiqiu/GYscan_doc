# mkdir命令

## 命令简介

`mkdir`命令是Windows中的一个命令，用于创建目录。

## 语法

```cmd
mkdir [选项] 目录名
```

## 选项

| 选项 | 描述 |
|------|------|
| `/p` | 创建父目录（如果不存在） |

## 示例

### 创建单个目录

```cmd
# 创建单个目录
mkdir test

# 创建多个目录
mkdir dir1 dir2 dir3
```

### 创建嵌套目录

```cmd
# 创建嵌套目录（需要父目录存在）
mkdir test\subdir

# 创建嵌套目录（自动创建父目录）
mkdir /p test\subdir\subsubdir
```

### 创建带空格的目录

```cmd
# 创建带空格的目录
mkdir "my directory"

# 创建带空格的嵌套目录
mkdir /p "my directory\sub directory"
```

## 高级用法

### 批量创建目录

```cmd
# 创建一个批处理文件
@echo off

# 批量创建目录
for /l %%i in (1,1,5) do (
    mkdir "dir%%i"
)

echo 目录创建完成
```

### 创建日期命名的目录

```cmd
# 创建以当前日期命名的目录
mkdir %date:~0,4%-%date:~5,2%-%date:~8,2%
```

## 常见问题

### 问题：无法创建目录

**原因**：可能是权限不足，或者路径不存在。

**解决方法**：以管理员身份运行命令提示符，使用`/p`选项创建父目录。

### 问题：目录名包含特殊字符

**原因**：目录名包含了Windows不允许的特殊字符。

**解决方法**：使用合法的目录名，避免使用特殊字符。

## 命令对比

| 命令 | 功能 | 特点 |
|------|------|------|
| `mkdir` | 创建目录 | 基本的目录创建命令 |
| `md` | 创建目录 | 与mkdir命令功能相同 |
| `rmdir` | 删除目录 | 用于删除目录 |
| `rd` | 删除目录 | 与rmdir命令功能相同 |

## 注意事项

1. **权限要求**：创建目录需要对父目录有写入权限。

2. **路径格式**：Windows中使用反斜杠`\`作为路径分隔符。

3. **目录名长度**：Windows中目录名长度不能超过255个字符。

4. **特殊字符**：目录名不能包含以下字符：`\ / : * ? " < > |`

5. **适用场景**：`mkdir`命令适用于创建目录的场景。
