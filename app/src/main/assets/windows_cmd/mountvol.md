# mountvol 命令

## 命令说明
挂载卷

## 语法
```
mountvol [驱动器:] [卷名] [选项]
```

## 功能
管理卷的挂载点。

## 选项
- `/L`：显示卷标
- `/P`：永久卸载卷
- `/D`：删除挂载点
- `/N`：不自动挂载卷

## 示例
```
mountvol
mountvol E: \\?\Volume{12345678-1234-1234-1234-123456789012}\
mountvol E: /D
```

## 注意事项
- 用于管理卷的挂载点
- 可以创建、删除和列出挂载点
- 需要管理员权限才能使用
- 适用于磁盘管理和挂载网络卷
