# format 命令

## 命令说明
格式化

## 语法
```
format [驱动器:] [选项]
```

## 功能
格式化磁盘。

## 选项
- `/q`：快速格式化
- `/y`：自动确认
- `/fs:文件系统`：指定文件系统（FAT、FAT32、NTFS）
- `/a:分配单元大小`：指定分配单元大小
- `/v:卷标`：指定卷标

## 示例
```
format C: /fs:NTFS /q
format D: /fs:FAT32 /v:Data
format E: /y
```

## 注意事项
- 用于格式化磁盘
- 格式化会删除所有数据
- 快速格式化只删除文件表，不检查坏扇区
- 适用于准备新磁盘或清除现有磁盘
