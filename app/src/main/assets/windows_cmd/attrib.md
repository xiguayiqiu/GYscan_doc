# attrib 命令

## 命令说明
查看或修改文件属性

## 语法
```
attrib [+r|-r] [+a|-a] [+s|-s] [+h|-h] [文件或目录]
```

## 功能
显示或修改文件和目录的属性。

## 选项
- `+r`：设置只读属性
- `-r`：清除只读属性
- `+a`：设置存档属性
- `-a`：清除存档属性
- `+s`：设置系统属性
- `-s`：清除系统属性
- `+h`：设置隐藏属性
- `-h`：清除隐藏属性

## 示例
```
attrib +h secret.txt
attrib -r readme.txt
attrib
```

## 注意事项
- 不带参数时，显示当前目录中所有文件的属性
- 可以使用通配符（如 `*.txt`）来操作多个文件
