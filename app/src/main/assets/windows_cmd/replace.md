# replace 命令

## 命令说明
替换文件

## 语法
```
replace [源文件] [目标文件] [选项]
```

## 功能
替换或添加文件到指定位置。

## 选项
- `/a`：添加文件，不替换现有文件
- `/p`：替换前提示
- `/r`：替换只读文件
- `/s`：替换子目录中的文件
- `/w`：等待用户插入磁盘

## 示例
```
replace file.txt C:\backup
replace /a newfile.txt C:\data
replace /s *.txt D:\docs
```

## 注意事项
- 可以使用通配符（如 `*.txt`）来替换多个文件
- 可以使用相对路径或绝对路径
- 默认情况下，只替换已存在的文件
