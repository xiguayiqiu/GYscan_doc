# color命令

## 命令简介

`color`命令是Windows中的一个命令，用于设置命令窗口颜色。

## 语法

```cmd
color [颜色代码]
```

## 颜色代码

| 代码 | 颜色 |
|------|------|
| 0 | 黑色 |
| 1 | 蓝色 |
| 2 | 绿色 |
| 3 | 湖蓝色 |
| 4 | 红色 |
| 5 | 紫色 |
| 6 | 黄色 |
| 7 | 白色 |
| 8 | 灰色 |
| 9 | 淡蓝色 |
| A | 淡绿色 |
| B | 淡湖蓝色 |
| C | 淡红色 |
| D | 淡紫色 |
| E | 淡黄色 |
| F | 亮白色 |

## 示例

### 设置命令窗口颜色

```cmd
# 设置命令窗口颜色为蓝色背景白色文字
color 17

# 设置命令窗口颜色为红色背景黄色文字
color 4E

# 恢复默认颜色
color

# 在批处理文件中使用
@echo off
color 0A
echo 绿色文字黑色背景
echo 操作完成
```

## 高级用法

### 批量设置命令窗口颜色

```cmd
# 创建一个批处理文件
@echo off

# 批量设置命令窗口颜色
echo 蓝色背景白色文字
color 17
pause

echo 红色背景黄色文字
color 4E
pause

echo 绿色背景黑色文字
color 20
pause

echo 恢复默认颜色
color
echo 操作完成
```

## 常见问题

### 问题：color命令执行失败

**原因**：可能是权限不足，或者颜色代码错误。

**解决方法**：以管理员身份运行命令提示符，确保颜色代码正确。

### 问题：设置颜色失败

**原因**：可能是颜色代码错误，或者命令语法错误。

**解决方法**：确保颜色代码正确，确保命令语法正确。

## 命令对比

| 命令 | 功能 | 特点 |
|------|------|------|
| `color` | 设置命令窗口颜色 | 用于设置命令窗口的背景和文字颜色 |
| `cmd /t` | 设置命令窗口颜色 | 用于启动命令提示符时设置颜色 |
| `mode con` | 配置控制台 | 用于配置控制台的各种设置 |

## 注意事项

1. **权限要求**：使用`color`命令需要管理员权限。

2. **颜色代码**：需要确保颜色代码正确。

3. **适用场景**：`color`命令适用于设置命令窗口颜色的场景。
