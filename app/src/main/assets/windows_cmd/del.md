# del 命令

## 命令说明
删除文件

## 语法
```
del [文件]
del [选项] [文件]
```

## 功能
删除一个或多个文件。

## 选项
- `/f`：强制删除只读文件
- `/s`：删除指定目录及其子目录中的文件
- `/q`：安静模式，删除时不提示
- `/a`：根据属性删除文件

## 示例
```
del file.txt
del /s *.tmp
del /q /f oldfile.txt
```

## 注意事项
- 可以使用通配符（如 `*.txt`）来删除多个文件
- 此命令只能删除文件，不能删除目录
- 删除的文件不会进入回收站，而是直接被删除
