# copy 命令

## 命令说明
复制文件

## 语法
```
copy [源文件] [目标文件]
copy [源文件1] [源文件2] ... [目标目录]
```

## 功能
复制一个或多个文件到指定位置。

## 选项
- `/y`：覆盖现有文件而不提示
- `/-y`：覆盖现有文件前提示
- `/a`：以 ASCII 模式复制
- `/b`：以二进制模式复制

## 示例
```
copy file1.txt file2.txt
copy *.txt backup\
copy /y old.txt new.txt
```

## 注意事项
- 可以使用通配符（如 `*.txt`）来复制多个文件
- 可以使用相对路径或绝对路径
- 如果目标是目录，源文件将被复制到该目录中
