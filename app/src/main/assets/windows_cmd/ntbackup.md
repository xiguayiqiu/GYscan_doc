# ntbackup 命令

## 命令说明
NT备份

## 语法
```
ntbackup [选项]
```

## 功能
执行备份和还原操作。

## 选项
- `/backup`：执行备份操作
- `/restore`：执行还原操作
- `/schedule`：管理备份计划
- `/help`：显示帮助信息

## 示例
```
ntbackup /backup
ntbackup /restore
ntbackup /schedule
```

## 注意事项
- 用于执行系统备份和还原操作
- 在 Windows Vista 及以后版本中已被 Windows 备份和还原中心替代
- 支持备份整个系统或特定文件和文件夹
