# shutdown命令

## 命令简介

`shutdown`命令是Windows中的一个命令，用于关闭或重启计算机。

## 语法

```cmd
shutdown [选项]
```

## 选项

| 选项 | 描述 |
|------|------|
| `/s` | 关闭计算机 |
| `/r` | 重启计算机 |
| `/h` | 休眠计算机 |
| `/t <秒>` | 设置延迟时间 |
| `/c "消息"` | 显示关闭消息 |
| `/a` | 取消关闭操作 |

## 示例

### 关闭计算机

```cmd
# 立即关闭计算机
shutdown /s

# 5分钟后关闭计算机
shutdown /s /t 300

# 关闭计算机并显示消息
shutdown /s /t 60 /c "计算机将在1分钟后关闭"
```

### 重启计算机

```cmd
# 立即重启计算机
shutdown /r

# 5分钟后重启计算机
shutdown /r /t 300

# 重启计算机并显示消息
shutdown /r /t 60 /c "计算机将在1分钟后重启"
```

### 休眠计算机

```cmd
# 休眠计算机
shutdown /h
```

### 取消关闭操作

```cmd
# 取消关闭操作
shutdown /a
```

## 高级用法

### 批量管理计算机

```cmd
# 创建一个批处理文件
@echo off

# 批量管理计算机
echo 5分钟后重启计算机
shutdown /r /t 300 /c "系统维护需要重启"

echo 操作完成
```

## 常见问题

### 问题：shutdown命令执行失败

**原因**：可能是权限不足，或者语法错误。

**解决方法**：以管理员身份运行命令提示符，确保语法正确。

### 问题：关闭操作未执行

**原因**：可能是延迟时间过长，或者被其他操作取消。

**解决方法**：检查延迟时间，确保没有其他操作取消关闭。

## 注意事项

1. **权限要求**：使用`shutdown`命令需要管理员权限。

2. **延迟时间**：可以通过`/t`选项设置关闭或重启的延迟时间。

3. **适用场景**：`shutdown`命令适用于关闭或重启计算机的场景。
