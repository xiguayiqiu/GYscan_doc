# shutdown 命令

## 命令说明
关闭系统

## 语法
```
shutdown [选项]
```

## 功能
关闭、重启或注销系统。

## 选项
- `/s`：关闭系统
- `/r`：重启系统
- `/l`：注销当前用户
- `/h`：休眠系统
- `/t`：设置延迟时间（秒）
- `/c`：添加注释

## 示例
```
shutdown /s /t 60 /c "系统将在1分钟后关闭"
shutdown /r /t 0
shutdown /l
```

## 注意事项
- 用于关闭、重启或注销系统
- 可以设置延迟时间和添加注释
- 需要管理员权限才能关闭或重启系统
- 适用于系统维护和管理
