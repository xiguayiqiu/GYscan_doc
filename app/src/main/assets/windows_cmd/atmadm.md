# atmadm命令

## 命令简介

`atmadm`命令是Windows中的一个命令，用于管理ATM（Asynchronous Transfer Mode）网络。

## 语法

```cmd
atmadm [选项]
```

## 选项

| 选项 | 描述 |
|------|------|
| `/c` | 显示ATM连接 |
| `/d` | 显示ATM设备 |
| `/l` | 显示ATM接口 |
| `/s` | 显示ATM统计信息 |

## 示例

### 显示ATM设备

```cmd
# 显示ATM设备
atmadm /d
```

### 显示ATM接口

```cmd
# 显示ATM接口
atmadm /l
```

### 显示ATM连接

```cmd
# 显示ATM连接
atmadm /c
```

### 显示ATM统计信息

```cmd
# 显示ATM统计信息
atmadm /s
```

## 高级用法

### 综合查询

```cmd
# 显示所有ATM相关信息
atmadm /d && atmadm /l && atmadm /c && atmadm /s
```

## 常见问题

### 问题：atmadm命令不可用

**原因**：可能是系统中没有安装ATM相关组件，或者命令已被弃用。

**解决方法**：确保系统中已安装ATM相关组件，或者使用其他网络管理工具。

### 问题：命令执行失败

**原因**：可能是权限不足，或者ATM服务未运行。

**解决方法**：以管理员身份运行命令提示符，确保ATM服务已启动。

## 命令对比

| 命令 | 功能 | 特点 |
|------|------|------|
| `atmadm` | 管理ATM网络 | 专门用于ATM网络管理 |
| `ipconfig` | 显示网络配置 | 用于显示IP网络配置 |
| `netsh` | 网络配置命令 | 功能更强大的网络配置工具 |

## 注意事项

1. **命令可用性**：`atmadm`命令可能在较新的Windows版本中已被弃用。

2. **权限要求**：使用`atmadm`命令需要管理员权限。

3. **适用场景**：`atmadm`命令适用于管理ATM网络的场景。
