# edit 命令

## 命令说明
编辑文件

## 语法
```
edit [options] [filename]
```

## 功能
编辑文本文件。

## 选项
- `/b`：以二进制模式编辑
- `/h`：以十六进制模式编辑
- `/r`：以只读模式打开
- `/s`：以小缓冲区模式打开

## 示例
```
# 编辑文件

edit file.txt

# 以二进制模式编辑

edit /b file.bin

# 以十六进制模式编辑

edit /h file.bin
```

## 注意事项
- 用于编辑文本文件
- 支持二进制和十六进制模式
- 适用于命令行环境下的文件编辑
- 在现代Windows系统中可能已被其他编辑器替代
