# extend命令

## 命令简介

`extend`命令是Windows中的一个命令，用于扩展分区。

## 语法

```cmd	extend [选项]
```

## 选项

| 选项 | 描述 |
|------|------|
| `size=<大小>` | 指定扩展大小（以MB为单位） |
| `disk=<磁盘编号>` | 指定磁盘编号 |
| `noerr` | 忽略错误 |

## 示例

### 扩展分区

```cmd
# 扩展分区	extend

# 扩展分区指定大小	extend size=1024

# 扩展分区指定磁盘	extend disk=0

# 扩展分区指定大小和磁盘	extend size=1024 disk=0
```

## 高级用法

### 批量扩展分区

```cmd
# 创建一个批处理文件
@echo off

# 批量扩展分区
echo 扩展分区
diskpart
select disk 0
select partition 1	extend size=1024
exit

echo 操作完成
```

## 常见问题

### 问题：extend命令执行失败

**原因**：可能是权限不足，或者磁盘空间不足。

**解决方法**：以管理员身份运行命令提示符，确保磁盘有足够的空间。

### 问题：分区扩展失败

**原因**：可能是分区被占用，或者权限不足。

**解决方法**：确保分区未被占用，以管理员身份运行命令提示符。

## 命令对比

| 命令 | 功能 | 特点 |
|------|------|------|
| `extend` | 扩展分区 | 用于扩展分区 |
| `shrink` | 收缩分区 | 用于收缩分区 |
| `create partition` | 创建分区 | 用于创建分区 |

## 注意事项

1. **权限要求**：使用`extend`命令需要管理员权限。

2. **磁盘空间**：需要确保磁盘有足够的空间。

3. **适用场景**：`extend`命令适用于扩展分区的场景。
