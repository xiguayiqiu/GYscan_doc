# robocopy 命令

## 命令说明
高级文件复制

## 语法
```
robocopy [源目录] [目标目录] [文件] [选项]
```

## 功能
高级文件复制工具，支持镜像复制、增量复制等功能。

## 选项
- `/s`：复制子目录（不包括空目录）
- `/e`：复制子目录（包括空目录）
- `/purge`：删除目标中不存在于源中的文件
- `/mir`：镜像复制（相当于 `/e` 和 `/purge`）
- `/xo`：仅复制比目标文件新的文件
- `/xc`：排除已更改的文件
- `/xn`：排除较新的文件
- `/xd`：排除指定目录

## 示例
```
robocopy C:\source D:\backup /mir
robocopy C:\docs D:\archive *.docx /s
robocopy C:\data D:\backup /e /xo
```

## 注意事项
- 比 `copy` 命令功能更强大
- 支持多线程复制，速度更快
- 可以复制文件权限和属性
- 可以生成复制日志
