# robocopy命令

## 命令简介

`robocopy`命令是Windows中的一个高级文件复制命令，用于复制文件和目录，支持镜像复制、增量复制等高级功能。

## 语法

```cmd
robocopy [源目录] [目标目录] [文件] [选项]
```

## 选项

| 选项 | 描述 |
|------|------|
| `/s` | 复制子目录（不包括空目录） |
| `/e` | 复制子目录（包括空目录） |
| `/z` | 可重启模式复制 |
| `/zb` | 可重启模式复制，失败时使用备份模式 |
| `/r:<n>` | 失败时重试次数（默认1000000） |
| `/w:<n>` | 重试间隔（默认30秒） |
| `/tbd` | 等待共享名释放 |
| `/np` | 不显示复制进度百分比 |
| `/v` | 详细输出 |
| `/mt:<n>` | 使用多线程复制（默认8线程） |
| `/xd` | 排除指定目录 |
| `/xf` | 排除指定文件 |
| `/xo` | 只复制比目标文件新的文件 |
| `/xn` | 只复制比目标文件旧的文件 |
| `/xc` | 排除已更改的文件 |
| `/xn` | 排除较新的文件 |
| `/xl` | 排除较旧的文件 |
| `/xx` | 排除多余的文件和目录 |
| `/purge` | 删除目标中多余的文件和目录 |
| `/mir` | 镜像复制（相当于 /e /purge） |

## 示例

### 基本复制

```cmd
# 复制文件和子目录（不包括空目录）
robocopy C:\Source D:\Destination /s

# 复制文件和子目录（包括空目录）
robocopy C:\Source D:\Destination /e
```

### 镜像复制

```cmd
# 镜像复制（保持目标与源一致）
robocopy C:\Source D:\Destination /mir
```

### 增量复制

```cmd
# 只复制比目标文件新的文件
robocopy C:\Source D:\Destination /xo

# 只复制比目标文件旧的文件
robocopy C:\Source D:\Destination /xn
```

### 多线程复制

```cmd
# 使用16线程复制
robocopy C:\Source D:\Destination /mt:16
```

### 排除文件和目录

```cmd
# 排除指定目录
robocopy C:\Source D:\Destination /xd temp logs

# 排除指定文件
robocopy C:\Source D:\Destination /xf *.tmp *.log
```

## 高级用法

### 备份文件

```cmd
# 备份文件到外部磁盘
robocopy C:\Users\Username\Documents E:\Backup /e /zb /mt:8 /xd "AppData" "Local Settings"
```

### 复制大文件

```cmd
# 复制大文件，使用可重启模式
robocopy C:\Source D:\Destination /z /mt:16
```

### 监控文件变化

```cmd
# 监控文件变化并自动复制
robocopy C:\Source D:\Destination /mir /mon:1 /mot:1
```

## 常见问题

### 问题：复制速度慢

**原因**：可能是网络带宽限制，或者文件数量过多。

**解决方法**：使用多线程复制（`/mt`选项），增加线程数量。

### 问题：复制失败

**原因**：可能是权限不足，或者文件正在被使用。

**解决方法**：以管理员身份运行命令提示符，确保文件没有被其他程序使用，使用可重启模式（`/z`选项）。

## 命令对比

| 命令 | 功能 | 特点 |
|------|------|------|
| `robocopy` | 高级文件复制 | 功能强大，支持镜像复制、增量复制等 |
| `copy` | 复制文件 | 基本的文件复制命令 |
| `xcopy` | 复制文件和目录 | 功能较robocopy弱 |
| `move` | 移动文件或目录 | 用于移动文件 |

## 注意事项

1. **权限要求**：复制文件需要对源和目标目录有相应的权限。

2. **文件状态**：复制文件时，确保文件没有被其他程序使用。

3. **多线程复制**：使用多线程复制（`/mt`选项）可以提高复制速度，但可能会增加系统负担。

4. **镜像复制**：使用镜像复制（`/mir`选项）会删除目标中多余的文件和目录，使用时要谨慎。

5. **适用场景**：`robocopy`命令适用于需要高级文件复制功能的场景，如备份、镜像等。
