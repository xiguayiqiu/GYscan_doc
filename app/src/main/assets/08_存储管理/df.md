# df - 显示文件系统空间使用情况

## 1. 命令简介
df 命令用于显示文件系统空间使用情况，是 Linux 系统中常用的存储管理命令之一。

## 2. 命令语法
```bash
df [选项] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -h | 以人类可读的格式显示 |
| -a | 显示所有文件系统 |
| -T | 显示文件系统类型 |
| -i | 显示inode使用情况 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 显示文件系统空间使用情况
```bash
df -h
```

### 4.2 显示所有文件系统
```bash
df -a
```

### 4.3 显示文件系统类型
```bash
df -T
```

### 4.4 显示inode使用情况
```bash
df -i
```

## 5. 注意事项
- df 命令用于显示文件系统空间使用情况，包括已用空间、可用空间和挂载点
- df 命令是一个常用的系统管理工具，用于监控文件系统空间
