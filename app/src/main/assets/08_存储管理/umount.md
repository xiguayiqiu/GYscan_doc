# umount - 卸载文件系统

## 1. 命令简介
umount 命令用于卸载文件系统，是 Linux 系统中常用的存储管理命令之一。

## 2. 命令语法
```bash
umount [选项] [设备或挂载点]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -a | 卸载所有已挂载的文件系统 |
| -f | 强制卸载 |
| -l | 延迟卸载 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 卸载文件系统
```bash
umount /mnt
```

### 4.2 强制卸载
```bash
umount -f /mnt
```

### 4.3 延迟卸载
```bash
umount -l /mnt
```

### 4.4 卸载所有已挂载的文件系统
```bash
umount -a
```

## 5. 注意事项
- umount 命令用于卸载文件系统，将设备从文件系统树中移除
- umount 命令需要 root 权限
- umount 命令是一个常用的系统管理工具，用于文件系统管理
