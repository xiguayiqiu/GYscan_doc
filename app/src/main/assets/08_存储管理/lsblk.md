# lsblk - 显示块设备信息

## 1. 命令简介
lsblk 命令用于显示块设备信息，是 Linux 系统中常用的存储管理命令之一。

## 2. 命令语法
```bash
lsblk [选项]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -a | 显示所有设备 |
| -b | 以字节为单位显示大小 |
| -f | 显示文件系统信息 |
| -m | 显示权限信息 |
| -o | 自定义输出列 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 显示块设备信息
```bash
lsblk
```

### 4.2 显示所有设备
```bash
lsblk -a
```

### 4.3 显示文件系统信息
```bash
lsblk -f
```

### 4.4 自定义输出列
```bash
lsblk -o NAME,SIZE,TYPE,MOUNTPOINT
```

## 5. 注意事项
- lsblk 命令用于显示块设备信息，包括磁盘、分区和挂载点
- lsblk 命令是一个常用的系统管理工具，用于查看存储设备信息
