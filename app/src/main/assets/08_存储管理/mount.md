# mount - 挂载文件系统

## 1. 命令简介
mount 命令用于挂载文件系统，是 Linux 系统中常用的存储管理命令之一。

## 2. 命令语法
```bash
mount [选项] [设备] [挂载点]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -a | 挂载所有在 /etc/fstab 中定义的文件系统 |
| -t | 指定文件系统类型 |
| -o | 指定挂载选项 |
| -r | 以只读方式挂载 |
| -w | 以读写方式挂载 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 挂载文件系统
```bash
mount /dev/sda1 /mnt
```

### 4.2 以只读方式挂载
```bash
mount -r /dev/sda1 /mnt
```

### 4.3 挂载所有文件系统
```bash
mount -a
```

### 4.4 指定文件系统类型
```bash
mount -t ext4 /dev/sda1 /mnt
```

## 5. 注意事项
- mount 命令用于挂载文件系统，将设备连接到文件系统树
- mount 命令需要 root 权限
- mount 命令是一个常用的系统管理工具，用于文件系统管理
