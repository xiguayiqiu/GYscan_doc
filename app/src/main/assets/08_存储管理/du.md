# du - 显示目录大小

## 1. 命令简介
du 命令用于显示目录大小，是 Linux 系统中常用的存储管理命令之一。

## 2. 命令语法
```bash
du [选项] [目录]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -h | 以人类可读的格式显示 |
| -s | 显示总大小 |
| -a | 显示所有文件和目录 |
| -c | 显示总计 |
| -d | 指定深度 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 显示目录大小
```bash
du -h directory
```

### 4.2 显示总大小
```bash
du -s directory
```

### 4.3 显示所有文件和目录
```bash
du -a directory
```

### 4.4 显示总计
```bash
du -c directory
```

## 5. 注意事项
- du 命令用于显示目录大小，计算目录及其子目录的大小
- du 命令是一个常用的系统管理工具，用于监控目录大小
