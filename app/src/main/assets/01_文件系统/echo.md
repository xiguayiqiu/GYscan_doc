# echo - 显示文本或变量值

## 1. 命令简介
echo 命令用于显示文本或变量值，是 Linux 系统中最常用的命令之一。

## 2. 命令语法
```bash
echo [选项] [文本]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -n | 不输出换行符 |
| -e | 启用转义字符 |

## 4. 常用转义字符
| 转义字符 | 说明 |
|----------|------|
| \n | 换行 |
| \t | 制表符 |
| \\ | 反斜杠 |
| \" | 双引号 |
| \' | 单引号 |

## 5. 实战示例

### 5.1 显示文本
```bash
echo "Hello, World!"
```

### 5.2 显示变量值
```bash
name="John"
echo "Hello, $name!"
```

### 5.3 不输出换行符
```bash
echo -n "Hello, "
echo "World!"
```

### 5.4 启用转义字符
```bash
echo -e "Hello\nWorld"
```

### 5.5 显示制表符
```bash
echo -e "Name\tAge"
echo -e "John\t25"
```

### 5.6 显示命令执行结果
```bash
echo "Current directory: $(pwd)"
```

## 6. 注意事项
- 文本如果包含空格，需要用引号括起来
- 使用 -e 选项可以启用转义字符
- 可以使用 $() 或 `` 来执行命令并显示其结果
