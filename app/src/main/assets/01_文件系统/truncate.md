# truncate - 截断文件

## 1. 命令简介
truncate 命令用于截断文件，是 Linux 系统中常用的文件系统命令之一。

## 2. 命令语法
```bash
truncate [选项] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -s | 指定文件大小 |
| -c | 不创建文件 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 大小单位
| 单位 | 说明 |
|------|------|
| B | 字节 |
| K | 千字节 |
| M | 兆字节 |
| G | 吉字节 |
| T | 太字节 |

## 5. 实战示例

### 5.1 截断文件到指定大小
```bash
truncate -s 1M file.txt
```

### 5.2 清空文件
```bash
truncate -s 0 file.txt
```

### 5.3 不创建文件
```bash
truncate -c -s 1M file.txt
```

## 6. 注意事项
- truncate 命令用于截断文件到指定大小
- 如果文件大小大于指定大小，文件会被截断
- 如果文件大小小于指定大小，文件会被扩展
- 扩展的部分会填充零
- truncate 命令是一个常用的文件操作工具
