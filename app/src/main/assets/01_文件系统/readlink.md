# readlink - 读取链接

## 1. 命令简介
readlink 命令用于读取符号链接的目标，是 Linux 系统中常用的文件系统命令之一。

## 2. 命令语法
```bash
readlink [选项] [链接]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -f | 跟随所有符号链接，返回绝对路径 |
| -e | 跟随所有符号链接，检查文件是否存在 |
| -n | 不输出换行符 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 读取符号链接的目标
```bash
readlink link.txt
```

### 4.2 跟随所有符号链接，返回绝对路径
```bash
readlink -f link.txt
```

### 4.3 不输出换行符
```bash
readlink -n link.txt
```

## 5. 注意事项
- readlink 命令用于读取符号链接的目标
- 使用 -f 选项可以跟随所有符号链接，返回绝对路径
- readlink 命令是一个常用的文件系统工具，用于处理符号链接
