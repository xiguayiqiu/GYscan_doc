# mknod - 创建特殊文件

## 1. 命令简介
mknod 命令用于创建特殊文件，是 Linux 系统中常用的文件系统命令之一。

## 2. 命令语法
```bash
mknod [选项] [文件] [类型] [主设备号] [次设备号]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -m | 设置文件权限 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 文件类型
| 类型 | 说明 |
|------|------|
| b | 块设备 |
| c | 字符设备 |
| p | 命名管道 |

## 5. 实战示例

### 5.1 创建字符设备文件
```bash
mknod /dev/null c 1 3
```

### 5.2 创建块设备文件
```bash
mknod /dev/sda b 8 0
```

### 5.3 创建命名管道
```bash
mknod pipe p
```

## 6. 注意事项
- mknod 命令用于创建特殊文件，如设备文件和命名管道
- 创建设备文件需要 root 权限
- 主设备号和次设备号用于标识设备
- mknod 命令是一个常用的文件系统工具，用于创建特殊文件
