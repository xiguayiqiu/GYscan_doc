# more - 分页显示文件内容

## 1. 命令简介
more 命令用于分页显示文件内容，是 Linux 系统中常用的文件系统命令之一。

## 2. 命令语法
```bash
more [选项] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -d | 显示帮助信息 |
| -f | 计数长行 |
| -l | 不暂停显示 |
| -p | 清屏后显示 |
| -s | 压缩空行 |
| -u | 不显示下划线 |

## 4. 常用操作
| 操作 | 说明 |
|------|------|
| 空格键 | 向下翻页 |
| Enter | 向下翻一行 |
| b | 向上翻页 |
| q | 退出 |

## 5. 实战示例

### 5.1 分页显示文件内容
```bash
more file.txt
```

### 5.2 压缩空行
```bash
more -s file.txt
```

### 5.3 清屏后显示
```bash
more -p file.txt
```

## 6. 注意事项
- more 命令用于分页显示文件内容
- more 命令只能向前浏览文件，不能向后浏览
- less 命令是 more 命令的增强版本，功能更强大
- more 命令是一个常用的文本查看工具
