# split - 分割文件

## 1. 命令简介
split 命令用于分割文件，是 Linux 系统中常用的文件系统命令之一。

## 2. 命令语法
```bash
split [选项] [文件] [输出前缀]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -b | 按字节大小分割 |
| -l | 按行数分割 |
| -a | 指定后缀长度 |
| -d | 使用数字后缀 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 按字节大小分割
```bash
split -b 1M file.txt
```

### 4.2 按行数分割
```bash
split -l 100 file.txt
```

### 4.3 使用数字后缀
```bash
split -d file.txt
```

### 4.4 指定输出前缀
```bash
split file.txt part_
```

## 5. 注意事项
- split 命令用于分割文件，将大文件分割成多个小文件
- 使用 -b 选项可以按字节大小分割
- 使用 -l 选项可以按行数分割
- split 命令是一个常用的文件处理工具，用于分割大文件
