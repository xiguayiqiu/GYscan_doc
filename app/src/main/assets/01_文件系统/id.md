# id - 显示用户和组信息

## 1. 命令简介
id 命令用于显示当前用户或指定用户的用户ID、组ID和所属组信息，是 Linux 系统中最常用的命令之一。

## 2. 命令语法
```bash
id [选项] [用户名]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -u | 仅显示用户ID |
| -g | 仅显示主组ID |
| -G | 仅显示所有组ID |
| -n | 显示名称而不是ID |
| -r | 显示实际ID而不是有效ID |

## 4. 实战示例

### 4.1 显示当前用户的信息
```bash
id
```

### 4.2 显示指定用户的信息
```bash
id username
```

### 4.3 仅显示用户ID
```bash
id -u
```

### 4.4 仅显示主组ID
```bash
id -g
```

### 4.5 仅显示所有组ID
```bash
id -G
```

### 4.6 显示名称而不是ID
```bash
id -n
```

## 5. 注意事项
- id 命令不需要任何参数就可以执行，默认显示当前用户的信息
- 使用 -n 选项可以显示名称而不是ID，更加直观
- id 命令的输出可以用于脚本中，以获取用户和组的信息
