# cd - 切换目录

## 1. 命令简介
切换当前工作目录，是 Linux 中最基本的导航命令。

## 2. 命令语法
```bash
cd [目录路径]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| ~ | 切换到用户主目录 |
| - | 切换到上一个工作目录 |
| . | 当前目录 |
| .. | 上级目录 |

## 4. 实战示例

### 4.1 切换到主目录
```bash
cd ~
```

### 4.2 切换到上级目录
```bash
cd ..
```

### 4.3 切换到指定目录
```bash
cd /path/to/directory
```

### 4.4 切换到上一个工作目录
```bash
cd -
```

## 5. 注意事项
- 当不指定路径时，默认切换到用户主目录
- 使用绝对路径（以 / 开头）或相对路径指定目标目录
- 目录名称中包含空格时，需要用引号包裹或使用转义字符
