# chmod - 修改文件权限

## 1. 命令简介
chmod 命令用于修改文件或目录的权限，是 Linux 系统中最常用的文件权限管理命令之一。

## 2. 命令语法
```bash
chmod [选项] 权限 文件
```

## 3. 权限表示方法

### 3.1 数字表示法
| 数字 | 权限 |
|------|------|
| 0 | 无权限 |
| 1 | 执行权限 |
| 2 | 写入权限 |
| 3 | 写入和执行权限 |
| 4 | 读取权限 |
| 5 | 读取和执行权限 |
| 6 | 读取和写入权限 |
| 7 | 读取、写入和执行权限 |

### 3.2 符号表示法
| 符号 | 说明 |
|------|------|
| u | 所有者 |
| g | 组 |
| o | 其他 |
| a | 所有 |
| + | 添加权限 |
| - | 移除权限 |
| = | 设置权限 |
| r | 读取权限 |
| w | 写入权限 |
| x | 执行权限 |

## 4. 常用选项
| 选项 | 说明 |
|------|------|
| -R | 递归修改目录及其内容的权限 |
| -v | 显示修改的权限 |
| -c | 只显示修改的权限 |

## 5. 实战示例

### 5.1 使用数字表示法修改权限
```bash
chmod 755 file.txt
```

### 5.2 使用符号表示法修改权限
```bash
chmod u+rwx file.txt
chmod g+rx file.txt
chmod o+rx file.txt
```

### 5.3 递归修改目录及其内容的权限
```bash
chmod -R 755 directory
```

### 5.4 移除权限
```bash
chmod u-x file.txt
```

### 5.5 设置权限
```bash
chmod u=rwx file.txt
```

### 5.6 显示修改的权限
```bash
chmod -v 755 file.txt
```

## 6. 注意事项
- chmod 命令需要文件的所有者或 root 权限才能修改文件权限
- 使用 -R 选项可以递归修改目录及其内容的权限
- 权限分为所有者、组和其他三个类别，每个类别可以有读取、写入和执行权限
