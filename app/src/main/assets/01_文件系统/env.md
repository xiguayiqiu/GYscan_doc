# env - 显示环境变量

## 1. 命令简介
env 命令用于显示当前环境变量，或在指定环境中执行命令，是 Linux 系统中最常用的命令之一。

## 2. 命令语法
```bash
env [选项] [变量=值] [命令]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -i | 忽略环境变量，使用空环境 |
| -u | 移除指定的环境变量 |

## 4. 实战示例

### 4.1 显示当前环境变量
```bash
env
```

### 4.2 在指定环境中执行命令
```bash
env VAR=value command
```

### 4.3 忽略环境变量，使用空环境执行命令
```bash
env -i command
```

### 4.4 移除指定的环境变量后执行命令
```bash
env -u VAR command
```

## 5. 注意事项
- env 命令不需要任何参数就可以执行，默认显示当前环境变量
- 使用 env 命令可以在不修改当前环境的情况下，临时设置环境变量执行命令
- env 命令的输出可以用于脚本中，以获取环境变量信息
