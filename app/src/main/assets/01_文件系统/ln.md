# ln - 创建链接

## 1. 命令简介
ln 命令用于创建文件链接，是 Linux 系统中常用的命令之一。

## 2. 命令语法
```bash
ln [选项] 源文件 目标文件
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -s | 创建符号链接 |
| -f | 强制创建链接，覆盖已存在的文件 |
| -i | 交互式创建链接，提示确认 |
| -v | 显示创建的链接 |

## 4. 链接类型
| 类型 | 说明 |
|------|------|
| 硬链接 | 指向文件的inode，删除源文件后链接仍然有效 |
| 符号链接 | 指向文件的路径，删除源文件后链接失效 |

## 5. 实战示例

### 5.1 创建硬链接
```bash
ln file.txt link.txt
```

### 5.2 创建符号链接
```bash
ln -s file.txt link.txt
```

### 5.3 强制创建链接
```bash
ln -f file.txt link.txt
```

### 5.4 交互式创建链接
```bash
ln -i file.txt link.txt
```

### 5.5 显示创建的链接
```bash
ln -v file.txt link.txt
```

### 5.6 为目录创建符号链接
```bash
ln -s directory link
```

## 6. 注意事项
- 硬链接只能在同一个文件系统中创建，符号链接可以跨文件系统
- 硬链接不能指向目录，符号链接可以指向目录
- 删除源文件后，硬链接仍然有效，符号链接会失效
