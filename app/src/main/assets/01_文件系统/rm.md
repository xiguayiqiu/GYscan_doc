# rm - 删除文件或目录

## 1. 命令简介
rm 命令用于删除文件或目录，是 Linux 系统中最常用的文件操作命令之一。

## 2. 命令语法
```bash
rm [选项] 文件或目录
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -f | 强制删除，不提示 |
| -i | 交互式删除，删除前询问 |
| -r | 递归删除目录及其内容 |
| -v | 详细模式，显示删除的文件 |
| -d | 删除空目录 |

## 4. 实战示例

### 4.1 删除文件
```bash
rm file.txt
```

### 4.2 强制删除文件
```bash
rm -f file.txt
```

### 4.3 交互式删除文件
```bash
rm -i file.txt
```

### 4.4 递归删除目录
```bash
rm -r directory/
```

### 4.5 强制递归删除目录
```bash
rm -rf directory/
```

### 4.6 显示删除的文件
```bash
rm -v file.txt
```

## 5. 注意事项
- rm 命令删除的文件默认不会放入回收站，删除前请谨慎确认
- 删除目录时需要使用 -r 选项
- 使用 -f 选项会跳过所有提示，非常危险，建议谨慎使用
