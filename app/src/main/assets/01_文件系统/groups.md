# groups - 显示用户所属的组

## 1. 命令简介
groups 命令用于显示当前用户或指定用户所属的组，是 Linux 系统中最常用的命令之一。

## 2. 命令语法
```bash
groups [用户名]
```

## 3. 实战示例

### 3.1 显示当前用户所属的组
```bash
groups
```

### 3.2 显示指定用户所属的组
```bash
groups username
```

## 4. 退出状态
- 0：命令执行成功
- 非0：命令执行失败

## 5. 注意事项
- groups 命令不需要任何参数就可以执行，默认显示当前用户所属的组
- groups 命令的输出可以用于脚本中，以获取用户所属的组信息
- 与 `id -Gn` 命令的功能相同
