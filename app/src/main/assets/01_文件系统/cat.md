# cat - 连接并显示文件内容

## 1. 命令简介
cat 命令用于连接并显示文件内容，是 Linux 系统中最常用的文件查看命令之一。

## 2. 命令语法
```bash
cat [选项] 文件
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -n | 显示行号 |
| -b | 显示行号，但不包括空行 |
| -s | 压缩连续的空行 |
| -T | 显示制表符为 ^I |
| -A | 显示所有控制字符 |

## 4. 实战示例

### 4.1 显示文件内容
```bash
cat file.txt
```

### 4.2 显示文件内容并显示行号
```bash
cat -n file.txt
```

### 4.3 显示文件内容并显示行号（不包括空行）
```bash
cat -b file.txt
```

### 4.4 压缩连续的空行
```bash
cat -s file.txt
```

### 4.5 显示制表符为 ^I
```bash
cat -T file.txt
```

### 4.6 连接多个文件
```bash
cat file1.txt file2.txt > combined.txt
```

## 5. 注意事项
- cat 命令适合查看小文件，对于大文件，建议使用 less 或 more 命令
- 使用重定向符 > 可以将输出重定向到文件
- 使用管道符 | 可以将输出传递给其他命令
