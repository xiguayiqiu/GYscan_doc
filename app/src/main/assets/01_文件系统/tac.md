# tac - 反向显示文件内容

## 1. 命令简介
tac 命令用于反向显示文件内容，是 Linux 系统中常用的文件系统命令之一。

## 2. 命令语法
```bash
tac [选项] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -b | 在每行前添加分隔符 |
| -r | 反向分隔符 |
| -s | 指定分隔符 |

## 4. 实战示例

### 4.1 反向显示文件内容
```bash
tac file.txt
```

### 4.2 指定分隔符
```bash
tac -s "---" file.txt
```

## 5. 注意事项
- tac 命令用于反向显示文件内容，与 cat 命令相反
- tac 命令会从文件末尾开始显示，逐行向上
- tac 命令是一个常用的文本处理工具，用于反向查看文件内容
