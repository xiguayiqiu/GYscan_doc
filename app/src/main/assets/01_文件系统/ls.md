# ls - 列出目录内容

## 1. 命令简介
ls 命令用于列出目录中的文件和子目录，是 Linux 系统中最常用的命令之一。

## 2. 命令语法
```bash
ls [选项] [文件或目录]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -a | 显示所有文件，包括隐藏文件（以.开头的文件） |
| -l | 以长格式显示文件信息，包括权限、所有者、大小、修改时间等 |
| -h | 以人类可读的格式显示文件大小（如K、M、G） |
| -t | 按修改时间排序，最新的文件排在前面 |
| -r | 反向排序 |
| -R | 递归显示子目录中的文件 |
| -i | 显示文件的inode号 |
| -F | 在文件名后添加类型指示符（如/表示目录，*表示可执行文件） |

## 4. 实战示例

### 4.1 列出当前目录中的文件和子目录
```bash
ls
```

### 4.2 列出所有文件，包括隐藏文件
```bash
ls -a
```

### 4.3 以长格式显示文件信息
```bash
ls -l
```

### 4.4 以人类可读的格式显示文件大小
```bash
ls -lh
```

### 4.5 按修改时间排序显示文件
```bash
ls -lt
```

### 4.6 递归显示子目录中的文件
```bash
ls -R
```

## 5. 注意事项
- ls 命令默认不显示隐藏文件，需要使用 -a 选项
- 使用 -l 选项可以查看文件的详细信息，包括权限、所有者、大小等
- 在不同的 Linux 发行版中，ls 命令的默认行为可能有所不同
