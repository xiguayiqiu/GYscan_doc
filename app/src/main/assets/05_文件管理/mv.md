# mv - 移动或重命名文件

## 1. 命令简介
mv 命令用于移动或重命名文件，是 Linux 系统中常用的文件管理命令之一。

## 2. 命令语法
```bash
mv [选项] [源文件] [目标文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -f | 强制移动 |
| -i | 交互式移动 |
| -v | 显示移动过程 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 重命名文件
```bash
mv file.txt newfile.txt
```

### 4.2 移动文件到目录
```bash
mv file.txt directory/
```

### 4.3 强制移动
```bash
mv -f file.txt directory/
```

### 4.4 交互式移动
```bash
mv -i file.txt directory/
```

## 5. 注意事项
- mv 命令用于移动或重命名文件
- mv 命令可以在不同的文件系统之间移动文件
- mv 命令是一个常用的文件管理工具，用于移动和重命名文件
