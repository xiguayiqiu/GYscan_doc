# rm - 删除文件或目录

## 1. 命令简介
rm 命令用于删除文件或目录，是 Linux 系统中常用的文件管理命令之一。

## 2. 命令语法
```bash
rm [选项] [文件或目录]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -f | 强制删除 |
| -i | 交互式删除 |
| -r | 递归删除 |
| -v | 显示删除过程 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 删除文件
```bash
rm file.txt
```

### 4.2 强制删除
```bash
rm -f file.txt
```

### 4.3 递归删除目录
```bash
rm -r directory
```

### 4.4 交互式删除
```bash
rm -i file.txt
```

## 5. 注意事项
- rm 命令用于删除文件或目录
- rm 命令删除的文件不会进入回收站，删除前请确认
- rm 命令的 -r 选项用于删除目录及其内容
- rm 命令是一个常用的文件管理工具，用于删除文件和目录
