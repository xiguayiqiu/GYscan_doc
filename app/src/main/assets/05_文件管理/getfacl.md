# getfacl 命令

## 命令说明

`getfacl` 是 Linux 系统中用于获取文件或目录访问控制列表（ACL）的命令。

## 语法

```bash
getfacl [选项] 文件...
```

## 常用选项

| 选项 | 描述 |
|------|------|
| `-a` | 显示访问控制列表 |
| `-d` | 显示默认访问控制列表 |
| `-c` | 简洁输出 |
| `-e` | 显示所有有效权限 |
| `-R` | 递归显示目录及其内容的访问控制列表 |

## 示例

### 显示文件的访问控制列表

```bash
getfacl file.txt
```

### 递归显示目录及其内容的访问控制列表

```bash
getfacl -R directory/
```

### 简洁输出访问控制列表

```bash
getfacl -c file.txt
```

### 显示默认访问控制列表

```bash
getfacl -d directory/
```

## 注意事项

- `getfacl` 命令用于获取文件或目录的访问控制列表
- 访问控制列表提供了比传统权限更细粒度的权限控制
- 使用 `setfacl` 命令可以修改文件的访问控制列表
- 访问控制列表的输出格式包括文件所有者、所属组、其他用户的权限，以及额外的用户和组权限
