# ls - 列出目录内容

## 1. 命令简介
ls 命令用于列出目录内容，是 Linux 系统中常用的文件管理命令之一。

## 2. 命令语法
```bash
ls [选项] [目录]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -a | 显示所有文件，包括隐藏文件 |
| -l | 显示详细信息 |
| -h | 以人类可读的格式显示大小 |
| -t | 按修改时间排序 |
| -r | 反向排序 |
| -i | 显示inode号 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 列出目录内容
```bash
ls
```

### 4.2 显示详细信息
```bash
ls -l
```

### 4.3 显示所有文件
```bash
ls -a
```

### 4.4 以人类可读的格式显示
```bash
ls -lh
```

## 5. 注意事项
- ls 命令用于列出目录内容，包括文件和子目录
- ls 命令的 -l 选项显示详细信息，包括权限、所有者、大小等
- ls 命令是一个常用的文件管理工具，用于查看目录内容
