# lsattr 命令

## 命令说明

`lsattr` 是 Linux 系统中用于显示文件或目录特殊属性的命令。

## 语法

```bash
lsattr [选项] 文件...
```

## 常用选项

| 选项 | 描述 |
|------|------|
| `-R` | 递归显示目录及其内容的属性 |
| `-a` | 显示所有文件，包括隐藏文件 |
| `-d` | 显示目录本身的属性，而不是目录内容 |
| `-v` | 显示文件版本号 |

## 属性含义

| 属性 | 描述 |
|------|------|
| `a` | 只能追加内容，不能删除或修改 |
| `c` | 自动压缩文件 |
| `d` | 不在dump备份中包含 |
| `i` | 不可修改，不能删除、重命名或修改 |
| `s` | 删除时安全擦除 |
| `u` | 删除时保留数据，可恢复 |
| `t` | 尾部合并 |
| `e` | 扩展格式 |

## 示例

### 显示文件属性

```bash
lsattr file.txt
```

### 递归显示目录及其内容的属性

```bash
lsattr -R directory/
```

### 显示所有文件的属性，包括隐藏文件

```bash
lsattr -a
```

### 显示目录本身的属性

```bash
lsattr -d directory/
```

## 注意事项

- `lsattr` 命令用于显示文件或目录的特殊属性
- 这些特殊属性是通过 `chattr` 命令设置的
- 特殊属性包括不可修改、只能追加等特性
- 只有 root 用户可以修改这些特殊属性
