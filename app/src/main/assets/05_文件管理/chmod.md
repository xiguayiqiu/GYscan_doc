# chmod - 修改文件权限

## 1. 命令简介
chmod 命令用于修改文件权限，是 Linux 系统中常用的文件管理命令之一。

## 2. 命令语法
```bash
chmod [选项] [权限] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -f | 强制修改 |
| -v | 显示修改过程 |
| -R | 递归修改 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 权限表示
| 权限 | 说明 |
|------|------|
| r | 读权限 |
| w | 写权限 |
| x | 执行权限 |
| 4 | 读权限 |
| 2 | 写权限 |
| 1 | 执行权限 |

## 5. 实战示例

### 5.1 修改文件权限
```bash
chmod 755 file.txt
```

### 5.2 递归修改目录权限
```bash
chmod -R 755 directory/
```

### 5.3 使用符号表示修改权限
```bash
chmod u+x file.txt
```

### 5.4 显示修改过程
```bash
chmod -v 755 file.txt
```

## 6. 注意事项
- chmod 命令用于修改文件权限
- chmod 命令可以使用数字或符号表示权限
- chmod 命令是一个常用的文件管理工具，用于管理文件权限
