# setfacl 命令

## 命令说明

`setfacl` 是 Linux 系统中用于设置文件或目录访问控制列表（ACL）的命令。

## 语法

```bash
setfacl [选项] 文件...
```

## 常用选项

| 选项 | 描述 |
|------|------|
| `-m` | 修改访问控制列表 |
| `-x` | 删除访问控制列表中的条目 |
| `-b` | 删除所有访问控制列表 |
| `-k` | 删除默认访问控制列表 |
| `-R` | 递归修改目录及其内容的访问控制列表 |
| `-d` | 设置默认访问控制列表 |

## 访问控制列表格式

```
[类型]:[标识符]:[权限]
```

| 类型 | 描述 |
|------|------|
| `u` | 用户 |
| `g` | 组 |
| `o` | 其他 |
| `m` | 掩码 |

## 示例

### 为用户添加读取权限

```bash
setfacl -m u:user:r file.txt
```

### 为组添加读写权限

```bash
setfacl -m g:group:rw file.txt
```

### 递归修改目录及其内容的访问控制列表

```bash
setfacl -R -m u:user:rwx directory/
```

### 设置默认访问控制列表

```bash
setfacl -d -m u:user:rwx directory/
```

### 删除访问控制列表中的条目

```bash
setfacl -x u:user file.txt
```

### 删除所有访问控制列表

```bash
setfacl -b file.txt
```

## 注意事项

- `setfacl` 命令用于设置文件或目录的访问控制列表
- 访问控制列表提供了比传统权限更细粒度的权限控制
- 只有文件所有者或 root 用户可以修改文件的访问控制列表
- 使用 `getfacl` 命令可以查看文件的访问控制列表
