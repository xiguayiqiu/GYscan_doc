# chattr 命令

## 命令说明

`chattr` 是 Linux 系统中用于修改文件或目录属性的命令，这些属性包括不可修改、只能追加等特殊属性。

## 语法

```bash
chattr [选项] [模式] 文件...
```

## 常用选项

| 选项 | 描述 |
|------|------|
| `-R` | 递归修改目录及其内容的属性 |
| `-v` | 显示详细的执行过程 |
| `-f` | 忽略错误信息 |

## 属性模式

| 属性 | 描述 |
|------|------|
| `a` | 只能追加内容，不能删除或修改 |
| `c` | 自动压缩文件 |
| `d` | 不在dump备份中包含 |
| `i` | 不可修改，不能删除、重命名或修改 |
| `s` | 删除时安全擦除 |
| `u` | 删除时保留数据，可恢复 |
| `t` | 尾部合并 |
| `e` | 扩展格式 |

## 示例

### 添加不可修改属性

```bash
chattr +i file.txt
```

### 添加只能追加属性

```bash
chattr +a file.txt
```

### 移除不可修改属性

```bash
chattr -i file.txt
```

### 递归修改目录及其内容的属性

```bash
chattr -R +i directory/
```

## 注意事项

- `chattr` 命令可以修改文件或目录的特殊属性
- 只有 root 用户可以修改文件的特殊属性
- `i` 属性（不可修改）是最常用的属性，可以防止文件被意外删除或修改
- `a` 属性（只能追加）常用于日志文件，确保只能添加内容，不能修改或删除
