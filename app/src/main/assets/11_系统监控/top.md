# top - 实时显示系统进程

## 1. 命令简介
top 命令用于实时显示系统进程的资源使用情况，是 Linux 系统中最常用的系统监控工具之一。

## 2. 命令语法
```bash
top [选项]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -d | 指定刷新间隔（秒） |
| -p | 显示指定进程的信息 |
| -u | 显示指定用户的进程 |
| -b | 批处理模式，适合输出到文件 |
| -n | 指定显示的次数 |
| -c | 显示完整的命令行 |

## 4. 交互式命令
| 命令 | 说明 |
|------|------|
| h | 显示帮助信息 |
| q | 退出top |
| P | 按CPU使用率排序 |
| M | 按内存使用率排序 |
| T | 按运行时间排序 |
| k | 终止指定进程 |
| r | 重新设置进程优先级 |
| u | 显示指定用户的进程 |
| z | 彩色显示 |

## 5. 实战示例

### 5.1 启动top
```bash
top
```

### 5.2 指定刷新间隔
```bash
top -d 2
```

### 5.3 显示指定进程
```bash
top -p 1234
```

### 5.4 显示指定用户的进程
```bash
top -u username
```

### 5.5 批处理模式输出到文件
```bash
top -b -n 1 > top_output.txt
```

### 5.6 显示完整的命令行
```bash
top -c
```

## 6. 注意事项
- top 命令默认按CPU使用率排序
- 按 q 键可以退出top
- 使用 -b 选项可以将top的输出重定向到文件
