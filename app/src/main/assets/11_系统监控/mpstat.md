# mpstat 命令

## 命令说明

`mpstat` 是 Linux 系统中用于显示多处理器系统 CPU 使用情况的命令。

## 语法

```bash
mpstat [选项] [延迟] [次数]
```

## 常用选项

| 选项 | 描述 |
|------|------|
| `-P` | 指定 CPU 核心，`ALL` 表示所有核心 |
| `-u` | 显示 CPU 使用率（默认） |
| `-V` | 显示版本信息 |

## 输出说明

| 列名 | 描述 |
|------|------|
| CPU | CPU 核心编号 |
| %usr | 用户空间使用的 CPU 时间百分比 |
| %nice | 运行 nice 进程使用的 CPU 时间百分比 |
| %sys | 系统空间使用的 CPU 时间百分比 |
| %iowait | 等待 I/O 的 CPU 时间百分比 |
| %irq | 处理中断的 CPU 时间百分比 |
| %soft | 处理软中断的 CPU 时间百分比 |
| %steal | 虚拟机被窃取的 CPU 时间百分比 |
| %guest | 运行虚拟机的 CPU 时间百分比 |
| %idle | 空闲 CPU 时间百分比 |

## 示例

### 显示所有 CPU 的使用情况

```bash
mpstat -P ALL
```

### 每 2 秒显示一次，共显示 5 次

```bash
mpstat 2 5
```

### 显示指定 CPU 的使用情况

```bash
mpstat -P 0
```

## 注意事项

- `mpstat` 命令不需要特殊权限
- 输出信息包括每个 CPU 核心的使用情况，便于分析多处理器系统的性能
- 可以使用 `mpstat` 命令监控系统的 CPU 使用情况，及时发现 CPU 瓶颈
- 延迟参数指定两次采样之间的时间间隔，次数参数指定采样的次数
