# iostat - 显示I/O统计

## 1. 命令简介
iostat 命令用于显示系统的I/O统计信息，是 Linux 系统中最常用的系统监控命令之一。

## 2. 命令语法
```bash
iostat [选项] [间隔时间] [次数]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -c | 显示CPU统计信息 |
| -d | 显示磁盘统计信息 |
| -k | 以KB为单位显示 |
| -m | 以MB为单位显示 |
| -t | 显示时间戳 |
| -x | 显示扩展统计信息 |

## 4. 实战示例

### 4.1 显示CPU和磁盘统计信息
```bash
iostat
```

### 4.2 显示磁盘统计信息
```bash
iostat -d
```

### 4.3 以KB为单位显示
```bash
iostat -k
```

### 4.4 每2秒显示一次，共显示3次
```bash
iostat 2 3
```

### 4.5 显示扩展统计信息
```bash
iostat -x
```

### 4.6 显示时间戳
```bash
iostat -t
```

## 5. 注意事项
- iostat 命令需要 sysstat 包，可能需要额外安装
- 使用间隔时间和次数可以持续监控系统的I/O状态
- 输出中的 tps 表示每秒传输次数，kB_read/s 表示每秒读取的KB数，kB_wrtn/s 表示每秒写入的KB数
