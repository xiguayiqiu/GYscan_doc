# vmstat 命令

## 命令说明

`vmstat` 是 Linux 系统中用于显示虚拟内存、进程、CPU 活动等系统状态的命令。

## 语法

```bash
vmstat [选项] [延迟] [次数]
```

## 常用选项

| 选项 | 描述 |
|------|------|
| `-a` | 显示活跃和非活跃内存 |
| `-f` | 显示启动后创建的进程数 |
| `-m` | 显示内存分配信息 |
| `-n` | 只显示一次头部信息 |
| `-s` | 显示内存统计信息 |
| `-d` | 显示磁盘统计信息 |
| `-p` | 显示指定分区的统计信息 |

## 输出说明

| 列名 | 描述 |
|------|------|
| procs | 进程信息 |
| r | 运行队列中的进程数 |
| b | 阻塞的进程数 |
| memory | 内存信息 |
| swpd | 使用的交换空间大小 |
| free | 空闲内存大小 |
| buff | 缓冲区使用的内存大小 |
| cache | 缓存使用的内存大小 |
| swap | 交换空间信息 |
| si | 从交换空间读入内存的速率 |
| so | 从内存写入交换空间的速率 |
| io | I/O 信息 |
| bi | 从块设备读入的速率 |
| bo | 写入块设备的速率 |
| system | 系统信息 |
| in | 中断率 |
| cs | 上下文切换率 |
| cpu | CPU 信息 |
| us | 用户空间使用的 CPU 时间百分比 |
| sy | 系统空间使用的 CPU 时间百分比 |
| id | 空闲 CPU 时间百分比 |
| wa | 等待 I/O 的 CPU 时间百分比 |

## 示例

### 显示系统状态

```bash
vmstat
```

### 每 2 秒显示一次，共显示 5 次

```bash
vmstat 2 5
```

### 显示内存统计信息

```bash
vmstat -s
```

### 显示磁盘统计信息

```bash
vmstat -d
```

## 注意事项

- `vmstat` 命令不需要特殊权限
- 输出信息包括进程、内存、交换空间、I/O、系统和 CPU 等方面的统计信息
- 可以使用 `vmstat` 命令监控系统的运行状态，及时发现系统瓶颈
- 延迟参数指定两次采样之间的时间间隔，次数参数指定采样的次数
