# expr - 表达式求值

## 1. 命令简介
expr 命令用于表达式求值，是 Linux 系统中常用的 shell 环境命令之一。

## 2. 命令语法
```bash
expr [表达式]
```

## 3. 常用操作符
| 操作符 | 说明 |
|-------|------|
| + | 加法 |
| - | 减法 |
| \* | 乘法 |
| / | 除法 |
| % | 取余 |
| = | 等于 |
| != | 不等于 |
| < | 小于 |
| <= | 小于等于 |
| > | 大于 |
| >= | 大于等于 |
| & | 与 |
| \| | 或 |
| : | 模式匹配 |

## 4. 实战示例

### 4.1 加法
```bash
expr 1 + 2
```

### 4.2 乘法
```bash
expr 2 \* 3
```

### 4.3 模式匹配
```bash
expr "hello world" : "hello"
```

### 4.4 比较
```bash
expr 10 \> 5
```

## 5. 注意事项
- expr 命令用于表达式求值，包括算术运算、比较和模式匹配
- expr 命令的操作符需要转义，如 \*、\> 等
- expr 命令是一个常用的 shell 工具，用于表达式求值
