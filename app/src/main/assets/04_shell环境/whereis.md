# whereis - 查找命令路径

## 1. 命令简介
whereis 命令用于查找命令路径，是 Linux 系统中常用的 shell 环境命令之一。

## 2. 命令语法
```bash
whereis [选项] [命令]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -b | 只显示二进制文件 |
| -m | 只显示手册页 |
| -s | 只显示源代码 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 查找命令路径
```bash
whereis ls
```

### 4.2 只显示二进制文件
```bash
whereis -b ls
```

### 4.3 只显示手册页
```bash
whereis -m ls
```

## 5. 注意事项
- whereis 命令用于查找命令的二进制文件、手册页和源代码
- whereis 命令搜索系统默认的路径
- whereis 命令是一个常用的 shell 工具，用于查找命令路径
