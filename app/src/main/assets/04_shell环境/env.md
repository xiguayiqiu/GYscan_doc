# env - 显示环境变量

## 1. 命令简介
env 命令用于显示环境变量，是 Linux 系统中常用的 shell 环境命令之一。

## 2. 命令语法
```bash
env [选项] [命令]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -u | 取消环境变量 |
| -i | 清除环境变量 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 显示环境变量
```bash
env
```

### 4.2 执行命令时清除环境变量
```bash
env -i command
```

### 4.3 取消环境变量
```bash
env -u VAR command
```

## 5. 注意事项
- env 命令用于显示环境变量，或者在特定环境中执行命令
- env 命令是一个常用的 shell 工具，用于管理环境变量
