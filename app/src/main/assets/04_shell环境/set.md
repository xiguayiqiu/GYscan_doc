# set - 设置Shell变量

## 1. 命令简介
set 命令用于设置 Shell 变量，是 Linux 系统中常用的 shell 环境命令之一。

## 2. 命令语法
```bash
set [选项] [变量=值]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -a | 自动导出变量 |
| -b | 通知作业完成 |
| -e | 命令出错时退出 |
| -f | 禁用文件名扩展 |
| -h | 启用别名扩展 |
| -n | 读取命令但不执行 |
| -u | 未设置变量时出错 |
| -v | 显示输入的命令 |
| -x | 显示执行的命令 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 设置变量
```bash
set VAR=value
```

### 4.2 显示所有变量
```bash
set
```

### 4.3 启用命令出错时退出
```bash
set -e
```

### 4.4 显示执行的命令
```bash
set -x
```

## 5. 注意事项
- set 命令用于设置 Shell 变量和选项
- set 命令在 shell 脚本中常用于设置执行环境
- set 命令是一个常用的 shell 工具，用于管理 shell 环境
