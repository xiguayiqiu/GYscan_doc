# stty - 设置终端属性

## 1. 命令简介
stty 命令用于设置终端属性，是 Linux 系统中常用的 shell 环境命令之一。

## 2. 命令语法
```bash
stty [选项] [参数]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -a | 显示所有终端属性 |
| -g | 显示当前设置 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 常用参数
| 参数 | 说明 |
|------|------|
| sane | 恢复默认设置 |
| erase | 设置退格键 |
| kill | 设置删除键 |
| intr | 设置中断键 |
| quit | 设置退出键 |

## 5. 实战示例

### 5.1 显示所有终端属性
```bash
stty -a
```

### 5.2 恢复默认设置
```bash
stty sane
```

### 5.3 设置退格键
```bash
stty erase ^H
```

### 5.4 设置中断键
```bash
stty intr ^C
```

## 6. 注意事项
- stty 命令用于设置终端属性，如按键映射、行编辑等
- stty 命令是一个常用的 shell 工具，用于终端设置
