# lscpu - 显示CPU信息

## 1. 命令简介
lscpu 命令用于显示CPU信息，是 Linux 系统中常用的系统信息命令之一。

## 2. 命令语法
```bash
lscpu [选项]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -a | 显示所有CPU信息 |
| -b | 以字节为单位显示大小 |
| -c | 显示CPU缓存信息 |
| -e | 显示扩展信息 |
| -p | 以解析友好的格式显示 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 显示CPU信息
```bash
lscpu
```

### 4.2 显示扩展信息
```bash
lscpu -e
```

### 4.3 以解析友好的格式显示
```bash
lscpu -p
```

## 5. 注意事项
- lscpu 命令用于显示CPU信息，包括CPU架构、核心数、缓存大小等
- lscpu 命令是一个常用的系统信息工具，用于获取CPU的详细信息
