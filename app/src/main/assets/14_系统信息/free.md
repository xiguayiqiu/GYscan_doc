# free - 显示内存使用情况

## 1. 命令简介
free 命令用于显示内存使用情况，是 Linux 系统中常用的系统信息命令之一。

## 2. 命令语法
```bash
free [选项]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -b | 以字节为单位显示 |
| -k | 以KB为单位显示 |
| -m | 以MB为单位显示 |
| -g | 以GB为单位显示 |
| -h | 以人类可读的格式显示 |
| -s | 持续显示 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 显示内存使用情况
```bash
free
```

### 4.2 以人类可读的格式显示
```bash
free -h
```

### 4.3 持续显示
```bash
free -s 1
```

### 4.4 以MB为单位显示
```bash
free -m
```

## 5. 注意事项
- free 命令用于显示内存使用情况，包括物理内存和交换内存
- free 命令是一个常用的系统信息工具，用于监控内存使用情况
