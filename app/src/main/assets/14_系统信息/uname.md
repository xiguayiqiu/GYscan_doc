# uname - 显示系统信息

## 1. 命令简介
uname 命令用于显示系统信息，是 Linux 系统中常用的系统信息命令之一。

## 2. 命令语法
```bash
uname [选项]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -a | 显示所有信息 |
| -s | 显示内核名称 |
| -n | 显示主机名 |
| -r | 显示内核版本 |
| -v | 显示内核版本详细信息 |
| -m | 显示机器硬件名称 |
| -p | 显示处理器类型 |
| -i | 显示硬件平台 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 显示所有信息
```bash
uname -a
```

### 4.2 显示内核名称
```bash
uname -s
```

### 4.3 显示内核版本
```bash
uname -r
```

### 4.4 显示机器硬件名称
```bash
uname -m
```

## 5. 注意事项
- uname 命令用于显示系统信息，包括内核名称、版本、主机名等
- uname 命令是一个常用的系统信息工具，用于获取系统的基本信息
