# lsmem - 显示内存信息

## 1. 命令简介
lsmem 命令用于显示内存信息，是 Linux 系统中常用的系统信息命令之一。

## 2. 命令语法
```bash
lsmem [选项]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -a | 显示所有内存信息 |
| -b | 以字节为单位显示大小 |
| -n | 不显示表头 |
| -o | 自定义输出列 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 显示内存信息
```bash
lsmem
```

### 4.2 以字节为单位显示大小
```bash
lsmem -b
```

### 4.3 不显示表头
```bash
lsmem -n
```

## 5. 注意事项
- lsmem 命令用于显示内存信息，包括内存块的大小和状态
- lsmem 命令是一个常用的系统信息工具，用于获取内存的详细信息
