# vmstat - 虚拟内存统计

## 1. 命令简介
vmstat 命令用于虚拟内存统计，是 Linux 系统中常用的系统信息命令之一。

## 2. 命令语法
```bash
vmstat [选项] [间隔] [次数]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -a | 显示活动和非活动内存 |
| -f | 显示启动以来的fork次数 |
| -m | 显示内存使用情况 |
| -s | 显示内存统计信息 |
| -d | 显示磁盘统计信息 |
| -p | 显示指定分区的统计信息 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 显示虚拟内存统计
```bash
vmstat
```

### 4.2 持续显示
```bash
vmstat 1
```

### 4.3 显示活动和非活动内存
```bash
vmstat -a
```

### 4.4 显示磁盘统计信息
```bash
vmstat -d
```

## 5. 注意事项
- vmstat 命令用于虚拟内存统计，显示系统的内存、CPU、磁盘等使用情况
- vmstat 命令是一个常用的系统信息工具，用于系统性能监控
