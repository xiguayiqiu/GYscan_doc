# masscan 命令

## 命令说明
masscan是一款功能强大的高速网络扫描工具，用于快速扫描大范围网络。它采用异步扫描技术，可以以极高的速率扫描大量IP地址和端口，是大规模网络安全评估和端口发现的重要工具。

## 语法
```
masscan [选项] 目标
```

## 常用选项
- `-p <端口>`：指定端口范围，如 80,443 或 1-65535
- `--rate <速率>`：设置扫描速率（数据包/秒）
- `-oG <文件>`：输出 grep 格式到文件
- `-oX <文件>`：输出 XML 格式到文件
- `-oJ <文件>`：输出 JSON 格式到文件
- `-oB <文件>`：输出二进制格式到文件
- `-e <接口>`：指定网络接口
- `-iL <文件>`：从文件读取目标列表
- `--exclude <IP>`：排除指定IP
- `--exclude-file <文件>`：从文件读取排除的IP列表
- `--banners`：获取服务横幅信息
- `--ping`：使用ICMP ping检测主机是否存活
- `--adapter-ip <IP>`：设置适配器IP地址
- `--adapter-port <端口>`：设置源端口
- `--adapter-mac <MAC>`：设置适配器MAC地址
- `--router-mac <MAC>`：设置路由器MAC地址
- `--help`：显示帮助信息
- `--version`：显示版本信息

## 技术原理

masscan采用以下技术实现高速扫描：

1. 异步发送和接收数据包，不等待响应
2. 使用原始套接字直接构建和发送数据包
3. 采用无状态扫描模式，不维护连接状态
4. 支持多线程并行扫描
5. 高效的内存管理和数据包处理

masscan的扫描速度可以达到每秒数百万个数据包，比传统的扫描工具如nmap快几个数量级。

## 示例

### 基本扫描
```bash
# 扫描整个C类网段的所有端口
masscan 192.168.1.0/24 -p 1-65535

# 扫描指定端口
masscan 192.168.1.0/24 -p 80,443,22,21

# 扫描多个网段
masscan 192.168.1.0/24 10.0.0.0/16 -p 80

# 从文件读取目标列表
masscan -iL targets.txt -p 80
```

### 高级扫描
```bash
# 设置扫描速率为每秒1000个数据包
masscan 192.168.1.0/24 -p 80 --rate 1000

# 获取服务横幅信息
masscan 192.168.1.0/24 -p 80 --banners

# 使用ICMP ping检测主机
masscan 192.168.1.0/24 -p 80 --ping

# 排除指定IP
masscan 192.168.1.0/24 -p 80 --exclude 192.168.1.100

# 从文件读取排除的IP
masscan 192.168.1.0/24 -p 80 --exclude-file exclude.txt
```

### 输出格式
```bash
# 输出grep格式
masscan 192.168.1.0/24 -p 80 -oG scan.txt

# 输出XML格式
masscan 192.168.1.0/24 -p 80 -oX scan.xml

# 输出JSON格式
masscan 192.168.1.0/24 -p 80 -oJ scan.json

# 输出二进制格式
masscan 192.168.1.0/24 -p 80 -oB scan.bin
```

### 网络接口设置
```bash
# 指定网络接口
masscan 192.168.1.0/24 -p 80 -e eth0

# 设置适配器IP
masscan 192.168.1.0/24 -p 80 --adapter-ip 192.168.1.100

# 设置源端口
masscan 192.168.1.0/24 -p 80 --adapter-port 12345

# 设置适配器MAC地址
masscan 192.168.1.0/24 -p 80 --adapter-mac 00:11:22:33:44:55

# 设置路由器MAC地址
masscan 192.168.1.0/24 -p 80 --router-mac 00:AA:BB:CC:DD:EE
```

## 高级技巧

### 与其他工具结合
```bash
# 扫描结果导入nmap进行详细扫描
masscan 192.168.1.0/24 -p 80 -oG scan.txt && grep "open" scan.txt | cut -d " " -f 2 | nmap -iL -

# 扫描并过滤结果
masscan 192.168.1.0/24 -p 80 -oJ scan.json | jq '.[] | select(.status == "open")'

# 批量扫描多个端口范围
for port in 80 443 22 21 3389; do
  masscan 192.168.1.0/24 -p $port --rate 1000 -oG scan_$port.txt
done
```

### 性能优化
- 根据网络带宽调整扫描速率
- 对于大型网络，使用多个扫描实例
- 合理设置端口范围，避免扫描不必要的端口
- 使用二进制输出格式提高性能

### 安全注意事项
- 获得授权后再进行扫描
- 注意扫描速率，避免对网络造成影响
- 遵守相关法律法规
- 避免扫描互联网上的随机IP地址

### 常见问题解决

- **问题**：扫描速度慢
  **解决方法**：增加扫描速率，使用更强大的硬件

- **问题**：扫描结果不准确
  **解决方法**：降低扫描速率，增加重试次数

- **问题**：被防火墙阻止
  **解决方法**：调整扫描速率，使用不同的源端口

- **问题**：内存使用过高
  **解决方法**：减少扫描范围，使用更小的批处理大小
