# radare2 命令

## 命令说明
radare2（简称 r2）是一个功能强大的开源逆向工程框架，用于分析二进制文件、执行调试、反汇编、反编译、修补程序等操作。它支持多种文件格式和架构，是安全研究人员、逆向工程师和恶意软件分析师的重要工具。

## 语法
```
r2 [选项] [文件路径]
```

## 常用选项
- `-A`：自动分析文件，执行完整的分析流程
- `-d`：以调试模式启动，允许调试目标程序
- `-w`：以可写模式打开文件，允许修改二进制文件
- `-b <bits>`：指定目标架构的位宽（如 16、32、64）
- `-e <key=value>`：设置配置选项
- `-f`：在打开文件后立即运行指定的命令
- `-i <file>`：从文件中读取并执行命令
- `-p`：以只读模式打开文件（默认）
- `-q`：安静模式，减少输出
- `-v`：显示版本信息

## 核心r2命令

### 分析命令
- `aa`：分析所有函数
- `aab`：分析所有函数和基本块
- `aaa`：执行完整的自动分析
- `af`：分析函数
- `afn <name> <addr>`：重命名函数

### 打印命令
- `p`：打印当前位置的字节
- `pd <n>`：打印 n 行反汇编
- `pdf`：打印函数的反汇编
- `px <n>`：以十六进制格式打印 n 字节
- `ps <n>`：打印 n 字节的字符串

### 搜索命令
- `/ <pattern>`：搜索指定模式
- `/x <hex>`：搜索十六进制模式
- `/s <string>`：搜索字符串
- `/a <asm>`：搜索汇编指令

### 信息命令
- `i`：显示文件信息
- `iS`：显示节信息
- `iF`：显示函数信息
- `ii`：显示导入符号
- `ie`：显示导出符号

### 可视化模式
- `V`：进入可视化模式
- `Vp`：进入全屏可视化模式
- `Vv`：进入视图模式

### 调试命令
- `db <addr>`：设置断点
- `dc`：继续执行
- `ds`：单步执行
- `dr`：显示寄存器
- `dw <reg> <value>`：写入寄存器值

### 导航命令
- `s <addr>`：跳转到指定地址
- `S`：列出所有段
- `f`：列出所有函数
- `u <size>`：向上移动指定大小
- `d <size>`：向下移动指定大小

## 高级功能

### 脚本和自动化
- `! <command>`：执行外部命令
- `. <script>`：执行 r2 脚本
- `?`：显示帮助信息

### 修补和修改
- `w <data>`：在当前位置写入数据
- `wa <asm>`：在当前位置写入汇编指令
- `wf <file>`：从文件写入数据

### 插件和扩展
- `L`：列出所有加载的插件
- `| <plugin>`：使用指定插件

## 示例

### 基本分析
```bash
# 打开并分析二进制文件
r2 /bin/ls

# 自动分析
[0x00404890]> aa

# 查看函数列表
[0x00404890]> f

# 查看主函数反汇编
[0x00404890]> s main
[0x00404890]> pdf
```

### 调试示例
```bash
# 以调试模式启动程序
r2 -d /bin/ls

# 设置断点
[0x7ffff7dd1090]> db main

# 开始执行
[0x7ffff7dd1090]> dc

# 查看寄存器
[0x00404890]> dr

# 单步执行
[0x00404890]> ds
```

### 搜索示例
```bash
# 搜索字符串
[0x00404890]> /s "Hello World"

# 搜索十六进制模式
[0x00404890]> /x 48 65 6c 6c 6f

# 搜索汇编指令
[0x00404890]> /a mov eax, 0x42
```

### 可视化示例
```bash
# 进入可视化模式
[0x00404890]> V

# 在可视化模式中，使用以下键：
# - p：切换视图模式
# - q：退出可视化模式
# - 方向键：导航
# - Enter：进入函数
# - Backspace：返回上一级
```

## 高级技巧

### 使用 r2pipe 进行自动化
r2pipe 是 radare2 的管道接口，允许通过脚本语言（如 Python、JavaScript 等）与 r2 进行交互，实现自动化分析和处理。

### 集成其他工具
radare2 可以与其他工具集成，如 Ghidra、IDA Pro 等，提供更强大的分析能力。

### 自定义配置
通过 `.radare2rc` 文件可以自定义 r2 的配置，包括快捷键、命令别名、默认选项等。
