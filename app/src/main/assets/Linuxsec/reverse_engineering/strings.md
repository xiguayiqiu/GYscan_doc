# strings 命令

## 命令说明
从二进制文件中提取可打印字符串。

## 语法
```
strings [选项] 文件
```

## 常用选项
- `-n`：指定最小字符串长度
- `-a`：扫描整个文件
- `-f`：显示文件名
- `-t`：显示字符串在文件中的偏移量

## 示例
```bash
# 提取字符串
strings /bin/ls

# 提取长度至少为8的字符串
strings -n 8 /bin/ls

# 显示字符串偏移量
strings -t x /bin/ls
```

## 适用场景
- 分析二进制文件中的硬编码字符串
- 查找密码、API密钥等敏感信息
- 识别文件类型
- 调试程序输出
