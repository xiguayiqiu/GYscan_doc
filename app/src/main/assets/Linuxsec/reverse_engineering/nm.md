# nm 命令

## 命令说明
nm是一款功能强大的符号查看工具，用于显示目标文件中的符号表。它可以显示目标文件中的函数、变量、全局符号等信息，是逆向工程和二进制分析的重要工具。

## 语法
```
nm [选项] 目标文件
```

## 常用选项
- `-a`：显示所有符号，包括调试符号
- `-C`：demangle C++符号
- `-D`：显示动态符号
- `-g`：只显示全局符号
- `-r`：按符号值反向排序
- `-s`：按符号名称排序
- `-u`：只显示未定义符号
- `-l`：显示符号的行号信息
- `-n`：按符号值排序
- `-p`：不排序，按符号在文件中的顺序显示
- `-t <格式>`：指定符号值的显示格式（d: 十进制, o: 八进制, x: 十六进制）
- `--help`：显示帮助信息
- `--version`：显示版本信息

## 技术原理

nm通过以下步骤工作：

1. 读取目标文件的符号表
2. 解析符号表中的条目
3. 根据指定的选项过滤和排序符号
4. 显示符号信息

nm可以处理多种目标文件格式，包括ELF、COFF、PE等，并且可以显示不同类型的符号。

## 示例

### 基本操作
```bash
# 显示目标文件中的所有符号
nm input.o

# 显示动态符号
nm -D input.so

# 只显示全局符号
nm -g input.o

# 只显示未定义符号
nm -u input.o
```

### 符号排序
```bash
# 按符号值排序
nm -n input.o

# 按符号值反向排序
nm -r input.o

# 按符号名称排序
nm -s input.o

# 不排序，按符号在文件中的顺序显示
nm -p input.o
```

### 符号信息
```bash
# 显示符号的行号信息
nm -l input.o

# demangle C++符号
nm -C input.o

# 显示所有符号，包括调试符号
nm -a input.o

# 指定符号值的显示格式（十六进制）
nm -t x input.o
```

### 实用场景
```bash
# 查找特定符号
nm input.o | grep "main"

# 查找全局函数符号
nm -g input.o | grep "T"

# 查找全局变量符号
nm -g input.o | grep "D"

# 分析动态库中的符号
nm -D libexample.so
```

## 符号类型

nm显示的符号类型包括：

- `A`：绝对符号
- `B`：未初始化的数据段符号
- `C`：未初始化的数据段符号（常见于common块）
- `D`：初始化的数据段符号
- `G`：初始化的数据段符号（小对象）
- `i`：间接引用的符号
- `N`：调试符号
- `p`：栈 unwind 信息
- `R`：只读数据段符号
- `S`：未初始化的数据段符号（小对象）
- `T`：文本段符号（代码）
- `U`：未定义符号
- `V`：弱对象符号
- `W`：弱符号

## 高级技巧

### 与其他工具结合
```bash
# 结合objdump分析符号
nm -n input.o | objdump -d input.o

# 查找特定类型的符号
nm input.o | grep "T" | sort

# 分析符号的大小
nm -S input.o

# 比较两个目标文件的符号差异
diff <(nm -n file1.o) <(nm -n file2.o)
```

### 批量处理
```bash
# 批量分析多个目标文件
for file in *.o; do
  echo "=== $file ==="
  nm -g $file
  echo
done

# 批量分析动态库
for lib in *.so; do
  echo "=== $lib ==="
  nm -D $lib
  echo
done
```

### 性能优化
- 对于大型文件，使用 `-g` 选项只显示全局符号
- 对于频繁的操作，创建脚本自动化处理
- 使用 `-n` 选项按符号值排序，便于分析
- 对于C++程序，使用 `-C` 选项demangle符号

### 常见问题解决

- **问题**：符号名称显示为乱码
  **解决方法**：对于C++程序，使用 `-C` 选项demangle符号

- **问题**：无法找到符号
  **解决方法**：使用 `-a` 选项显示所有符号，包括调试符号

- **问题**：符号值显示格式不合适
  **解决方法**：使用 `-t` 选项指定符号值的显示格式

- **问题**：输出信息过多
  **解决方法**：使用 grep 过滤需要的符号，或使用 `-g` 选项只显示全局符号
