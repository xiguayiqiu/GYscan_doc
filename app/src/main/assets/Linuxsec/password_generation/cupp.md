# cupp 命令

## 命令说明
CUPP (Common User Passwords Profiler)是一款功能强大的密码字典生成工具，用于根据用户信息生成个性化的密码字典。它可以根据用户的个人信息（如姓名、生日、电话号码等）生成可能的密码组合，是密码测试和安全评估的重要工具。

## 语法
```
python cupp.py [选项]
```

## 常用选项
- `-i`：交互式模式
- `-w <文件>`：使用单词列表
- `-l`：生成小写密码
- `-u`：生成大写密码
- `-n`：生成包含数字的密码
- `-s`：生成包含特殊字符的密码
- `-v`：详细模式
- `-h`：显示帮助信息

## 核心功能

CUPP提供了多种核心功能：
- **交互式密码字典生成**：通过交互式问答生成个性化密码字典
- **基于用户信息**：根据用户的个人信息生成密码
- **自定义选项**：可自定义密码长度、字符类型等
- **单词列表模式**：基于单词列表生成密码
- **多语言支持**：支持多种语言的密码生成

## 技术原理

CUPP通过以下步骤工作：

1. 收集用户的个人信息（如姓名、生日、电话号码等）
2. 分析用户信息，生成可能的密码组合
3. 根据用户选择的选项（如长度、字符类型等）过滤密码
4. 将生成的密码保存到文件中

CUPP使用多种算法来生成密码，包括组合用户信息、添加常见后缀、替换字符等。

## 示例

### 基本操作
```bash
# 交互式模式

# 基于单词列表
python cupp.py -w wordlist.txt

# 生成小写密码
python cupp.py -i -l

# 生成包含数字的密码
python cupp.py -i -n
```

### 高级选项
```bash

python cupp.py -i -u
```

## 常见问题解决

- **问题**：CUPP无法启动
  **解决方法**：检查Python环境是否正确，确保脚本文件完整，验证命令参数

- **问题**：生成的密码字典过大
  **解决方法**：限制密码长度，减少组合选项，使用更具体的用户信息

- **问题**：生成的密码字典质量不高
  **解决方法**：提供更详细的用户信息，调整生成选项，使用单词列表模式

- **问题**：生成速度慢
  **解决方法**：优化系统资源，减少生成选项，使用更简单的参数
