# ls - 列出目录内容

## 1. 命令简介
ls 命令用于列出目录内容，是 Linux 系统中最常用的命令之一。

## 2. 命令语法
```bash
ls [选项] [文件或目录]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -l | 以长格式显示 |
| -a | 显示所有文件，包括隐藏文件 |
| -h | 以人类可读的格式显示大小 |
| -r | 反向排序 |
| -t | 按修改时间排序 |
| -S | 按大小排序 |
| -d | 显示目录本身，而不是目录内容 |
| -i | 显示inode号 |

## 4. 实战示例

### 4.1 列出目录内容
```bash
ls
```

### 4.2 以长格式显示
```bash
ls -l
```

### 4.3 显示所有文件，包括隐藏文件
```bash
ls -a
```

### 4.4 以人类可读的格式显示大小
```bash
ls -lh
```

### 4.5 按修改时间排序
```bash
ls -lt
```

### 4.6 按大小排序
```bash
ls -lS
```

## 5. 注意事项
- ls 命令默认只显示非隐藏文件
- 使用 -l 选项可以显示文件的详细信息，包括权限、所有者、大小和修改时间
- 使用 -a 选项可以显示隐藏文件，即以 . 开头的文件
