# man - 显示命令手册

## 1. 命令简介
man 命令用于显示命令的手册页，是 Linux 系统中最常用的帮助工具之一。

## 2. 命令语法
```bash
man [选项] 命令
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -f | 显示命令的简短描述 |
| -k | 搜索包含指定关键词的手册页 |
| -a | 显示所有匹配的手册页 |
| -t | 以troff格式输出 |
| -H | 在浏览器中显示手册页 |

## 4. 手册页部分
| 部分 | 内容 |
|------|------|
| 1 | 可执行命令 |
| 2 | 系统调用 |
| 3 | 库函数 |
| 4 | 特殊文件 |
| 5 | 文件格式 |
| 6 | 游戏 |
| 7 | 杂项 |
| 8 | 系统管理命令 |

## 5. 实战示例

### 5.1 显示命令的手册页
```bash
man ls
```

### 5.2 显示命令的简短描述
```bash
man -f ls
```

### 5.3 搜索包含指定关键词的手册页
```bash
man -k file
```

### 5.4 显示特定部分的手册页
```bash
man 3 printf
```

### 5.5 显示所有匹配的手册页
```bash
man -a passwd
```

## 6. 注意事项
- man 命令是获取命令帮助信息的最常用方式
- 手册页通常包含命令的详细说明、选项、示例等信息
- 使用上下箭头键可以在手册页中导航，按 q 键退出
