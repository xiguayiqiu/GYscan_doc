# id - 显示用户和组信息

## 1. 命令简介
id 命令用于显示用户和组信息，是 Linux 系统中常用的命令之一。

## 2. 命令语法
```bash
id [选项] [用户]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -u | 只显示用户ID |
| -g | 只显示主组ID |
| -G | 显示所有组ID |
| -n | 显示名称而不是ID |
| -a | 显示所有信息 |

## 4. 实战示例

### 4.1 显示当前用户信息
```bash
id
```

### 4.2 只显示用户ID
```bash
id -u
```

### 4.3 只显示主组ID
```bash
id -g
```

### 4.4 显示所有组ID
```bash
id -G
```

### 4.5 显示用户名称而不是ID
```bash
id -un
```

### 4.6 显示指定用户信息
```bash
id username
```

## 5. 注意事项
- id 命令默认显示当前用户的信息
- 使用 -n 选项可以显示名称而不是ID
- 可以指定用户名来显示其他用户的信息
