# date - 显示或设置系统日期和时间

## 1. 命令简介
date 命令用于显示或设置系统的日期和时间，是 Linux 系统中常用的命令之一。

## 2. 命令语法
```bash
date [选项] [格式]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -s | 设置系统日期和时间 |
| -d | 显示指定日期和时间 |
| -u | 显示UTC时间 |
| -R | 显示RFC 2822格式的日期 |
| -I | 显示ISO 8601格式的日期 |

## 4. 常用格式
| 格式 | 说明 |
|------|------|
| %Y | 年份（4位） |
| %m | 月份（01-12） |
| %d | 日期（01-31） |
| %H | 小时（00-23） |
| %M | 分钟（00-59） |
| %S | 秒（00-59） |
| %a | 星期几（缩写） |
| %A | 星期几（完整） |
| %b | 月份（缩写） |
| %B | 月份（完整） |

## 5. 实战示例

### 5.1 显示当前日期和时间
```bash
date
```

### 5.2 以指定格式显示日期和时间
```bash
date +"%Y-%m-%d %H:%M:%S"
```

### 5.3 设置系统日期和时间
```bash
sudo date -s "2023-12-25 12:00:00"
```

### 5.4 显示指定日期和时间
```bash
date -d "tomorrow"
date -d "next week"
date -d "last month"
```

### 5.5 显示UTC时间
```bash
date -u
```

### 5.6 显示ISO 8601格式的日期
```bash
date -I
```

## 6. 注意事项
- date 命令默认显示当前系统的日期和时间
- 使用 -s 选项设置系统日期和时间需要 root 权限
- 可以使用不同的格式选项来显示不同格式的日期和时间
