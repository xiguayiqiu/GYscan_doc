# halt - 停止系统

## 1. 命令简介
halt 命令用于停止系统，是 Linux 系统中常用的系统管理命令之一。

## 2. 命令语法
```bash
halt [选项]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -f | 强制停止系统，不调用 shutdown 命令 |
| -i | 在停止系统前关闭所有网络接口 |
| -p | 停止系统后关闭电源 |
| --help | 显示帮助信息 |

## 4. 实战示例

### 4.1 停止系统
```bash
sudo halt
```

### 4.2 强制停止系统
```bash
sudo halt -f
```

### 4.3 停止系统后关闭电源
```bash
sudo halt -p
```

## 5. 注意事项
- halt 命令需要 root 权限，使用 sudo 执行
- halt 命令会停止系统，但默认不会关闭电源
- 使用 -p 选项可以在停止系统后关闭电源
- 在现代 Linux 系统中，halt 命令通常是 systemctl 命令的别名
