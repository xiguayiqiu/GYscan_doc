# type - 显示命令类型

## 1. 命令简介
type 命令用于显示命令的类型，是 Linux 系统中常用的命令之一。

## 2. 命令语法
```bash
type [选项] 命令
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -t | 只显示命令类型 |
| -p | 只显示可执行文件的路径 |
| -a | 显示所有匹配的命令 |

## 4. 命令类型
| 类型 | 说明 |
|------|------|
| alias | 命令别名 |
| function | 函数 |
| builtin | 内置命令 |
| file | 可执行文件 |
| keyword | 关键字 |

## 5. 实战示例

### 5.1 显示命令类型
```bash
type ls
```

### 5.2 只显示命令类型
```bash
type -t ls
```

### 5.3 只显示可执行文件的路径
```bash
type -p ls
```

### 5.4 显示所有匹配的命令
```bash
type -a ls
```

## 6. 注意事项
- type 命令可以显示命令的类型，如别名、函数、内置命令或可执行文件
- 使用 -t 选项可以只显示命令类型，更加简洁
- 与 which 命令不同，type 可以显示内置命令和函数
