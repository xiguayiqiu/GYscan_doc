# od - 八进制转储

## 1. 命令简介
od 命令用于生成文件的八进制转储，是 Linux 系统中常用的系统工具命令之一。

## 2. 命令语法
```bash
od [选项] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -t | 指定输出格式 |
| -A | 指定地址格式 |
| -j | 跳过指定字节数 |
| -N | 只显示指定字节数 |
| -v | 显示所有数据，不压缩重复行 |

## 4. 常用输出格式
| 格式 | 说明 |
|------|------|
| o | 八进制 |
| d | 十进制 |
| x | 十六进制 |
| c | ASCII码 |
| u | 无符号十进制 |

## 5. 实战示例

### 5.1 生成八进制转储
```bash
od file.txt
```

### 5.2 指定输出格式为十六进制
```bash
od -t x1 file.txt
```

### 5.3 指定输出格式为ASCII码
```bash
od -t c file.txt
```

### 5.4 跳过指定字节数
```bash
od -j 10 file.txt
```

### 5.5 只显示指定字节数
```bash
od -N 100 file.txt
```

### 5.6 显示所有数据，不压缩重复行
```bash
od -v file.txt
```

## 6. 注意事项
- od 命令用于生成文件的八进制转储，也可以指定其他格式
- 使用 -t 选项可以指定输出格式
- 使用 -j 选项可以跳过文件开头的指定字节数
- 使用 -N 选项可以只显示文件的前指定字节数
