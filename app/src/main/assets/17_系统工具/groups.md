# groups - 显示用户所属的组

## 1. 命令简介
groups 命令用于显示用户所属的组，是 Linux 系统中常用的命令之一。

## 2. 命令语法
```bash
groups [用户]
```

## 3. 实战示例

### 3.1 显示当前用户所属的组
```bash
groups
```

### 3.2 显示指定用户所属的组
```bash
groups username
```

## 4. 退出状态
- 0：命令执行成功
- 非0：命令执行失败

## 5. 注意事项
- groups 命令默认显示当前用户所属的组
- 可以指定用户名来显示其他用户所属的组
- 输出结果中，第一个组是用户的主组，后面的是附加组
