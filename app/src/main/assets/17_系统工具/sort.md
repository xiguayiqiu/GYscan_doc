# sort - 排序文本

## 1. 命令简介
sort 命令用于对文本文件进行排序，是 Linux 系统中常用的命令之一。

## 2. 命令语法
```bash
sort [选项] 文件
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -n | 按数字排序 |
| -r | 反向排序 |
| -k | 按指定列排序 |
| -t | 指定分隔符 |
| -u | 去除重复行 |
| -f | 忽略大小写 |
| -b | 忽略前导空白 |

## 4. 实战示例

### 4.1 对文件进行排序
```bash
sort file.txt
```

### 4.2 按数字排序
```bash
sort -n file.txt
```

### 4.3 反向排序
```bash
sort -r file.txt
```

### 4.4 按指定列排序
```bash
sort -k 2 file.txt
```

### 4.5 指定分隔符
```bash
sort -t, -k 2 file.csv
```

### 4.6 去除重复行
```bash
sort -u file.txt
```

## 5. 注意事项
- sort 命令默认按字典顺序排序
- 使用 -n 选项可以按数字排序
- 使用 -k 选项可以按指定列排序，列号从1开始
