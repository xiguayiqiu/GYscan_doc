# watch - 定期执行命令

## 1. 命令简介
watch 命令用于定期执行指定的命令并显示结果，是 Linux 系统中常用的命令之一。

## 2. 命令语法
```bash
watch [选项] 命令
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -n | 指定执行命令的间隔时间（秒） |
| -d | 高亮显示变化的部分 |
| -t | 不显示标题 |
| -b | 命令执行失败时发出警报 |
| -e | 命令执行失败时退出 |

## 4. 实战示例

### 4.1 每2秒执行一次命令
```bash
watch -n 2 ls -la
```

### 4.2 高亮显示变化的部分
```bash
watch -d ls -la
```

### 4.3 不显示标题
```bash
watch -t ls -la
```

### 4.4 命令执行失败时发出警报
```bash
watch -b ls -la non_existent_file
```

### 4.5 命令执行失败时退出
```bash
watch -e ls -la non_existent_file
```

## 5. 注意事项
- watch 命令默认每2秒执行一次指定的命令
- 使用 -n 选项可以指定执行命令的间隔时间
- 使用 -d 选项可以高亮显示变化的部分，便于观察变化
