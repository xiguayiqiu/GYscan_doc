# lsusb - 列出USB设备

## 1. 命令简介
lsusb 命令用于列出USB设备，是 Linux 系统中常用的系统工具命令之一。

## 2. 命令语法
```bash
lsusb [选项]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -v | 显示详细信息 |
| -s | 显示指定总线和设备号的设备 |
| -d | 显示指定厂商和产品ID的设备 |
| -t | 以树状结构显示 |

## 4. 实战示例

### 4.1 列出所有USB设备
```bash
lsusb
```

### 4.2 显示详细信息
```bash
lsusb -v
```

### 4.3 显示指定总线和设备号的设备
```bash
lsusb -s 001:002
```

### 4.4 显示指定厂商和产品ID的设备
```bash
lsusb -d 046d:
```

### 4.5 以树状结构显示
```bash
lsusb -t
```

## 5. 注意事项
- lsusb 命令用于显示系统中的USB设备信息
- 使用 -v 选项可以显示更详细的设备信息
- 使用 -s 选项可以指定显示特定的USB总线和设备号的设备
- lsusb 命令的输出结果包括设备的厂商、型号、USB总线地址等信息
