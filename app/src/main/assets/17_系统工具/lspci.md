# lspci - 列出PCI设备

## 1. 命令简介
lspci 命令用于列出PCI设备，是 Linux 系统中常用的系统工具命令之一。

## 2. 命令语法
```bash
lspci [选项]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -v | 显示详细信息 |
| -vv | 显示更详细的信息 |
| -vvv | 显示最详细的信息 |
| -n | 以数字形式显示PCI ID |
| -s | 显示指定PCI总线设备 |
| -d | 显示指定厂商和设备ID的设备 |
| -i | 指定PCI ID数据库文件 |
| -m | 以机器可读格式显示 |
| -t | 以树状结构显示 |

## 4. 实战示例

### 4.1 列出所有PCI设备
```bash
lspci
```

### 4.2 显示详细信息
```bash
lspci -v
```

### 4.3 以数字形式显示PCI ID
```bash
lspci -n
```

### 4.4 显示指定PCI总线设备
```bash
lspci -s 00:02.0
```

### 4.5 以树状结构显示
```bash
lspci -t
```

### 4.6 显示指定厂商和设备ID的设备
```bash
lspci -d 10de:
```

## 5. 注意事项
- lspci 命令用于显示系统中的PCI设备信息
- 使用 -v 选项可以显示更详细的设备信息
- 使用 -s 选项可以指定显示特定的PCI总线设备
- lspci 命令的输出结果包括设备的厂商、型号、PCI总线地址等信息
