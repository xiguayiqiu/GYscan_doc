# hexdump - 十六进制转储

## 1. 命令简介
hexdump 命令用于生成文件的十六进制转储，是 Linux 系统中常用的系统工具命令之一。

## 2. 命令语法
```bash
hexdump [选项] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -C | 显示十六进制和ASCII码 |
| -c | 显示ASCII码 |
| -d | 以双字节十进制显示 |
| -o | 以双字节八进制显示 |
| -x | 以双字节十六进制显示 |
| -s | 跳过指定字节数 |
| -n | 只显示指定字节数 |
| -v | 显示所有数据，不压缩重复行 |

## 4. 实战示例

### 4.1 生成十六进制转储
```bash
hexdump file.txt
```

### 4.2 显示十六进制和ASCII码
```bash
hexdump -C file.txt
```

### 4.3 显示ASCII码
```bash
hexdump -c file.txt
```

### 4.4 跳过指定字节数
```bash
hexdump -s 10 file.txt
```

### 4.5 只显示指定字节数
```bash
hexdump -n 100 file.txt
```

### 4.6 显示所有数据，不压缩重复行
```bash
hexdump -v file.txt
```

## 5. 注意事项
- hexdump 命令用于生成文件的十六进制转储，适合查看二进制文件
- 使用 -C 选项可以同时显示十六进制和ASCII码，方便查看
- 使用 -s 选项可以跳过文件开头的指定字节数
- 使用 -n 选项可以只显示文件的前指定字节数
