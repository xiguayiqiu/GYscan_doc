# env - 显示环境变量

## 1. 命令简介
env 命令用于显示环境变量，是 Linux 系统中常用的命令之一。

## 2. 命令语法
```bash
env [选项] [命令]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -i | 清除所有环境变量 |
| -u | 移除指定的环境变量 |
| -v | 详细模式 |

## 4. 实战示例

### 4.1 显示所有环境变量
```bash
env
```

### 4.2 清除所有环境变量并执行命令
```bash
env -i bash
```

### 4.3 移除指定的环境变量
```bash
env -u HOME bash
```

### 4.4 在特定环境变量下执行命令
```bash
env VAR=value command
```

## 5. 注意事项
- env 命令默认显示所有环境变量
- 使用 -i 选项可以清除所有环境变量
- 使用 -u 选项可以移除指定的环境变量
- 可以在执行命令时设置临时环境变量
