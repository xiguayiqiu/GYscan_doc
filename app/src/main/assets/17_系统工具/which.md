# which - 查找命令路径

## 1. 命令简介
which 命令用于查找命令的可执行文件路径，是 Linux 系统中最常用的命令之一。

## 2. 命令语法
```bash
which [选项] 命令
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -a | 显示所有匹配的路径 |
| -p | 显示命令的完整路径 |

## 4. 实战示例

### 4.1 查找命令的路径
```bash
which ls
```

### 4.2 显示所有匹配的路径
```bash
which -a ls
```

### 4.3 查找多个命令的路径
```bash
which ls cp mv
```

## 5. 注意事项
- which 命令根据 PATH 环境变量查找命令的可执行文件
- which 命令只返回第一个匹配的路径，使用 -a 选项可以显示所有匹配的路径
- 与 `whereis` 命令不同，which 只查找可执行文件
