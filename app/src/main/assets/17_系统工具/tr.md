# tr - 转换或删除字符

## 1. 命令简介
tr 命令用于转换或删除字符，是 Linux 系统中常用的命令之一。

## 2. 命令语法
```bash
tr [选项] 集合1 集合2
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -d | 删除集合1中的字符 |
| -s | 压缩重复的字符 |
| -c | 取反集合1 |
| -t | 截断集合1到集合2的长度 |

## 4. 实战示例

### 4.1 转换字符
```bash
echo "Hello World" | tr 'a-z' 'A-Z'
```

### 4.2 删除字符
```bash
echo "Hello World" | tr -d 'o'
```

### 4.3 压缩重复的字符
```bash
echo "Hello   World" | tr -s ' '
```

### 4.4 取反集合
```bash
echo "Hello World 123" | tr -c 'a-zA-Z' ' '
```

### 4.5 转换空格为换行符
```bash
echo "Hello World" | tr ' ' '\n'
```

### 4.6 删除非数字字符
```bash
echo "Hello 123 World 456" | tr -d -c '0-9'
```

## 5. 注意事项
- tr 命令从标准输入读取数据，将结果输出到标准输出
- 集合可以是字符范围，如 'a-z' 表示所有小写字母
- 使用 -d 选项可以删除指定的字符
