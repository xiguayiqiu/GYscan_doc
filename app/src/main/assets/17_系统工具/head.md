# head - 显示文件开头

## 1. 命令简介
head 命令用于显示文件的开头部分，是 Linux 系统中常用的命令之一。

## 2. 命令语法
```bash
head [选项] 文件
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -n | 指定显示的行数 |
| -c | 指定显示的字节数 |
| -q | 不显示文件名 |
| -v | 显示文件名 |

## 4. 实战示例

### 4.1 显示文件的前10行
```bash
head file.txt
```

### 4.2 显示文件的前5行
```bash
head -n 5 file.txt
```

### 4.3 显示文件的前100字节
```bash
head -c 100 file.txt
```

### 4.4 不显示文件名
```bash
head -q file1.txt file2.txt
```

### 4.5 显示文件名
```bash
head -v file.txt
```

## 5. 注意事项
- head 命令默认显示文件的前10行
- 可以使用 -n 选项指定显示的行数
- 可以使用 -c 选项指定显示的字节数
