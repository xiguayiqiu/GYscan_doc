# shutdown - 关闭系统

## 1. 命令简介
shutdown 命令用于关闭系统，是 Linux 系统中常用的系统管理命令之一。

## 2. 命令语法
```bash
shutdown [选项] [时间] [消息]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -h | 关闭系统后关闭电源 |
| -r | 重启系统 |
| -c | 取消正在进行的关机操作 |
| -k | 发送关机警告，但不实际关机 |
| --help | 显示帮助信息 |

## 4. 时间格式
| 格式 | 说明 |
|------|------|
| now | 立即关机 |
| +n | n分钟后关机 |
| hh:mm | 在指定时间关机 |

## 5. 实战示例

### 5.1 立即关机
```bash
sudo shutdown now
```

### 5.2 5分钟后关机
```bash
sudo shutdown +5
```

### 5.3 在指定时间关机
```bash
sudo shutdown 23:00
```

### 5.4 重启系统
```bash
sudo shutdown -r now
```

### 5.5 取消关机操作
```bash
sudo shutdown -c
```

### 5.6 发送关机警告
```bash
sudo shutdown -k +10 "系统将在10分钟后关机"
```

## 6. 注意事项
- shutdown 命令需要 root 权限，使用 sudo 执行
- shutdown 命令会向所有登录用户发送关机通知
- 使用 -r 选项可以重启系统
- 使用 -c 选项可以取消正在进行的关机操作
