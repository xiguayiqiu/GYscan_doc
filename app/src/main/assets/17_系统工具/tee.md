# tee - 读取标准输入并写入文件和标准输出

## 1. 命令简介
tee 命令用于读取标准输入并同时写入文件和标准输出，是 Linux 系统中常用的命令之一。

## 2. 命令语法
```bash
tee [选项] 文件
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -a | 追加到文件，而不是覆盖 |
| -i | 忽略中断信号 |
| -p | 诊断写入错误 |

## 4. 实战示例

### 4.1 将命令输出同时显示和写入文件
```bash
ls -la | tee output.txt
```

### 4.2 追加到文件
```bash
ls -la | tee -a output.txt
```

### 4.3 写入多个文件
```bash
ls -la | tee file1.txt file2.txt
```

### 4.4 忽略中断信号
```bash
ls -la | tee -i output.txt
```

## 5. 注意事项
- tee 命令可以同时将标准输入写入文件和标准输出，常用于同时查看和保存命令输出
- 使用 -a 选项可以追加到文件，而不是覆盖
- tee 命令可以写入多个文件，只需在命令后指定多个文件名
