# cd - 切换目录

## 1. 命令简介
cd 命令用于切换当前工作目录，是 Linux 系统中最常用的命令之一。

## 2. 命令语法
```bash
cd [目录]
```

## 3. 特殊目录
| 符号 | 说明 |
|------|------|
| ~ | 当前用户的主目录 |
| . | 当前目录 |
| .. | 上一级目录 |
| / | 根目录 |

## 4. 实战示例

### 4.1 切换到指定目录
```bash
cd /home/user/Documents
```

### 4.2 切换到用户主目录
```bash
cd ~
cd
```

### 4.3 切换到上一级目录
```bash
cd ..
```

### 4.4 切换到当前目录
```bash
cd .
```

### 4.5 切换到根目录
```bash
cd /
```

### 4.6 切换到上次的目录
```bash
cd -
```

## 5. 注意事项
- cd 命令是 shell 的内置命令，不是独立的可执行文件
- 可以使用相对路径或绝对路径来指定目录
- 使用 `cd -` 可以在两个目录之间快速切换
