# cat - 连接文件并打印到标准输出

## 1. 命令简介
cat 命令用于连接文件并打印到标准输出，是 Linux 系统中最常用的命令之一。

## 2. 命令语法
```bash
cat [选项] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -n | 显示行号 |
| -b | 显示行号（只对非空行） |
| -s | 压缩连续的空行 |
| -A | 显示所有字符，包括控制字符 |
| -E | 显示行尾的 $ 符号 |
| -T | 显示制表符为 ^I |

## 4. 实战示例

### 4.1 显示文件内容
```bash
cat file.txt
```

### 4.2 显示文件内容并显示行号
```bash
cat -n file.txt
```

### 4.3 连接多个文件
```bash
cat file1.txt file2.txt > combined.txt
```

### 4.4 压缩连续的空行
```bash
cat -s file.txt
```

### 4.5 显示所有字符
```bash
cat -A file.txt
```

### 4.6 从标准输入读取数据
```bash
echo "Hello" | cat
```

## 5. 注意事项
- cat 命令默认将文件内容输出到标准输出
- 使用重定向符 > 可以将输出写入文件，使用 >> 可以追加到文件
- cat 命令也可以从标准输入读取数据，如 `cat | grep pattern`
- 对于大文件，使用 less 或 more 命令可能更适合查看
