# uniq - 去除重复行

## 1. 命令简介
uniq 命令用于去除文件中的重复行，是 Linux 系统中常用的命令之一。

## 2. 命令语法
```bash
uniq [选项] 文件
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -c | 显示每行出现的次数 |
| -d | 只显示重复的行 |
| -u | 只显示不重复的行 |
| -i | 忽略大小写 |
| -f | 忽略前n个字段 |
| -s | 忽略前n个字符 |

## 4. 实战示例

### 4.1 去除文件中的重复行
```bash
sort file.txt | uniq
```

### 4.2 显示每行出现的次数
```bash
sort file.txt | uniq -c
```

### 4.3 只显示重复的行
```bash
sort file.txt | uniq -d
```

### 4.4 只显示不重复的行
```bash
sort file.txt | uniq -u
```

### 4.5 忽略大小写
```bash
sort -f file.txt | uniq -i
```

### 4.6 忽略前2个字段
```bash
sort file.txt | uniq -f 2
```

## 5. 注意事项
- uniq 命令只对相邻的重复行进行去重，因此通常需要先使用 sort 命令对文件进行排序
- 使用 -c 选项可以统计每行出现的次数
- 使用 -d 选项可以只显示重复的行
