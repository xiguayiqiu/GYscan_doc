# uname - 显示系统信息

## 1. 命令简介
uname 命令用于显示系统信息，如内核版本、硬件架构等，是 Linux 系统中最常用的系统信息工具之一。

## 2. 命令语法
```bash
uname [选项]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -a | 显示所有系统信息 |
| -s | 显示内核名称 |
| -n | 显示主机名 |
| -r | 显示内核版本 |
| -v | 显示内核版本的详细信息 |
| -m | 显示硬件架构 |
| -p | 显示处理器类型 |
| -i | 显示硬件平台 |
| -o | 显示操作系统类型 |

## 4. 实战示例

### 4.1 显示所有系统信息
```bash
uname -a
```

### 4.2 显示内核名称
```bash
uname -s
```

### 4.3 显示内核版本
```bash
uname -r
```

### 4.4 显示硬件架构
```bash
uname -m
```

### 4.5 显示主机名
```bash
uname -n
```

### 4.6 显示操作系统类型
```bash
uname -o
```

## 5. 注意事项
- uname 命令不需要任何参数就可以执行，默认显示内核名称
- 使用 -a 选项可以显示所有系统信息，是最常用的选项
- uname 命令的输出可以用于脚本中，以确定系统类型和版本
