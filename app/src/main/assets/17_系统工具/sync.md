# sync - 同步文件系统缓存

## 1. 命令简介
sync 命令用于同步文件系统缓存，是 Linux 系统中常用的系统工具命令之一。

## 2. 命令语法
```bash
sync [选项]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 同步文件系统缓存
```bash
sync
```

## 5. 注意事项
- sync 命令用于将内存中的文件系统缓存同步到磁盘
- sync 命令可以确保所有修改过的文件都被写入磁盘
- 在关闭系统或卸载文件系统之前，建议执行 sync 命令
- sync 命令没有参数，直接执行即可
