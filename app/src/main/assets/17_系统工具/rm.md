# rm - 删除文件或目录

## 1. 命令简介
rm 命令用于删除文件或目录，是 Linux 系统中最常用的命令之一。

## 2. 命令语法
```bash
rm [选项] 文件或目录
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -f | 强制删除，不提示 |
| -i | 交互式删除，提示确认 |
| -r | 递归删除目录及其内容 |
| -v | 显示删除的文件 |

## 4. 实战示例

### 4.1 删除文件
```bash
rm file.txt
```

### 4.2 强制删除文件
```bash
rm -f file.txt
```

### 4.3 交互式删除文件
```bash
rm -i file.txt
```

### 4.4 递归删除目录
```bash
rm -r directory
```

### 4.5 强制递归删除目录
```bash
rm -rf directory
```

### 4.6 显示删除的文件
```bash
rm -v file.txt
```

## 5. 注意事项
- rm 命令默认只删除文件，不删除目录
- 使用 -r 选项可以递归删除目录及其内容
- 使用 -f 选项可以强制删除，不提示确认，使用时要谨慎
- rm 命令删除的文件不会进入回收站，删除后无法恢复，使用时要谨慎
