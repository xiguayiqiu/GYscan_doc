# echo - 输出文本

## 1. 命令简介
echo 命令用于输出文本，是 Linux 系统中最常用的命令之一。

## 2. 命令语法
```bash
echo [选项] [文本]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -n | 不输出换行符 |
| -e | 启用转义字符 |

## 4. 常用转义字符
| 转义字符 | 说明 |
|----------|------|
| \n | 换行符 |
| \t | 制表符 |
| \r | 回车符 |
| \\ | 反斜杠 |
| \" | 双引号 |
| \' | 单引号 |

## 5. 实战示例

### 5.1 输出文本
```bash
echo "Hello World"
```

### 5.2 不输出换行符
```bash
echo -n "Hello World"
```

### 5.3 启用转义字符
```bash
echo -e "Hello\nWorld"
echo -e "Hello\tWorld"
```

### 5.4 输出变量
```bash
name="John"
echo "Hello $name"
```

### 5.5 输出命令执行结果
```bash
echo "Current directory: $(pwd)"
```

### 5.6 输出特殊字符
```bash
echo "\\"
```

## 6. 注意事项
- echo 命令默认会在输出的末尾添加一个换行符，使用 -n 选项可以禁用
- 使用 -e 选项可以启用转义字符，如 \n（换行）、\t（制表符）等
- 可以使用双引号或单引号来包含包含空格的文本，双引号会展开变量，单引号不会
