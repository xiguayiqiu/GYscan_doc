# sleep - 暂停执行

## 1. 命令简介
sleep 命令用于暂停执行指定的时间，是 Linux 系统中常用的命令之一。

## 2. 命令语法
```bash
sleep [时间]
```

## 3. 时间单位
| 单位 | 说明 |
|------|------|
| s | 秒（默认） |
| m | 分钟 |
| h | 小时 |
| d | 天 |

## 4. 实战示例

### 4.1 暂停1秒
```bash
sleep 1
```

### 4.2 暂停1分钟
```bash
sleep 1m
```

### 4.3 暂停1小时
```bash
sleep 1h
```

### 4.4 暂停1天
```bash
sleep 1d
```

### 4.5 在脚本中使用
```bash
echo "开始"
sleep 5
echo "5秒后"
```

## 5. 注意事项
- sleep 命令可以指定不同的时间单位，默认是秒
- sleep 命令常用于脚本中，用于延迟执行或定时执行
- sleep 命令会暂停当前进程的执行，直到指定的时间过去
