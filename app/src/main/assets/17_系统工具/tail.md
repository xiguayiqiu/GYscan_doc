# tail - 显示文件结尾

## 1. 命令简介
tail 命令用于显示文件的结尾部分，是 Linux 系统中常用的命令之一。

## 2. 命令语法
```bash
tail [选项] 文件
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -n | 指定显示的行数 |
| -c | 指定显示的字节数 |
| -f | 实时跟踪文件的变化 |
| -q | 不显示文件名 |
| -v | 显示文件名 |

## 4. 实战示例

### 4.1 显示文件的最后10行
```bash
tail file.txt
```

### 4.2 显示文件的最后5行
```bash
tail -n 5 file.txt
```

### 4.3 实时跟踪文件的变化
```bash
tail -f file.txt
```

### 4.4 显示文件的最后100字节
```bash
tail -c 100 file.txt
```

### 4.5 不显示文件名
```bash
tail -q file1.txt file2.txt
```

### 4.6 显示文件名
```bash
tail -v file.txt
```

## 5. 注意事项
- tail 命令默认显示文件的最后10行
- 使用 -f 选项可以实时跟踪文件的变化，常用于监控日志文件
- 可以使用 -n 选项指定显示的行数
