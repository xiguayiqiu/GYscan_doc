# cut - 剪切文本

## 1. 命令简介
cut 命令用于从文本文件中剪切字段，是 Linux 系统中常用的命令之一。

## 2. 命令语法
```bash
cut [选项] 文件
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -d | 指定分隔符 |
| -f | 指定要剪切的字段 |
| -c | 按字符位置剪切 |
| -b | 按字节位置剪切 |
| -s | 只显示包含分隔符的行 |

## 4. 实战示例

### 4.1 按字段剪切
```bash
cut -d, -f 1 file.csv
```

### 4.2 剪切多个字段
```bash
cut -d, -f 1,3 file.csv
```

### 4.3 按字符位置剪切
```bash
cut -c 1-5 file.txt
```

### 4.4 按字节位置剪切
```bash
cut -b 1-5 file.txt
```

### 4.5 只显示包含分隔符的行
```bash
cut -d, -s -f 1 file.csv
```

### 4.6 从标准输入读取数据
```bash
echo "Hello,World" | cut -d, -f 1
```

## 5. 注意事项
- cut 命令默认使用制表符作为分隔符
- 使用 -d 选项可以指定其他分隔符
- 使用 -f 选项可以指定要剪切的字段，字段号从1开始
