# cal - 显示日历

## 1. 命令简介
cal 命令用于显示日历，是 Linux 系统中常用的命令之一。

## 2. 命令语法
```bash
cal [选项] [月份] [年份]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -3 | 显示前一个月、当前月和后一个月的日历 |
| -y | 显示全年的日历 |
| -j | 显示一年中的第几天 |
| -m | 以周一为一周的开始 |

## 4. 实战示例

### 4.1 显示当前月的日历
```bash
cal
```

### 4.2 显示指定月份的日历
```bash
cal 12 2023
```

### 4.3 显示前一个月、当前月和后一个月的日历
```bash
cal -3
```

### 4.4 显示全年的日历
```bash
cal -y
```

### 4.5 显示一年中的第几天
```bash
cal -j
```

### 4.6 以周一为一周的开始
```bash
cal -m
```

## 5. 注意事项
- cal 命令默认显示当前月的日历
- 可以指定月份和年份来显示特定月份的日历
- 使用 -y 选项可以显示全年的日历
