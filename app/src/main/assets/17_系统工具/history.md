# history - 显示命令历史

## 1. 命令简介
history 命令用于显示命令历史记录，是 Linux 系统中常用的命令之一。

## 2. 命令语法
```bash
history [选项]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -c | 清除命令历史 |
| -d | 删除指定的历史记录 |
| -a | 将当前会话的历史记录追加到历史文件 |
| -n | 从历史文件中读取未读取的历史记录 |
| -r | 读取历史文件并覆盖当前历史记录 |
| -w | 将当前历史记录写入历史文件 |

## 4. 实战示例

### 4.1 显示命令历史
```bash
history
```

### 4.2 显示最近的10条命令历史
```bash
history 10
```

### 4.3 清除命令历史
```bash
history -c
```

### 4.4 删除指定的历史记录
```bash
history -d 10
```

### 4.5 将当前会话的历史记录追加到历史文件
```bash
history -a
```

### 4.6 搜索命令历史
```bash
history | grep command
```

## 5. 注意事项
- 命令历史默认保存在 ~/.bash_history 文件中
- 可以使用 ! 命令来执行历史记录中的命令，如 !10 执行第10条命令
- 可以使用 !! 来执行上一条命令
