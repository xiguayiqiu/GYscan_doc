# dmesg - 显示内核消息

## 1. 命令简介
dmesg 命令用于显示内核消息，是 Linux 系统中常用的系统工具命令之一。

## 2. 命令语法
```bash
dmesg [选项]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -c | 显示消息后清除缓冲区 |
| -d | 显示消息之间的时间差 |
| -f | 显示指定设施的消息 |
| -H | 人类可读格式 |
| -k | 只显示内核消息 |
| -l | 显示指定级别的消息 |
| -n | 设置控制台日志级别 |
| -r | 反转输出顺序 |
| -s | 缓冲区大小 |
| -T | 显示人类可读的时间戳 |
| -x | 解码设施和级别 |

## 4. 常用设施
| 设施 | 说明 |
|------|------|
| kern | 内核消息 |
| user | 用户空间消息 |
| mail | 邮件系统消息 |
| daemon | 守护进程消息 |
| auth | 认证消息 |
| syslog | 系统日志消息 |
| lpr | 打印系统消息 |
| news | 新闻系统消息 |

## 5. 常用级别
| 级别 | 说明 |
|------|------|
| emerg | 紧急消息 |
| alert | 警报消息 |
| crit | 关键消息 |
| err | 错误消息 |
| warn | 警告消息 |
| notice | 通知消息 |
| info | 信息消息 |
| debug | 调试消息 |

## 6. 实战示例

### 6.1 显示所有内核消息
```bash
dmesg
```

### 6.2 显示人类可读的时间戳
```bash
dmesg -T
```

### 6.3 显示错误消息
```bash
dmesg -l err
```

### 6.4 显示内核消息
```bash
dmesg -k
```

### 6.5 显示消息后清除缓冲区
```bash
dmesg -c
```

### 6.6 显示消息之间的时间差
```bash
dmesg -d
```

## 7. 退出状态
- 0：命令执行成功
- 非0：命令执行失败

## 8. 注意事项
- dmesg 命令可以显示内核启动和运行过程中的消息
- 使用 -T 选项可以显示人类可读的时间戳，使输出更加易读
- 使用 -l 选项可以过滤显示指定级别的消息
- 使用 -f 选项可以过滤显示指定设施的消息
