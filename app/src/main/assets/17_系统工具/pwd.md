# pwd - 显示当前工作目录

## 1. 命令简介
pwd 命令用于显示当前工作目录的绝对路径，是 Linux 系统中最常用的命令之一。

## 2. 命令语法
```bash
pwd [选项]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -L | 显示逻辑路径（默认） |
| -P | 显示物理路径，解析符号链接 |

## 4. 实战示例

### 4.1 显示当前工作目录
```bash
pwd
```

### 4.2 显示物理路径
```bash
pwd -P
```

### 4.3 在脚本中使用
```bash
echo "当前工作目录: $(pwd)"
```

## 5. 注意事项
- pwd 命令不需要任何参数就可以执行
- pwd 命令显示的是当前工作目录的绝对路径
- 使用 -P 选项可以解析符号链接，显示实际的物理路径
