# exit - 退出Shell

## 1. 命令简介
exit 命令用于退出当前Shell会话，是 Linux 系统中常用的命令之一。

## 2. 命令语法
```bash
exit [状态码]
```

## 3. 实战示例

### 3.1 退出Shell
```bash
exit
```

### 3.2 以指定状态码退出Shell
```bash
exit 0
```

## 4. 退出状态
- 0：命令执行成功
- 非0：命令执行失败

## 5. 注意事项
- exit 命令可以指定退出状态码，默认状态码是上一个命令的退出状态码
- 在Shell脚本中，exit 命令可以用于控制脚本的退出状态
- 退出状态码0表示成功，非0表示失败
