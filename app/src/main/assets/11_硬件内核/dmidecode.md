# dmidecode - 显示硬件信息

## 1. 命令简介
dmidecode 命令用于显示硬件信息，是 Linux 系统中常用的硬件内核命令之一。

## 2. 命令语法
```bash
dmidecode [选项]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -t | 指定类型 |
| -s | 指定字符串 |
| -h | 显示帮助信息 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 常用类型
| 类型 | 说明 |
|------|------|
| 0 | BIOS信息 |
| 1 | 系统信息 |
| 2 | 主板信息 |
| 3 | 机箱信息 |
| 4 | 处理器信息 |
| 5 | 内存控制器 |
| 6 | 内存模块 |
| 7 | 缓存信息 |
| 8 | 端口连接器 |
| 9 | 系统插槽 |

## 5. 实战示例

### 5.1 显示所有硬件信息
```bash
dmidecode
```

### 5.2 显示处理器信息
```bash
dmidecode -t 4
```

### 5.3 显示内存信息
```bash
dmidecode -t 6
```

### 5.4 显示系统信息
```bash
dmidecode -t 1
```

## 6. 注意事项
- dmidecode 命令用于显示硬件信息，通过读取 DMI (Desktop Management Interface) 数据
- dmidecode 命令需要 root 权限
- dmidecode 命令是一个常用的系统管理工具，用于获取硬件信息
