# rmmod - 卸载模块

## 1. 命令简介
rmmod 命令用于卸载内核模块，是 Linux 系统中常用的硬件内核命令之一。

## 2. 命令语法
```bash
rmmod [选项] [模块]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -v | 显示详细信息 |
| -f | 强制卸载 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 卸载模块
```bash
rmmod i915
```

### 4.2 显示详细信息
```bash
rmmod -v i915
```

### 4.3 强制卸载
```bash
rmmod -f i915
```

## 5. 注意事项
- rmmod 命令用于卸载内核模块
- rmmod 命令不会自动处理模块的依赖关系
- rmmod 命令需要 root 权限
- rmmod 命令是一个常用的系统管理工具，用于管理内核模块
