# dmesg - 显示内核消息

## 1. 命令简介
dmesg 命令用于显示内核消息，是 Linux 系统中常用的硬件内核命令之一。

## 2. 命令语法
```bash
dmesg [选项]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -c | 清除缓冲区 |
| -n | 设置日志级别 |
| -s | 设置缓冲区大小 |
| -t | 不显示时间戳 |
| -H | 人类可读格式 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 显示内核消息
```bash
dmesg
```

### 4.2 不显示时间戳
```bash
dmesg -t
```

### 4.3 人类可读格式
```bash
dmesg -H
```

### 4.4 清除缓冲区
```bash
dmesg -c
```

## 5. 注意事项
- dmesg 命令用于显示内核消息，包括硬件检测、驱动加载等信息
- dmesg 命令的输出通常很长，可以使用管道和其他命令进行过滤
- dmesg 命令是一个常用的系统管理工具，用于查看内核消息
