# modinfo - 显示模块信息

## 1. 命令简介
modinfo 命令用于显示内核模块的信息，是 Linux 系统中常用的硬件内核命令之一。

## 2. 命令语法
```bash
modinfo [选项] [模块]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -a | 显示作者 |
| -d | 显示描述 |
| -l | 显示许可证 |
| -p | 显示参数 |
| -n | 显示模块文件路径 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 显示模块信息
```bash
modinfo i915
```

### 4.2 显示模块文件路径
```bash
modinfo -n i915
```

### 4.3 显示模块参数
```bash
modinfo -p i915
```

## 5. 注意事项
- modinfo 命令用于显示内核模块的信息，包括作者、描述、许可证等
- modinfo 命令是一个常用的系统管理工具，用于了解内核模块的详细信息
