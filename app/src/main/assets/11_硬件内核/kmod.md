# kmod - 内核模块工具

## 1. 命令简介
kmod 命令是内核模块工具，是 Linux 系统中常用的硬件内核命令之一。

## 2. 命令语法
```bash
kmod [选项] [命令]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -h | 显示帮助信息 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 常用命令
| 命令 | 说明 |
|------|------|
| list | 列出模块 |
| load | 加载模块 |
| unload | 卸载模块 |
| info | 显示模块信息 |

## 5. 实战示例

### 5.1 列出模块
```bash
kmod list
```

### 5.2 加载模块
```bash
kmod load i915
```

### 5.3 卸载模块
```bash
kmod unload i915
```

### 5.4 显示模块信息
```bash
kmod info i915
```

## 6. 注意事项
- kmod 命令是内核模块工具，用于管理内核模块
- kmod 命令是一个常用的系统管理工具，用于管理内核模块
