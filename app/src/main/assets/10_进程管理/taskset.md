# taskset 命令

## 命令说明

`taskset` 是 Linux 系统中用于设置进程 CPU 亲和性的命令，它可以指定进程在哪些 CPU 核心上运行。

## 语法

```bash
taskset [选项] 掩码|列表 命令 [参数...]
taskset [选项] -p [掩码|列表] 进程ID
```

## 常用选项

| 选项 | 描述 |
|------|------|
| `-a` | 显示所有线程的 CPU 亲和性 |
| `-c` | 使用 CPU 列表而不是掩码 |
| `-p` | 操作指定进程的 CPU 亲和性 |

## 示例

### 以指定 CPU 亲和性运行命令

```bash
# 在 CPU 0 和 1 上运行命令
taskset -c 0,1 dd if=/dev/zero of=/dev/null bs=1M count=1000
```

### 查看指定进程的 CPU 亲和性

```bash
taskset -p 1234
```

### 修改指定进程的 CPU 亲和性

```bash
# 设置进程 1234 在 CPU 2 和 3 上运行
taskset -c 2,3 -p 1234
```

## 注意事项

- `taskset` 命令需要 root 权限来修改其他进程的 CPU 亲和性
- CPU 亲和性可以提高进程的性能，特别是对于多核心系统
- 掩码格式是十六进制的，每一位代表一个 CPU 核心
- 列表格式更直观，如 `0,1,2` 表示使用 CPU 0、1 和 2
