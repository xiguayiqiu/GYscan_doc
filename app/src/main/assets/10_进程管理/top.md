# top - 实时显示进程状态

## 1. 命令简介
top 命令用于实时显示进程状态，是 Linux 系统中常用的进程管理命令之一。

## 2. 命令语法
```bash
top [选项]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -d | 指定刷新间隔 |
| -p | 指定进程ID |
| -u | 指定用户 |
| -b | 批处理模式 |
| -n | 指定刷新次数 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 常用操作
| 操作 | 说明 |
|------|------|
| k | 终止进程 |
| r | 修改进程优先级 |
| q | 退出 |
| 1 | 显示所有CPU核心 |
| M | 按内存使用排序 |
| P | 按CPU使用排序 |
| T | 按时间排序 |

## 5. 实战示例

### 5.1 实时显示进程状态
```bash
top
```

### 5.2 指定刷新间隔
```bash
top -d 1
```

### 5.3 批处理模式
```bash
top -b -n 1
```

### 5.4 显示指定用户的进程
```bash
top -u user
```

## 6. 注意事项
- top 命令用于实时显示进程状态，包括CPU使用率、内存使用率等信息
- top 命令是一个常用的系统管理工具，用于监控系统资源使用情况
