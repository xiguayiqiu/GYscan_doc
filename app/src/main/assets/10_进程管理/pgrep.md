# pgrep - 查找进程

## 1. 命令简介
pgrep 命令用于查找进程，是 Linux 系统中常用的进程管理命令之一。

## 2. 命令语法
```bash
pgrep [选项] [模式]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -f | 匹配完整命令行 |
| -u | 按用户匹配 |
| -t | 按终端匹配 |
| -l | 显示进程名 |
| -n | 只显示最新的进程 |
| -o | 只显示最旧的进程 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 查找进程
```bash
pgrep process
```

### 4.2 匹配完整命令行
```bash
pgrep -f "command"
```

### 4.3 按用户匹配
```bash
pgrep -u user
```

### 4.4 显示进程名
```bash
pgrep -l process
```

## 5. 注意事项
- pgrep 命令用于查找进程，通过进程名或命令行匹配
- pgrep 命令返回匹配进程的ID
- pgrep 命令是一个常用的系统管理工具，用于查找进程
