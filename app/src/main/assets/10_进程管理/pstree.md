# pstree 命令

## 命令说明

`pstree` 是 Linux 系统中用于以树形结构显示进程的命令。

## 语法

```bash
pstree [选项] [进程ID|用户名]
```

## 常用选项

| 选项 | 描述 |
|------|------|
| `-a` | 显示命令行参数 |
| `-c` | 不合并相同的进程 |
| `-h` | 高亮显示当前进程及其父进程 |
| `-n` | 按进程 ID 排序 |
| `-p` | 显示进程 ID |
| `-u` | 显示用户切换 |

## 示例

### 显示所有进程的树形结构

```bash
pstree
```

### 显示指定进程的树形结构

```bash
pstree 1234
```

### 显示进程 ID

```bash
pstree -p
```

### 显示命令行参数

```bash
pstree -a
```

### 高亮显示当前进程

```bash
pstree -h
```

## 注意事项

- `pstree` 命令不需要特殊权限
- 输出以树形结构显示进程之间的关系，便于查看进程的层次结构
- 可以使用 `ps` 命令查看更详细的进程信息
- `pstree` 命令会默认合并相同的进程，使用 `-c` 选项可以禁用此功能
