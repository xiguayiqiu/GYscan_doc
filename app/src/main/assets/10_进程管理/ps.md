# ps - 显示进程状态

## 1. 命令简介
ps 命令用于显示进程状态，是 Linux 系统中常用的进程管理命令之一。

## 2. 命令语法
```bash
ps [选项]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -a | 显示所有进程 |
| -u | 显示用户信息 |
| -x | 显示无控制终端的进程 |
| -e | 显示所有进程 |
| -f | 显示详细信息 |
| -l | 显示长格式 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 显示所有进程
```bash
ps aux
```

### 4.2 显示详细信息
```bash
ps -ef
```

### 4.3 显示长格式
```bash
ps -l
```

### 4.4 显示指定用户的进程
```bash
ps -u user
```

## 5. 注意事项
- ps 命令用于显示进程状态，包括进程ID、用户、CPU使用率等信息
- ps 命令是一个常用的系统管理工具，用于查看进程状态
