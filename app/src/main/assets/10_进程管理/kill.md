# kill - 终止进程

## 1. 命令简介
kill 命令用于终止进程，是 Linux 系统中常用的进程管理命令之一。

## 2. 命令语法
```bash
kill [选项] [进程ID]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -l | 显示信号列表 |
| -s | 指定信号 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 常用信号
| 信号 | 说明 |
|------|------|
| 1 | SIGHUP - 挂起 |
| 2 | SIGINT - 中断 |
| 9 | SIGKILL - 强制终止 |
| 15 | SIGTERM - 终止 |

## 5. 实战示例

### 5.1 终止进程
```bash
kill 1234
```

### 5.2 强制终止进程
```bash
kill -9 1234
```

### 5.3 指定信号
```bash
kill -s SIGTERM 1234
```

### 5.4 显示信号列表
```bash
kill -l
```

## 6. 注意事项
- kill 命令用于终止进程，向进程发送信号
- kill 命令的默认信号是 SIGTERM (15)
- kill 命令是一个常用的系统管理工具，用于终止进程
