# pkill - 终止进程

## 1. 命令简介
pkill 命令用于终止进程，是 Linux 系统中常用的进程管理命令之一。

## 2. 命令语法
```bash
pkill [选项] [模式]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -f | 匹配完整命令行 |
| -u | 按用户匹配 |
| -t | 按终端匹配 |
| -l | 显示匹配的进程 |
| -n | 只匹配最新的进程 |
| -o | 只匹配最旧的进程 |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 终止匹配的进程
```bash
pkill process
```

### 4.2 匹配完整命令行
```bash
pkill -f "command"
```

### 4.3 按用户匹配
```bash
pkill -u user
```

### 4.4 显示匹配的进程
```bash
pkill -l process
```

## 5. 注意事项
- pkill 命令用于终止匹配的进程，通过进程名或命令行匹配
- pkill 命令是一个常用的系统管理工具，用于终止进程
