# prlimit 命令

## 命令说明

`prlimit` 是 Linux 系统中用于显示和修改进程资源限制的命令。

## 语法

```bash
prlimit [选项] [资源=限制] [命令 [参数...]]
```

## 常用选项

| 选项 | 描述 |
|------|------|
| `-p` | 指定进程 ID |
| `-o` | 以表格形式显示 |
| `-n` | 只显示软限制 |
| `-H` | 只显示硬限制 |

## 常用资源限制

| 资源 | 描述 |
|------|------|
| `AS` | 地址空间限制 |
| `CORE` | 核心文件大小限制 |
| `CPU` | CPU 时间限制 |
| `DATA` | 数据段大小限制 |
| `FSIZE` | 文件大小限制 |
| `NOFILE` | 打开文件数限制 |
| `NPROC` | 进程数限制 |
| `STACK` | 栈大小限制 |

## 示例

### 显示当前进程的资源限制

```bash
prlimit
```

### 显示指定进程的资源限制

```bash
prlimit -p 1234
```

### 修改指定进程的资源限制

```bash
prlimit --nofile=1024:2048 -p 1234
```

### 以指定资源限制运行命令

```bash
prlimit --nofile=1024:2048 ls -la
```

## 注意事项

- `prlimit` 命令需要 root 权限来修改其他进程的资源限制
- 资源限制分为软限制和硬限制，软限制可以在硬限制范围内调整
- 可以使用 `ulimit` 命令查看和修改当前 shell 的资源限制
- 资源限制的单位通常是字节，对于时间限制是秒
