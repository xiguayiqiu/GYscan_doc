# lonice 命令

## 命令说明

`lonice` 是 Linux 系统中用于设置进程 I/O 调度优先级的命令。

## 语法

```bash
lonice [选项] 命令 [参数...]
```

## 常用选项

| 选项 | 描述 |
|------|------|
| `-n` | 设置 I/O 优先级级别（0-7，0 最高，7 最低） |
| `-p` | 指定进程 ID |

## 优先级级别

| 级别 | 描述 |
|------|------|
| 0 | 最高优先级 |
| 1-2 | 高优先级 |
| 3-4 | 正常优先级 |
| 5-6 | 低优先级 |
| 7 | 最低优先级 |

## 示例

### 以指定 I/O 优先级运行命令

```bash
lonice -n 5 dd if=/dev/zero of=/dev/null bs=1M count=1000
```

### 修改指定进程的 I/O 优先级

```bash
lonice -n 3 -p 1234
```

## 注意事项

- `lonice` 命令需要 root 权限来修改其他进程的 I/O 优先级
- I/O 优先级影响进程对磁盘 I/O 的访问优先级
- 较高的 I/O 优先级会获得更多的磁盘带宽
- 可以使用 `ionice` 命令替代 `lonice` 命令，它们功能相同
