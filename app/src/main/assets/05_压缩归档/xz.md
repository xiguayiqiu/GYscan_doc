# xz - 使用xz压缩

## 1. 命令简介
xz 命令用于使用 xz 算法压缩文件，是 Linux 系统中常用的压缩归档命令之一。

## 2. 命令语法
```bash
xz [选项] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -z | 压缩模式 |
| -d | 解压缩模式 |
| -k | 保留原始文件 |
| -f | 强制覆盖 |
| -v | 显示详细信息 |
| -0 到 -9 | 压缩级别（默认6） |
| --help | 显示帮助信息 |
| --version | 显示版本信息 |

## 4. 实战示例

### 4.1 压缩文件
```bash
xz file.txt
```

### 4.2 保留原始文件
```bash
xz -k file.txt
```

### 4.3 解压缩文件
```bash
xz -d file.txt.xz
```

### 4.4 指定压缩级别
```bash
xz -9 file.txt
```

## 5. 注意事项
- xz 命令用于使用 xz 算法压缩文件，压缩率比 gzip 和 bzip2 更高
- xz 命令会删除原始文件，使用 -k 选项可以保留
- xz 命令是一个常用的压缩工具，用于压缩文件
