# cpio - 复制文件到归档

## 1. 命令简介
cpio 命令用于复制文件到归档，是 Linux 系统中常用的压缩归档命令之一。

## 2. 命令语法
```bash
cpio [选项] [模式]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -o | 输出模式，创建归档 |
| -i | 输入模式，提取归档 |
| -p | 传递模式，复制文件 |
| -t | 列出归档内容 |
| -v | 显示详细信息 |
| -d | 创建目录 |
| -u | 覆盖现有文件 |
| -m | 保留修改时间 |

## 4. 实战示例

### 4.1 创建归档
```bash
find . -name "*.txt" | cpio -o > archive.cpio
```

### 4.2 提取归档
```bash
cpio -i < archive.cpio
```

### 4.3 列出归档内容
```bash
cpio -t < archive.cpio
```

### 4.4 传递模式复制文件
```bash
find . -name "*.txt" | cpio -p /path/to/destination
```

## 5. 注意事项
- cpio 命令用于复制文件到归档，支持多种归档格式
- cpio 命令通常与 find 命令配合使用，以选择要归档的文件
- cpio 命令是一个常用的归档工具，用于创建和提取归档
