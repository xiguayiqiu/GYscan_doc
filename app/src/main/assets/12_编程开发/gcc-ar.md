# gcc-ar - GCC归档工具

## 1. 命令简介
gcc-ar 命令是 GCC 归档工具，是 Linux 系统中常用的编程开发命令之一。

## 2. 命令语法
```bash
gcc-ar [选项] [归档文件] [文件]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -d | 从归档文件中删除文件 |
| -m | 移动归档文件中的文件 |
| -p | 显示归档文件中的文件内容 |
| -q | 快速添加文件到归档文件 |
| -r | 替换归档文件中的文件 |
| -s | 生成归档索引 |
| -t | 显示归档文件中的文件列表 |
| -x | 从归档文件中提取文件 |

## 4. 实战示例

### 4.1 创建归档文件
```bash
gcc-ar r libhello.a hello.o world.o
```

### 4.2 显示归档文件中的文件列表
```bash
gcc-ar t libhello.a
```

### 4.3 从归档文件中提取文件
```bash
gcc-ar x libhello.a
```

### 4.4 从归档文件中删除文件
```bash
gcc-ar d libhello.a hello.o
```

## 5. 注意事项
- gcc-ar 命令是 GCC 提供的归档工具，用于创建和管理静态库
- gcc-ar 命令与 ar 命令功能类似，但与 GCC 编译器集成更紧密
- 使用 gcc-ar 命令创建的静态库可以被 gcc 编译器链接使用
