# m4 命令

## 命令简介

`m4` 是一个通用的宏处理器，它可以用于处理文本文件，替换宏，是 autoconf 和 automake 等工具的基础。

## 命令语法

```bash
m4 [options] [file...]
```

## 命令选项

| 选项 | 说明 |
|------|------|
| `-h`, `--help` | 显示帮助信息 |
| `-V`, `--version` | 显示版本信息 |
| `-v`, `--verbose` | 详细输出 |
| `-D NAME=VALUE` | 定义宏 |
| `-U NAME` | 取消定义宏 |
| `-I DIR` | 添加包含目录 |
| `-s`, `--synclines` | 生成同步行 |
| `-t`, `--trace=NAME` | 跟踪宏展开 |
| `-E`, `--prefix-builtins` | 为内置宏添加前缀 |

## 命令示例

### 处理文件

```bash
m4 input.m4 > output.txt
```

### 定义宏

```bash
m4 -D VERSION=1.0 input.m4 > output.txt
```

### 跟踪宏展开

```bash
m4 -t macro input.m4
```

### 使用包含目录

```bash
m4 -I include_dir input.m4 > output.txt
```

## 常用组合

### 处理 autoconf 配置文件

```bash
m4 aclocal.m4 configure.ac > configure
```

### 生成配置文件

```bash
m4 -D HOST=localhost -D PORT=8080 template.m4 > config.txt
```

## 相关命令

- `autoconf`: 生成 configure 脚本
- `automake`: Makefile 生成器
- `aclocal`: 生成 aclocal.m4 文件
