# cal - 显示日历

## 1. 命令简介
cal 命令用于显示日历，是 Linux 系统中常用的时间日期命令之一。

## 2. 命令语法
```bash
cal [选项] [月份] [年份]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -1 | 显示当前月份 |
| -3 | 显示前一个月、当前月和后一个月 |
| -y | 显示全年日历 |
| -j | 显示儒略日 |
| -m | 显示指定月份 |

## 4. 实战示例

### 4.1 显示当前月份
```bash
cal
```

### 4.2 显示前一个月、当前月和后一个月
```bash
cal -3
```

### 4.3 显示全年日历
```bash
cal -y
```

### 4.4 显示指定月份
```bash
cal 12 2024
```

### 4.5 显示儒略日
```bash
cal -j
```

## 5. 注意事项
- cal 命令用于显示日历，默认显示当前月份
- 使用 -3 选项可以显示三个月的日历
- 使用 -y 选项可以显示全年日历
- 使用 -j 选项可以显示儒略日（一年中的第几天）
- cal 命令是一个简单但实用的工具，用于查看日期和日历
