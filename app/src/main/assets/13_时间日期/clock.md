# clock 命令

## 命令说明

`clock` 是 Linux 系统中用于设置和显示硬件时钟（RTC - Real Time Clock）的命令。

## 语法

```bash
clock [选项]
```

## 常用选项

| 选项 | 描述 |
|------|------|
| `-r` | 读取并显示硬件时钟 |
| `-w` | 将系统时间写入硬件时钟 |
| `-s` | 将硬件时钟同步到系统时间 |
| `-u` | 使用 UTC 时间 |
| `-l` | 使用本地时间 |
| `-h` | 显示帮助信息 |
| `-V` | 显示版本信息 |

## 示例

### 读取并显示硬件时钟

```bash
clock -r
```

### 将系统时间写入硬件时钟

```bash
clock -w
```

### 将硬件时钟同步到系统时间

```bash
clock -s
```

### 使用 UTC 时间

```bash
clock -u -r
```

### 使用本地时间

```bash
clock -l -r
```

## 注意事项

- `clock` 命令需要 root 权限
- 硬件时钟是存储在主板上的实时时钟，在系统关机后仍然运行
- 系统时间和硬件时钟可能不同步，需要定期同步
- 现代 Linux 系统通常使用 `timedatectl` 命令来管理时间和时区
- `clock` 命令在某些系统上可能被 `hwclock` 命令替代
