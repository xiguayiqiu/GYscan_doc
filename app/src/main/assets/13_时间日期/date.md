# date - 显示日期时间

## 1. 命令简介
date 命令用于显示和设置系统日期时间，是 Linux 系统中常用的时间日期命令之一。

## 2. 命令语法
```bash
date [选项] [格式]
```

## 3. 常用选项
| 选项 | 说明 |
|------|------|
| -d | 显示指定日期时间 |
| -s | 设置系统日期时间 |
| -u | 显示UTC时间 |
| -R | 显示RFC格式的日期时间 |
| -I | 显示ISO格式的日期时间 |
| -r | 显示文件的最后修改时间 |

## 4. 常用格式
| 格式 | 说明 |
|------|------|
| %Y | 年份（4位） |
| %m | 月份（01-12） |
| %d | 日期（01-31） |
| %H | 小时（00-23） |
| %M | 分钟（00-59） |
| %S | 秒（00-59） |
| %A | 星期全称 |
| %a | 星期缩写 |
| %B | 月份全称 |
| %b | 月份缩写 |

## 5. 实战示例

### 5.1 显示当前日期时间
```bash
date
```

### 5.2 显示指定格式的日期时间
```bash
date +"%Y-%m-%d %H:%M:%S"
```

### 5.3 显示UTC时间
```bash
date -u
```

### 5.4 显示RFC格式的日期时间
```bash
date -R
```

### 5.5 显示ISO格式的日期时间
```bash
date -I
```

### 5.6 显示文件的最后修改时间
```bash
date -r file.txt
```

## 6. 注意事项
- date 命令用于显示和设置系统日期时间
- 使用不同的格式选项可以显示不同格式的日期时间
- 设置系统日期时间需要 root 权限
- date 命令是一个常用的系统工具，用于各种日期时间相关的操作
