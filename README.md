# 弈安录简单介绍

**弈安录** 是 GYscan 的子项目，是一款专注于网络安全学习的 Android 应用。该应用提供了全面的网络安全相关学习资料，包括安全基础知识、Linux 命令手册、Windows 命令手册以及 Linux 常用网安工具命令手册等内容。

## 功能模块

### 1. 安全基础知识
- **网络安全基础**：介绍网络安全的基本概念、威胁类型、防护措施以及相关法律法规
- **安全概述**：提供网络安全的整体概览
- **高级渗透测试**：包含高级渗透测试相关内容

### 2. Linux命令
- **Linux系统自带命令手册**：涵盖文件操作、进程管理、网络配置、系统监控等常用命令
- **命令分类**：包括文件系统、系统管理、软件包管理、进程管理、网络工具、shell环境等

### 3. Windows命令
- **Windows系统命令手册**：涵盖文件操作、系统管理、磁盘管理、网络配置等常用命令

### 4. 网络安全工具
- **Linux常用网安工具命令手册**：涵盖逆向分析、网络扫描、网络抓包、渗透测试和日志分析等常用工具

### 5. 其他功能
- **关于页面**：显示应用的版本信息和开发者信息
- **相关项目页面**：展示相关的项目信息
- **条款页面**：显示用户协议和隐私政策
- **彩蛋页面**：提供一些有趣的内容

## 技术实现

### 架构设计
- 采用标准的 Android 应用架构
- 使用 Activity 作为页面容器
- 采用 RecyclerView 实现列表展示
- 使用 DrawerLayout 实现侧边栏导航

### 界面设计
- 适配暗色模式和亮色模式
- 适配系统主题
- 适配系统语言
- 适配系统字体
- 适配系统图标
- 适配所有组件的显示问题
- 适配挖孔屏、水滴屏、刘海屏的显示问题

### 数据存储
- 使用 SharedPreferences 存储用户偏好设置
- 使用 assets 目录存储命令手册的 Markdown 文件

## 使用说明

### 首次启动
1. 打开应用，首先会看到用户协议页面，需要点击同意才能继续使用
2. 进入主页面后，点击"开始学习"按钮
3. 首次点击会显示应用介绍页面，了解应用的基本功能
4. 介绍页面结束后，会自动打开侧边栏导航

### 导航使用
1. 点击主页面左上角的菜单按钮，打开侧边栏导航
2. 在侧边栏中选择需要学习的模块
3. 点击模块后，会进入相应的模块页面

### 命令手册使用
1. 进入 Linux 命令或 Windows 命令模块
2. 选择相应的命令分类
3. 在分类页面中选择具体的命令
4. 查看命令的详细说明和使用方法

### 安全知识学习
1. 进入安全基础知识模块
2. 选择需要学习的安全知识类别
3. 查看相关的安全知识内容

## 项目结构

```
gyscan-doc/
├── app/
│   ├── src/
│   │   ├── main/
│   │   │   ├── java/com/gyscan_doc/
│   │   │   │   ├── CyberSecurity/  # 网络安全工具模块
│   │   │   │   ├── Home/           # 主页相关
│   │   │   │   ├── Intro/          # 应用介绍
│   │   │   │   ├── Linux/          # Linux命令模块
│   │   │   │   ├── Security/       # 安全知识模块
│   │   │   │   ├── Windows/        # Windows命令模块
│   │   │   │   ├── adapter/        # 适配器
│   │   │   │   ├── model/          # 数据模型
│   │   │   │   ├── utils/          # 工具类
│   │   │   │   ├── MainActivity.java        # 主页面
│   │   │   │   ├── AboutActivity.java       # 关于页面
│   │   │   │   ├── EasterEggActivity.java   # 彩蛋页面
│   │   │   │   ├── TermsActivity.java       # 条款页面
│   │   │   │   └── RelatedProjectsActivity.java  # 相关项目页面
│   │   │   ├── res/                # 资源文件
│   │   │   ├── assets/             # 命令手册Markdown文件
│   │   │   └── AndroidManifest.xml # 应用配置文件
│   └── build.gradle                # 应用构建配置
└── build.gradle                    # 项目构建配置
```
